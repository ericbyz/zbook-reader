戴维南和诺顿说：不用。他们发现了一条惊人的规律——**任何一个复杂的线性电路，对于外接的负载来说，都可以等效成"一个电源 + 一个电阻"这么简单的东西。** 几十个元件变成一个，10 个方程变成一个方程。这就是本章的核心。

## 1. 先别急着看公式——生活里的"等效"思维

### 1a. 生活例子 1：快递系统

你在淘宝下单买了一个手机壳。几天后，一个快递员敲你的门，把包裹递给你。

你只看到了一个快递员、一个包裹。但快递公司内部呢？你的包裹从商家仓库出发，经过分拣中心 A，装车运到分拣中心 B，再转运到分拣中心 C，最后分配到末端站点，由快递员送上门。这个过程中间经过了卡车、传送带、手持扫码枪、自动分拣机……几十个环节。

但对你来说，这一切都不重要。你只需要知道：**一个快递员从一个仓库出发，送到你家。**

快递公司内部的复杂网络，对你的收货体验来说，等效于"仓库→快递员→你家"这条最简单的链路。

这就是"等效"（Equivalent）的思维：**保留对外接口的行为不变，把内部的一切复杂结构压缩掉。**

### 1b. 生活例子 2：餐厅后厨

你去餐厅点了一份宫保鸡丁。服务员把菜端上桌。你吃到的是一盘菜，但从厨房传菜口出来的那一刻之前发生了什么？

后厨里有主厨、配菜、打荷、洗碗工。灶台上有六个炉头。有人切菜、有人调味、有人掌勺、有人摆盘。这盘宫保鸡丁可能经过了三个人的手：配菜工切好鸡丁和花生，打荷工准备好调料汁，主厨下锅翻炒。

但对服务员来说，厨房就是"一个出菜口"。不管你点什么菜，服务员都只和这个出菜口打交道。厨师换人了？炉头坏了？只要出菜口的菜还是那个味道，服务员和顾客都无所谓。

这就是戴维南和诺顿定理的思想：**不管电路内部多复杂，对外端口的行为可以等效为一个最简单的"二元件模型"。** 对负载来说，它不关心你电路里有几个电阻、几个电源、怎么连的。它只关心两件事：你给我的电压是多少？你能供出的电流有多大？

## 2. 戴维南定理（Thévenin's Theorem）

### 2a. 直观理解：黑盒子里有什么？

想象你面前有一个黑盒子，上面只有两个端子（接线柱）露出来。你不知道盒子里是什么。可能是一个 9 V 电池串了一个电阻，也可能是一个复杂的电路板上有几十个元件。你只关心一件事：**把负载接到这两个端子上，负载上的电压和电流是多少？**

戴维南定理（Thévenin's Theorem，有时也译作"戴维宁定理"）告诉你一个惊人的事实：

> **任何一个由线性元件（电阻、独立源、受控源）组成的二端（一端口）网络，对外部电路来说，总可以等效为一个理想电压源 $V_{th}$ 串联一个电阻 $R_{th}$。**

画出来就是这样：

```
    原电路：                        等效电路：
  ┌─────────────┐                 ┌─────────────┐
  │             │                 │             │
  │  复杂线性   │  a              │    Rth      │  a
  │   电路      ├────────        ─┤[~~~]───+   ├────────
  │  (很多元件)  │                │         │   │
  │             │  b             │    Vth  │   │  b
  └─────────────┘                └─────────┴───┘
```

左边那个复杂电路可能有 20 个电阻、5 个电源，右边只有 2 个元件。但只要你把同样的负载接在 a、b 端口上，**负载两端的电压和流过的电流，在两种情况下完全一样。**

### 2b. 形式化定义

**戴维南定理**：任何一个线性有源一端口网络 N，对于其外部电路而言，可以用一个电压源 $V_{th}$ 与一个电阻 $R_{th}$ 的串联组合来等效替代。其中：

- $V_{th}$（戴维南等效电压）= 网络 N 的**开路电压**（端口不接任何负载时，a、b 两端的电压）
- $R_{th}$（戴维南等效电阻）= 网络 N 内部所有**独立源置零**后，从 a、b 端口看进去的等效电阻

"独立源置零"是什么意思？这是初学者最容易困惑的地方，我们马上详细解释。

先把这个定义翻译成人话：

$$
V_{th} = V_{oc} \quad \text{（端口开路时的电压）}
$$

$$
R_{th} = R_{eq} \quad \text{（所有独立源"关掉"后从端口看进去的电阻）}
$$

所谓"关掉"独立源，就是：
- **电压源** → 短路（用一根导线代替，因为理想电压源内阻为零）
- **电流源** → 开路（直接拿走，因为理想电流源内阻为无穷大）

> ⚠️ 注意：只有**独立源**可以"关掉"。受控源（依赖电压/电流控制的源）**不能关掉**，必须保留在电路里。这个我们第 5 节会重点讲。

### 2c. 为什么等效？

现在你可能有一个问题：凭什么说外面接什么负载，等效电路都和原电路表现一样？

直觉上可以这样理解：一个线性电路的端口行为，完全由两个参数决定——它"能推多大力"（开路电压 $V_{th}$）和"自己有多大阻力"（内阻 $R_{th}$）。这两个参数一旦确定，不管外面接什么，端口上的电压和电流关系就唯一确定了。

从代数的角度看，如果你对原电路列节点方程，把所有未知量消去，只保留端口电压 $v$ 和端口电流 $i$ 的关系，你一定会得到一个一次方程：

$$
v = V_{th} - R_{th} \cdot i
$$

这就是戴维南等效电路的 V-I 方程。它是一条直线。任何一个线性一端口网络的端口 V-I 特性都是一条直线，而一条直线只需要两个参数（截距 $V_{th}$ 和斜率 $-R_{th}$）就能完全确定。

**验证这个公式**：当端口开路时，$i = 0$，代入得 $v = V_{th}$。没错，$V_{th}$ 就是开路电压。当端口短路时，$v = 0$，代入得 $0 = V_{th} - R_{th} \cdot I_{sc}$，即 $I_{sc} = V_{th}/R_{th}$。短路电流 $I_{sc}$ 可以从原电路直接算出来。这也提供了一个求 $R_{th}$ 的方法：$R_{th} = V_{oc} / I_{sc}$（我们第 3 节细讲）。

### 2d. 手算示例 1：最基本的电阻分压电路

来看一个最简单的情况，先把"求等效"的过程跑一遍。

**题目**：下图电路中，$V_s = 12\text{ V}$，$R_1 = 4\text{ Ω}$，$R_2 = 6\text{ Ω}$。求 a、b 端口（$R_2$ 两端）的戴维南等效电路。

```
         R1 (4Ω)
    ────[~~~]────┬──── a
    │            │
   (+)           │
    │   Vs=12V  [ ] R2 (6Ω)   ← 这是负载
    │            │
    └────────────┴──── b
```

等一下，这个电路里 $R_2$ 就是"负载"吗？先明确端口。题目说"求 a、b 端口的戴维南等效"，那 $R_2$ 就是负载，我们要等效的是 $R_2$ **左边**的网络。

把 $R_2$ 拿掉，剩下的电路是：12 V 电压源串一个 4 Ω 电阻。

**第 1 步：求 $V_{oc}$（开路电压）。**

把 $R_2$ 断开，a、b 之间是开路。开路时没有电流流过 $R_1$，所以 $R_1$ 上压降为 0。a、b 两端的电压就等于电压源电压：

$$
V_{oc} = V_s = 12\text{ V}
$$

所以 $V_{th} = 12\text{ V}$。

**第 2 步：求 $R_{th}$（等效内阻）。**

所有独立源置零。电压源 $V_s$ 短路（用导线代替）。从 a、b 看进去，$R_1$ 一端连 a（经过短路后的电压源位置），另一端连 b。所以从 a、b 看进去的电阻就是 $R_1$：

```
         R1 (4Ω)              ← 从 a 经过短路线到 R1 再到 b
    ────[~~~]────┬──── a
    │            │
    │  (短路)    │
    │            │  (R2 已移除)
    └────────────┴──── b
```

从 a、b 看进去的路径：a → 经过短路线到 $R_1$ 上端 → $R_1$ → b。电阻就是 $R_1 = 4\text{ Ω}$。

$$
R_{th} = 4\text{ Ω}
$$

**第 3 步：画出等效电路。**

戴维南等效电路：12 V 电压源串联 4 Ω 电阻。

```
         Rth (4Ω)     a
    ────[~~~]────+─────
                 │
            Vth  │  12V
                 │
    ─────────────┴─────
                      b
```

**第 4 步：验证。** 把原来的负载 $R_2 = 6\text{ Ω}$ 接回等效电路：

原电路中 $R_2$ 的电流：
$$
I_{原} = \frac{V_s}{R_1 + R_2} = \frac{12}{4 + 6} = 1.2\text{ A}
$$
$R_2$ 两端电压：
$$
V_{原} = I_{原} \times R_2 = 1.2 \times 6 = 7.2\text{ V}
$$

等效电路中 $R_2$ 的电流：
$$
I_{等效} = \frac{V_{th}}{R_{th} + R_2} = \frac{12}{4 + 6} = 1.2\text{ A}
$$
$$
V_{等效} = I_{等效} \times R_2 = 1.2 \times 6 = 7.2\text{ V}
$$

完全一致。等效成功。

### 2e. 手算示例 2：含两个电压源和三个电阻的电路

**题目**：求 a、b 端口的戴维南等效。$V_{s1}=10\text{ V}$, $V_{s2}=6\text{ V}$, $R_1=2\text{ Ω}$, $R_2=5\text{ Ω}$, $R_3=3\text{ Ω}$。

```
         R1 (2Ω)    R3 (3Ω)        a
    ────[~~~]───┬────[~~~]─────────┬───
    │           │                  │
   (+)          │                  │
    │  Vs1=10V [ ] R2 (5Ω)        [ ] RL (负载, 先移除)
    │           │                  │
    │           │    Vs2=6V        │
    └───────────┼────(+)───────────┼───
                │                  │
                └──────────────────┴─── b
```

**第 1 步：求 $V_{oc}$（a、b 开路）。**

设 b = 0 V（参考点）。各节点：
- $V_{s1}$ 上端 = $0 + 10 = 10\text{ V}$
- c 点（$V_{s2}$ 上端、$R_2$ 右端）= $0 + 6 = 6\text{ V}$（$V_{s2}$ 固定 c 为 6 V）
- d 点（$R_1$ 右端、$R_2$ 左端、$R_3$ 左端）= 未知，设为 $v_d$

a 端开路，$R_3$ 上无电流，所以 $v_a = v_d$。

在 d 点 KCL（只有 $R_1$ 和 $R_2$ 有电流，$R_3$ 无电流）：
$$
\frac{10 - v_d}{2} = \frac{v_d - 6}{5}
$$

两边乘 10：$5(10 - v_d) = 2(v_d - 6)$ → $50 - 5v_d = 2v_d - 12$ → $62 = 7v_d$ → $v_d = \frac{62}{7} \approx 8.857\text{ V}$。

因此：
$$
V_{th} = V_{oc} = v_a = v_d = \frac{62}{7} \approx 8.857\text{ V}
$$

**第 2 步：求 $R_{th}$（独立源置零法）。**

$V_{s1}$ 短路，$V_{s2}$ 短路。从 a、b 看进去：$R_3$ 串联 $(R_1 \parallel R_2)$。因为短路后 $R_1$ 左端和 $R_2$ 右端都直接连 b，$R_1$ 和 $R_2$ 共享 d 和 b 两点，是并联关系。

$$
R_{th} = R_3 + \frac{R_1 R_2}{R_1 + R_2} = 3 + \frac{2 \times 5}{2 + 5} = 3 + \frac{10}{7} = \frac{31}{7} \approx 4.429\text{ Ω}
$$

**第 3 步：验证（接 $R_L = 10\text{ Ω}$）。**

等效电路：$I_L = \dfrac{62/7}{31/7 + 10} = \dfrac{62/7}{101/7} = \dfrac{62}{101} \approx 0.614\text{ A}$，$V_L = 6.14\text{ V}$。

原电路（节点法验证）：设 $v_d$、$v_a$，KCL at d 和 a 联立解得 $v_d = 806/101$, $v_a = 8060/1313$。$I_L = v_a/10 = 806/1313 = 62/101\text{ A}$。完全一致。

所以：
$$
V_{th} = \frac{62}{7}\text{ V}, \quad R_{th} = \frac{31}{7}\text{ Ω}
$$

## 3. 求 Rth 的三种方法

求 $V_{th}$（开路电压）相对直接，用节点法或网孔法通常一步到位。求 $R_{th}$ 则更讲究，因为有三种不同的方法，每种适用于不同的情况。掌握这三种方法，你就有了对付任何电路的工具箱。

### 3a. 方法 1：独立源置零法（最直观）

**做法**：
1. 移除负载
2. 所有**独立电压源**→短路（导线代替）
3. 所有**独立电流源**→开路（直接拿掉）
4. 从端口看进去，计算等效电阻（用串并联化简）

**适用场景**：
- 电路里只有**独立源**，没有受控源
- 或者受控源在置零后不影响电阻计算（这种情况很少见）

**手算示例**：用方法 1 求下面电路的 $R_{th}$。

```
         R1 (6Ω)         R3 (4Ω)
    ────[~~~]───┬─────────[~~~]──── a
    │           │                   │
   (+)          │                  [ ] RL (移除)
    │  Vs=18V  [ ] R2 (3Ω)         │
    │           │                   │
    └───────────┴─────────────────── b
         R4 (2Ω)
```

先拿掉 $R_L$。求 $R_{th}$：$V_s$ 短路，$R_4$ 在左边，$R_1$ 上端经过短路的 $V_s$ 到 b。

等等，看电路图：$V_s$ 下端连 b，上端连 $R_1$ 左端和 $R_4$ 上端。所以 $V_s$ 短路后，$R_1$ 左端和 $R_4$ 上端都直接连 b。$R_4$ 下端直接连 b。

所以短路后：$R_4$ 两端都被短路（上端和下端都连 b），$R_4$ 被"旁路"了，等效电阻贡献为 0。

现在看 a→b：先过 $R_3 = 4\text{ Ω}$，到达中间节点。从中间节点出发，$R_2 = 3\text{ Ω}$ 下端连 b（经过 $R_4$ 的短路导线），$R_1 = 6\text{ Ω}$ 左端也连 b（经过短路的 $V_s$）。所以 $R_1$ 和 $R_2$ 并联。

$$
R_{th} = R_3 + (R_1 \parallel R_2) = 4 + \frac{6 \times 3}{6 + 3} = 4 + \frac{18}{9} = 4 + 2 = 6\text{ Ω}
$$

额外说明：$R_4$ 被短接后，从端口看进去等效于不存在。这也是一个常见的"陷阱"——短路并联的电阻会被旁路。

### 3b. 方法 2：开路电压/短路电流法（最通用）

**做法**：
1. 求端口开路电压 $V_{oc}$
2. 将端口短路，求短路电流 $I_{sc}$（从 a 流向 b 的电流）
3. $R_{th} = \dfrac{V_{oc}}{I_{sc}}$

**为什么这个公式成立？** 看戴维南等效电路：如果端口短路（a、b 用导线连接），电路中只有 $V_{th}$ 和 $R_{th}$。短路电流 $I_{sc} = V_{th} / R_{th}$。所以 $R_{th} = V_{th} / I_{sc}$。

注意方向：$I_{sc}$ 的方向是从 a 经短路线流到 b。这和 $V_{oc}$ 的极性一致（$V_{oc} = v_a - v_b$）。

**适用场景**：
- 含受控源的电路（因为受控源不能置零）
- 电路结构复杂，置零后不好直接算电阻
- **任何情况都能用**，是最通用的方法

**手算示例**：用方法 2 求刚才示例 1 的 $R_{th}$（已知 $V_s = 12\text{ V}$，$R_1 = 4\text{ Ω}$，$R_2$ 是负载）。

已知 $V_{oc} = 12\text{ V}$。

把端口短路：a、b 之间用导线连接。此时 $R_1$ 两端：一端是 $V_s = 12\text{ V}$，另一端是短路到 b = 0 V。所以短路电流：
$$
I_{sc} = \frac{V_s}{R_1} = \frac{12}{4} = 3\text{ A}
$$

$$
R_{th} = \frac{V_{oc}}{I_{sc}} = \frac{12}{3} = 4\text{ Ω}
$$

与方法 1 结果一致。

### 3c. 方法 3：外加电源法（测试源法）

**做法**：
1. 所有独立源置零
2. 在端口 a、b 之间外加一个**测试电压源** $V_t$（或测试电流源 $I_t$）
3. 计算从测试源流出的电流 $I_t$（或测试源两端的电压 $V_t$）
4. $R_{th} = \dfrac{V_t}{I_t}$（如果加的是电压源）或 $R_{th} = \dfrac{V_t}{I_t}$（如果加的是电流源，本质一样）

**适用场景**：
- 含受控源的电路（独立源已置零，但受控源保留）
- 置零后的电路不适合直接串并联化简
- 与方法 2 互为补充

**手算示例**：用方法 3 求示例 1 的 $R_{th}$。

独立源 $V_s$ 置零（短路）。在 a、b 之间加测试电压源 $V_t$。求 $I_t$。

$V_t$ 接在 a、b 两端。a 经过短路的 $V_s$ 连到 b 再经过 $R_1$……等等。短路后电路就是 $R_1$ 直接跨接在 a 和 b 之间。所以：
$$
I_t = \frac{V_t}{R_1} = \frac{V_t}{4}
$$

$$
R_{th} = \frac{V_t}{I_t} = \frac{V_t}{V_t/4} = 4\text{ Ω}
$$

又是一个 4 Ω。三种方法一致，这个电路的 $R_{th}$ 确定是 4 Ω。

### 3d. 对比总结

| 方法 | 核心操作 | 需要置零独立源 | 受控源如何处理 | 适用场景 |
|------|---------|:---:|------|------|
| 独立源置零法 | 关掉源，直接算电阻 | ✅ 是 | 不能置零（一般也遇不到） | 纯电阻 + 独立源电路 |
| 开路/短路法 | $R_{th} = V_{oc}/I_{sc}$ | ❌ 否 | 保留，正常参与计算 | 通用，特别是含受控源 |
| 外加电源法 | 加测试源求 $V/I$ | ✅ 是 | 保留，参与电路计算 | 含受控源，或串并联不便 |

三种方法中，方法 2（开路/短路法）对期末考试最实用。因为它不依赖串并联，任何电路都能用。方法 1 最直观，适合验证。方法 3 是方法 2 的"替身"，在受控源电路中尤其重要。

### 3e. 手算：同一电路用三种方法求 Rth

**题目**：求下图中 a、b 端口的 $R_{th}$。

```
    ────[~~~]───┬──── a
        R1=4Ω   │
                │
    Is=2A  ↑   [ ] R2=6Ω
                │
    ────────────┴──── b
```

电路说明：电流源 $I_s = 2\text{ A}$（箭头向上，从 b 流向中间节点），$R_1$ 在 a 和中间节点之间，$R_2$ 在中间节点和 b 之间。a、b 是端口，负载从 a、b 接入。

首先，这不是一个简单的串并联。让我们分别用三种方法。

**方法 1：独立源置零法。**

移除负载。电流源置零 → 开路（直接拿掉，相当于断线）。

拿掉电流源后，电路变成：$R_1$ 上端连 a，下端连中间节点。$R_2$ 上端连中间节点，下端连 b。而原来的电流源位置是开路的。

从 a 到 b：路径是 a → $R_1$ → 中间节点 → $R_2$ → b。$R_1$ 和 $R_2$ 串联。

$$
R_{th} = R_1 + R_2 = 4 + 6 = 10\text{ Ω}
$$

**方法 2：开路电压/短路电流法。**

先求 $V_{oc}$（a、b 开路）。

电流源 $I_s = 2\text{ A}$ 从 b 流向中间节点，然后经 $R_2$ 回到 b（$R_1$ 支路因 a 开路无电流）。所以 2 A 全部流过 $R_2$：

$$
V_{R2} = 2 \times 6 = 12\text{ V}
$$

a 端通过 $R_1$（无电流流过）连到中间节点，$V_a$ = 中间节点电压。所以：
$$
V_{oc} = V_a - V_b = V_{R2} = 12\text{ V}
$$

现在求 $I_{sc}$（a、b 短路）。

将 a、b 短接。此时三个元件：$I_s = 2\text{ A}$，$R_1 = 4\text{ Ω}$，$R_2 = 6\text{ Ω}$。

电流源提供 2 A 的恒定电流。它到达中间节点后分流：一部分经 $R_1$ 到 a，然后经短路线到 b；另一部分经 $R_2$ 直接到 b。

设流过 $R_1$ 的电流为 $i_1$（从中间节点到 a 方向），流过 $R_2$ 的电流为 $i_2$（从中间节点到 b 方向）。

KCL at 中间节点：
$$
2 = i_1 + i_2
$$

a、b 短路意味着 $V_a = V_b = 0$。已知 b = 0 V，设中间节点电压为 $v_m$。

电流 $i_1 = (v_m - 0)/R_1 = v_m/4$（从中间节点经 $R_1$、短路到 b 的路径）

电流 $i_2 = (v_m - 0)/R_2 = v_m/6$

代入 KCL：
$$
2 = \frac{v_m}{4} + \frac{v_m}{6} = v_m \left(\frac{3+2}{12}\right) = v_m \cdot \frac{5}{12}
$$

$$
v_m = 2 \times \frac{12}{5} = \frac{24}{5} = 4.8\text{ V}
$$

短路电流 $I_{sc}$ 是从 a 经过短路到 b 的电流，即 $i_1$：
$$
I_{sc} = i_1 = \frac{v_m}{4} = \frac{4.8}{4} = 1.2\text{ A}
$$

所以：
$$
R_{th} = \frac{V_{oc}}{I_{sc}} = \frac{12}{1.2} = 10\text{ Ω}
$$

与方法 1 一致！

**方法 3：外加电源法。**

所有独立源置零。电流源 $I_s$ 开路（拿掉）。在 a、b 加测试电压源 $V_t$，求 $I_t$。

电路变成：$R_1$ 和 $R_2$ 串联在 a 和 b 之间。测试源 $V_t$ 接在 a、b 之间。所以：
$$
I_t = \frac{V_t}{R_1 + R_2} = \frac{V_t}{10}
$$

$$
R_{th} = \frac{V_t}{I_t} = 10\text{ Ω}
$$

三种方法，结果一致。哪个方法最省力取决于电路结构。**建议：纯独立源电路用方法 1，含受控源用方法 2 或方法 3。**

## 4. 诺顿定理（Norton's Theorem）

### 4a. 核心思想：电压源串联 → 电流源并联

戴维南说：等效成一个电压源串电阻。
诺顿说：等价的表述是，等效成一个电流源并电阻。

哪个对？都对。它们是**完全的等价变换**，就像 $2+2=4$ 和 $1+3=4$ 说的是同一个数。

**诺顿定理**：任何一个线性有源一端口网络，对外部电路而言，总可以等效为一个理想电流源 $I_n$ 并联一个电阻 $R_n$。

其中：
- $I_n$（诺顿等效电流）= 端口**短路电流** $I_{sc}$（a、b 短接时从 a 流向 b 的电流）
- $R_n$（诺顿等效电阻）= $R_{th}$（**同一个数！** 等效电阻不随戴维南/诺顿形式而变）

画出诺顿等效电路：

```
         a                          a
    ──────┬──────             ──────┬──────
          │                        │
         [ ] Rn                 [ ] Rn = Rth
          │                        │
          │    In                  │    In = Isc
          ↑                       ↑
          │                        │
    ──────┴──────             ──────┴──────
         b                          b
     (诺顿等效)               (标注了 In 方向)
```

### 4b. 戴维南 ↔ 诺顿互相转换

这两个等效形式之间有一个简单的转换关系。你只要记住四个字：**源变换**（Source Transformation）。

**戴维南 → 诺顿**：

已知 $V_{th}$ 串联 $R_{th}$，转换为诺顿等效：

$$
I_n = \frac{V_{th}}{R_{th}}, \quad R_n = R_{th}
$$

画法：$R_n$ 与 $I_n$ 并联，$I_n$ 的箭头方向从 a 经电流源内部到 b（即使得 a 点为高电位）。

```
 戴维南形式：                  诺顿形式：
     Rth                         a
  ──[~~~]──+── a              ───┬──────
           │                    │
       Vth │                   [ ] Rn=Rth
           │                    │
  ─────────┴── b              [↑] In=Vth/Rth
                                 │
                              ───┴────── b
```

**诺顿 → 戴维南**：

已知 $I_n$ 并联 $R_n$，转换为戴维南等效：

$$
V_{th} = I_n \times R_n, \quad R_{th} = R_n
$$

这就是电压源和电流源之间的"恒等变形"。本质上，任何一个串联组合（电压源 + 电阻）都可以变成并联组合（电流源 + 同阻值的电阻），电流大小 = 电压/电阻。反过来也一样。

### 4c. 什么时候用哪个？

- **串联为主的电路** → 戴维南更方便。因为串联元件共享电流，电压源串联电阻很自然。
- **并联为主的电路** → 诺顿更方便。因为并联元件共享电压，电流源并联电阻很自然。
- **综合题** → 先求戴维南（求 $V_{oc}$ 通常比求 $I_{sc}$ 直观），最后根据需要转换。

### 4d. 手算示例：双源电路的诺顿等效

**题目**：求下图中 a、b 端口的诺顿等效电路。

```
         R1 (8Ω)         R2 (4Ω)
    ────[~~~]───┬─────────[~~~]──── a
    │           │                   │
   (+)          │                  │
    │  Vs=24V  [↑] Is=3A          │
    │           │                  │
    └───────────┴────────────────── b
```

$a$、$b$ 是输出端口。负载未画出。$I_s$ 箭头向上（电流从 b 流向中间节点）。

**第 1 步：求 $I_n$（短路电流）。**

将 a、b 短接。$R_2$ 被短路（两端都 = 0）。电流源的 3 A 从 b 流向中间节点（设为 $v_m$）。

流过 $R_1$ 的电流：$i_1 = (24 - v_m)/8$。在中间节点 KCL：
$$
i_1 + 3 = i_{sc}
$$
其中 $i_{sc}= v_m/R_2 = v_m/4$（$R_2$ 两端电压 = $v_m - 0$）。所以：
$$
\frac{24 - v_m}{8} + 3 = \frac{v_m}{4}
$$

两边乘 8：
$$
24 - v_m + 24 = 2v_m
$$

$$
48 = 3v_m
$$

$$
v_m = 16\text{ V}
$$

$$
I_{sc} = \frac{16}{4} = 4\text{ A}
$$

所以 $I_n = I_{sc} = 4\text{ A}$。

**第 2 步：求 $R_n$（= $R_{th}$）。**

用方法 1（独立源置零）：$V_s$ 短路，$I_s$ 开路。从 a、b 看进去。

独立源置零后：$R_1$ 左端通过短路线连到 b，右端连中间节点。电流源位置开路。$R_2$ 左端连中间节点，右端连 a。

从 a 看：$R_2$ → 中间节点 → $R_1$ → b。串联。

$$
R_n = R_{th} = R_1 + R_2 = 8 + 4 = 12\text{ Ω}
$$

**第 3 步：诺顿等效电路。**

$$
I_n = 4\text{ A}, \quad R_n = 12\text{ Ω}
$$

并联形式：$R_n$ 并 $I_n$，$I_n$ 方向从 b 经电流源到 a（即使 a 为高电位）。

```
         a
    ─────┬──────
         │
        [ ] Rn=12Ω
         │
        [↑] In=4A
         │
    ─────┴──────
         b
```

**第 4 步：验证（转为戴维南形式）。**

转为戴维南：$V_{th} = I_n \times R_n = 4 \times 12 = 48\text{ V}$，$R_{th} = 12\text{ Ω}$。

接 $R_L = 6\text{ Ω}$ 负载到端口。戴维南等效：
$$
I_L = \frac{48}{12+6} = \frac{48}{18} = \frac{8}{3} \approx 2.667\text{ A}
$$

诺顿等效：$R_n \parallel R_L = 12 \parallel 6 = 12 \times 6 / (12 + 6) = 72/18 = 4\text{ Ω}$。$I_n = 4\text{ A}$ 分流到 $R_n$ 和 $R_L$：
$$
I_L = I_n \times \frac{R_n}{R_n + R_L} = 4 \times \frac{12}{12 + 6} = 4 \times \frac{12}{18} = \frac{48}{18} = \frac{8}{3}\text{ A}
$$

一致。

## 5. 含受控源的戴维南等效

前面都是只有独立源的电路。一旦电路里出现**受控源**（dependent source / controlled source），规则就变了。

### 5a. 受控源回忆

受控源是一种"不独立的电源"，它的电压或电流不是固定的，而是依赖于电路中某个位置的电压或电流。四种类型：

| 缩写 | 全称 | 含义 |
|------|------|------|
| VCVS | Voltage Controlled Voltage Source | 电压控制电压源 |
| VCCS | Voltage Controlled Current Source | 电压控制电流源 |
| CCVS | Current Controlled Voltage Source | 电流控制电压源 |
| CCCS | Current Controlled Current Source | 电流控制电流源 |

例：一个 VCVS 的值为 $0.5 v_x$，意思是"这个电压源的输出电压等于 $v_x$（电路中某个电压）的一半"。

### 5b. 关键区别：受控源不能置零！

这是初学者最容易犯的错误。

- **独立源**：电压是固定的（比如 12 V）、电流是固定的（比如 2 A）。可以"关掉"（置零）。
- **受控源**：电压/电流依赖于电路中的其他变量。**不能关掉。** 因为一旦你关掉它，电路的拓扑和行为就被改变了。受控源在求 $R_{th}$ 时**必须原样保留**。

所以，含受控源的电路在求 $R_{th}$ 时：
- **方法 1（独立源置零法）不能用。** 受控源不能置零，电路行为不"纯"。
- **方法 2（开路/短路法）** 可以用，但注意计算 $I_{sc}$ 时要正确处理受控源。
- **方法 3（外加电源法）** 是最常用的，但记住：先把独立源置零，**受控源保持不动**。

### 5c. 手算示例：含 VCVS 的电路

**题目**：求下图 a、b 端口的戴维南等效。其中 $v_x$ 是 $R_1$ 两端电压（参考方向：左 + 右 -），VCVS 电压为 $2v_x$，极性上 + 下 -。

```
         R1 (4Ω)          R2 (2Ω)
    ────[~~~]───┬─────────[~~~]──── a
    │           │                    
   (+)          │ +                  
    │  Vs=20V   ├─┤ 2v_x  (VCVS)     
    │           │ -                  
    └───────────┤                    
                │                    
                └──────────────────── b
```

先把负载从 a、b 移除。a、b 就是我们要等效的端口。

**第一步：求 $V_{oc}$（a、b 开路电压）。**

a、b 开路，没有电流流出。设 b = 0 V（参考点）。

此时 VCVS 是电路中唯一连到 a 的元件。VCVS 下端正极连 b = 0，所以 VCVS 下端电压 = 0 V（正极相对于底部）。

等等，VCVS 标记为"上 + 下 -"，意思是上端是正极、下端是负极。所以 VCVS 的电压约束是 $V_{上} - V_{下} = 2v_x$（上比下高 $2v_x$）。

VCVS 下端正极连 b = 0………不对！标记是"上 + 下 -"，这意味着上端为正极，下端为负极。上端连中间节点，下端连 b。所以约束是 $V_{中} - V_{b} = 2v_x$ = $V_{中} = 2v_x$。

$R_2$ 在中间节点和 a 之间。a 开路，没有电流流过 $R_2$，所以 $R_2$ 两端电压 = 0，a 点电压 = 中间节点电压。

现在分析 $R_1$ 和 $V_s$。$V_s = 20\text{ V}$ 的下端连 b = 0，上端 = 20 V。$R_1$ 左端 = 20 V，右端 = 中间节点（电压未知，设为 $v_m$）。

$v_x$ 是 $R_1$ 两端电压（左 + 右 -）：$v_x = 20 - v_m$。

流过 $R_1$ 的电流：$i_{R1} = (20 - v_m)/4$（从左到右）。

这个电流到了中间节点后去哪了？两条路：
- 经 $R_2$ 到 a（a 开路，无电流）
- 经 VCVS 到 b（VCVS 是电压源，可以吸收/提供电流）

设流过 VCVS 的电流为 $i_{vcvs}$（从上端流向下端，即从中间节点流向 b）。

KCL at 中间节点：
$$
\frac{20 - v_m}{4} = i_{vcvs} + 0
$$

VCVS 两端电压：$v_m - 0 = 2 v_x = 2(20 - v_m)$

$$
v_m = 2(20 - v_m) = 40 - 2v_m
$$

$$
3 v_m = 40
$$

$$
v_m = \frac{40}{3} \approx 13.333\text{ V}
$$

$a$ 点电压 = $v_m = 40/3\text{ V}$。b = 0。所以：
$$
V_{oc} = v_a - v_b = \frac{40}{3}\text{ V}
$$

$$
V_{th} = \frac{40}{3} \approx 13.333\text{ V}
$$

**第二步：求 $R_{th}$（用开路/短路法）。**

求 $I_{sc}$：将 a、b 短路。a = b = 0 V。

此时 VCVS 下端 = b = 0。VCVS 上端 = 中间节点（$v_m$）。VCVS 电压约束：$v_m - 0 = 2 v_x$ → $v_m = 2 v_x$。

$R_1$ 左端 = 20 V，右端 = $v_m$。$v_x = 20 - v_m$。

所以 $v_m = 2(20 - v_m)$ → $v_m = 40 - 2v_m$ → $3v_m = 40$ → $v_m = 40/3\text{ V}$。

$R_2$ 在 $v_m$ 和 a = 0 之间。流过 $R_2$ 的电流（即 $I_{sc}$）：
$$
I_{sc} = \frac{v_m - 0}{R_2} = \frac{40/3}{2} = \frac{20}{3} \approx 6.667\text{ A}
$$

现在 $R_{th} = V_{oc} / I_{sc}$：
$$
R_{th} = \frac{40/3}{20/3} = \frac{40}{3} \times \frac{3}{20} = \frac{40}{20} = 2\text{ Ω}
$$

**第三步：戴维南等效。**

$$
V_{th} = \frac{40}{3}\text{ V}, \quad R_{th} = 2\text{ Ω}
$$

**第四步：验证（接负载 $R_L = 4\text{ Ω}$）。**

等效电路：$I_L = \frac{40/3}{2+4} = \frac{40/3}{6} = \frac{40}{18} = \frac{20}{9} \approx 2.222\text{ A}$。

原电路验证（略，可用节点法或网孔法验证一致性）。

这就是含受控源的戴维南等效的完整流程。核心要点就一条：**受控源在求 $R_{th}$ 时不能置零，必须用开路/短路法或外加电源法。**

## 6. 戴维南定理的应用场景

学了定理，得知道它能干什么。戴维南定理在工程中有三大应用场景。

### 6a. 分析负载变化

假设你设计了一个传感器接口电路，电路很复杂，输出端要接不同型号的 ADC（模数转换器）模块。每个模块的输入电阻不一样：有 50 Ω 的、100 Ω 的、1 kΩ 的。

你当然可以给每种 ADC 列一遍节点方程。或者，你只要先求出电路的戴维南等效 $V_{th}$ 和 $R_{th}$，然后接不同的负载电阻 $R_L$ 时，只需要一个公式：

$$
V_L = V_{th} \times \frac{R_L}{R_{th} + R_L}
$$

$$
I_L = \frac{V_{th}}{R_{th} + R_L}
$$

这就是戴维南定理最大的实用价值：**一次等效，终身受用**。不管换多少种负载，你都不需要重新分析原电路。

### 6b. 最大功率传输（预习）

有一个经典问题：给定一个戴维南等效电路（$V_{th}$ 和 $R_{th}$ 固定），接一个可变电阻 $R_L$ 作为负载。$R_L$ 多大时负载获得的功率最大？

答案是：**当 $R_L = R_{th}$ 时，负载获得最大功率。**

推导留到下一章。但你现在就能看出为什么戴维南等效这么有用：它把一个"在复杂电路中求最佳负载"的问题，简化成了一个"两个电阻分功率"的初中数学题。

$$
P_{L} = I^2 R_L = \left(\frac{V_{th}}{R_{th} + R_L}\right)^2 R_L
$$

对这个式子求极值（令 $dP_L/dR_L = 0$），就得到 $R_L = R_{th}$。

这在音响、射频电路、太阳能电池中都是核心设计原则。下一章会详细展开。

### 6c. 电路分级分析（模块化思维）

大型电路不会一次分析完。工程师的处理方式是"分而治之"：

1. 把电路拆成几个模块（比如放大器模块、滤波模块、功率输出模块）
2. 把每个模块对其"下游"电路做戴维南（或诺顿）等效
3. 用等效后的简化模块连接起来，做系统级分析

这就是"黑盒思维"的工程实践。每个模块对外只暴露两个参数：$V_{th}$ 和 $R_{th}$。内部怎么实现的，可以单独设计和优化，不影响系统级分析。

> 举个例子：你在设计一个 Arduino 项目，需要驱动一个电机。Arduino 输出引脚对你来说就是一个戴维南等效：大约 5 V 的开路电压（$V_{th}$），大约几十欧姆的内阻（$R_{th}$）。你不需要知道 Arduino 芯片内部有几十万个晶体管怎么连的。你只需要知道这两个数，就能算出能驱动多大的电机。

## 7. Python：自动化戴维南等效计算与可视化

以下代码演示如何用 Python 自动完成戴维南等效的计算，并画出验证图。

### 7a. 电路描述

我们用一个具体电路：$V_{s1} = 10\text{ V}$，$R_1 = 2\text{ Ω}$，$R_2 = 3\text{ Ω}$，$R_3 = 6\text{ Ω}$。

电路连接方式：$V_{s1}$ 正极接 $R_1$，$R_1$ 另一端接节点 d。d 点引出 $R_2$（另一端接 $V_{s1}$ 负极，即 b 点）和 $R_3$（另一端接 a 点）。负载接 a、b 之间。

```
         R1 (2Ω)    R3 (6Ω)
    ────[~~~]───┬────[~~~]──── a
    │           │              │
   (+)          │              │
    │  Vs1=10V [ ] R2 (3Ω)    │  (负载 RL, 先移除)
    │           │              │
    └───────────┴────────────── b
```

### 7b. Python 代码

```python
# ============================================================
# 戴维南等效计算器
# 功能：对给定电路自动求 Vth 和 Rth，并画验证图
# ============================================================

import numpy as np
import matplotlib.pyplot as plt

# 设置 matplotlib 中文显示
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ============================================================
# 第一步：定义电路参数
# ============================================================
Vs1 = 10.0      # 电压源 (V)
R1, R2, R3 = 2.0, 3.0, 6.0  # 电阻 (Ω)

# ============================================================
# 第二步：求开路电压 Voc (即 Vth)
# ============================================================

# 设 b 点为参考 (0V)，设 d 点电压为 vd
# 电路方程 (KCL at d, a 开路所以 R3 上无电流):
#   (Vs1 - vd)/R1  =  (vd - 0)/R2
#   注意 Vs1 下端接 b = 0，所以 Vs1 上端 = 10V
#   流过 R1 的电流方向: 从左(Vs1 上端)到右(d 点)
#   流过 R2 的电流方向: 从上(d 点)到下(b 点)

# 解 KCL 方程:
# (10 - vd)/2 = vd/3
# 两边乘 6:  3(10 - vd) = 2vd
#            30 - 3vd = 2vd
#            30 = 5vd
#            vd = 6

vd = (Vs1 / R1) / (1/R1 + 1/R2)  # 电压分压公式 vd = Vs1 * (R2/(R1+R2))
# 验证: vd = 10 * 3/(2+3) = 10 * 3/5 = 6 V ✔
print(f"节点 d 的电压 vd = {vd:.4f} V")

# a 开路, R3 上无电流, a 点电压 = d 点电压
Voc = vd  # 开路电压即戴维南电压
Vth = Voc
print(f"戴维南电压 Vth = Voc = {Vth:.4f} V")

# ============================================================
# 第三步：求等效电阻 Rth (独立源置零法)
# ============================================================

# Vs1 短路后, R1 左端和 R2 下端都接地(b 点)
# 从 a 点看进去: R3 串联 (R1 并联 R2)
# R1 // R2 = (R1 * R2) / (R1 + R2)
R12_parallel = (R1 * R2) / (R1 + R2)
Rth = R3 + R12_parallel
print(f"R1 // R2 = {R12_parallel:.4f} Ω")
print(f"戴维南电阻 Rth = R3 + (R1//R2) = {Rth:.4f} Ω")

# ============================================================
# 第四步：用开路/短路法验证 Rth
# ============================================================

# 将 a、b 短路后求 Isc
# 短路后 a = b = 0, R3 两端电压 = vd - 0, 所以 i_R3 = vd/R3
# KCL at d: (10 - vd)/R1 = vd/R2 + vd/R3
# (10 - vd)/2 = vd/3 + vd/6
# 两边乘 6: 3(10 - vd) = 2vd + vd = 3vd
# 30 - 3vd = 3vd → 30 = 6vd → vd = 5

vd_sc = (Vs1 / R1) / (1/R1 + 1/R2 + 1/R3)
Isc = vd_sc / R3  # 流过 R3 的电流，即短路电流
Rth_method2 = Voc / Isc
print(f"\n开路/短路法验证:")
print(f"  短路时 vd = {vd_sc:.4f} V")
print(f"  短路电流 Isc = {Isc:.4f} A")
print(f"  Rth = Voc / Isc = {Rth_method2:.4f} Ω")

# ============================================================
# 第五步：验证等效电路对于不同负载的表现
# ============================================================

# 测试不同的负载电阻
RL_values = np.linspace(0.1, 50, 500)  # 从 0.1Ω 到 50Ω

# 原电路的负载电压 (用节点法精确解)
def original_VL(RL):
    """对于给定的 RL，计算原电路中负载两端的电压"""
    # KCL at d: (10-vd)/2 = vd/3 + (vd - vL)/6
    # KCL at a (vL): (vd - vL)/6 = vL/RL
    # 解这个方程组
    # 从 a 点方程: vd/R3 = vL/R3 + vL/RL → vd = vL + vL*R3/RL = vL(1 + R3/RL)
    # 代入 d 点:
    # Vs1/R1 - vd/R1 = vd/R2 + vL/R3
    # 整理得 vL 的解析解
    vL = Vs1 * RL / (R1 + RL + (R1 * RL / R2) + (R1 * R3 / R2))
    return vL

VL_original = np.array([original_VL(RL) for RL in RL_values])
VL_thevenin = Vth * RL_values / (Rth + RL_values)  # 戴维南等效的负载电压

# ============================================================
# 第六步：画图对比
# ============================================================

fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 图 1: 负载电压 vs 负载电阻
ax1 = axes[0]
ax1.plot(RL_values, VL_original, 'b-', linewidth=2, label='原电路（精确解）')
ax1.plot(RL_values, VL_thevenin, 'r--', linewidth=2, label='戴维南等效电路')
ax1.set_xlabel('负载电阻 $R_L$ (Ω)', fontsize=12)
ax1.set_ylabel('负载电压 $V_L$ (V)', fontsize=12)
ax1.set_title('负载电压对比：原电路 vs 戴维南等效', fontsize=13)
ax1.legend(fontsize=10)
ax1.grid(True, alpha=0.3)

# 图 2: 误差分析
ax2 = axes[1]
error = np.abs(VL_original - VL_thevenin)
ax2.semilogy(RL_values, error, 'k-', linewidth=1.5)
ax2.set_xlabel('负载电阻 $R_L$ (Ω)', fontsize=12)
ax2.set_ylabel('绝对误差 $|V_{原} - V_{等效}|$ (V)', fontsize=12)
ax2.set_title('等效误差分析（对数坐标）', fontsize=13)
ax2.grid(True, alpha=0.3)
# 标注最大误差
max_err_idx = np.argmax(error)
ax2.annotate(f'最大误差 ≈ {error[max_err_idx]:.2e} V',
             xy=(RL_values[max_err_idx], error[max_err_idx]),
             xytext=(RL_values[max_err_idx]+5, error[max_err_idx]*10),
             arrowprops=dict(arrowstyle='->', color='gray'),
             fontsize=9)

plt.tight_layout()
plt.savefig('thevenin_verification.png', dpi=150, bbox_inches='tight')
plt.show()

print(f"\n最大误差: {error.max():.6e} V  (数值计算精度, 理论上应为零)")
print(f"结论: 戴维南等效电路完美复现原电路的端口行为 ✓")
```

## 8. 完整实战：多源复杂电路的戴维南等效

**题目**：下面电路有 3 个独立源和 6 个电阻，求 a、b 端口的戴维南等效。

```
         R1 (1Ω)    R3 (2Ω)          R5 (3Ω)
    ────[~~~]───┬────[~~~]───┬───────[~~~]──── a
    │           │            │                 │
   (+)          │            │                 │
    │ Vs1=12V  [ ] R2 (4Ω)  [↑] Is=2A        [ ] R6 (5Ω)
    │           │            │                 │
    └───────────┤            │                 │
                │            │                 │
                │   Vs2=6V  │                 │
                ├──(+)───────┤                 │
                │   ─        │                 │
                │            │                 │
                └────────────┴───────────────── b
                        R4 (3Ω)
```

**电路解析**：设 b = 0 V。三个关键节点：节点 1（$R_1$ 右、$R_2$ 上、$R_3$ 左），节点 2（$R_2$ 下、$V_{s2}$ 上、$R_4$ 上、$I_s$ 下端），节点 3（$R_3$ 右、$R_4$ 下、$R_5$ 左、$I_s$ 上端）。$V_{s2}$ 固定节点 2 电压 = 6 V。$R_6$ 是内部电阻，端口 a、b 包含了它。

**求 $V_{oc}$**：a、b 开路时，$R_6$ 一端连 a（开路），另一端连 b（0 V）。实际上 $R_6$ 不影响开路电压，因为没有电流流经它。

开路时 a 点悬空，没有电流从 a 流出。所以 $R_5$ 上的电流 = 0（因为没有电流回路），$v_a = v_3$。

简化：$v_2 = 6\text{ V}$（被 $V_{s2}$ 固定）。

KCL at $v_1$：
$$
\frac{12 - v_1}{1} = \frac{v_1 - 6}{4} + \frac{v_1 - v_3}{2}
$$

KCL at $v_3$（流入 = 流出）：
$$
\frac{v_1 - v_3}{2} + 2 = \frac{v_3 - 6}{3}
$$
（$I_s = 2\text{ A}$ 从 $v_2 = 6\text{ V}$ 流向 $v_3$，是流入 $v_3$ 的电流；$R_4$ 电流从 $v_3$ 流出到 $v_2$；$R_5$ 上无电流因 a 开路。）

两边乘 6：
$$
3(v_1 - v_3) + 12 = 2(v_3 - 6)
$$
$$
3v_1 - 3v_3 + 12 = 2v_3 - 12
$$
$$
3v_1 + 12 + 12 = 5v_3
$$
$$
3v_1 + 24 = 5v_3
$$
$$
v_3 = \frac{3v_1 + 24}{5} = 0.6 v_1 + 4.8 \quad \text{(方程 A)}
$$

从方程 1（KCL at $v_1$）：
$$
12 - v_1 = \frac{v_1 - 6}{4} + \frac{v_1 - v_3}{2}
$$

两边乘 4：
$$
4(12 - v_1) = (v_1 - 6) + 2(v_1 - v_3)
$$
$$
48 - 4v_1 = v_1 - 6 + 2v_1 - 2v_3
$$
$$
48 - 4v_1 = 3v_1 - 6 - 2v_3
$$
$$
48 + 6 = 7v_1 - 2v_3
$$
$$
54 = 7v_1 - 2v_3
$$

代入 $v_3 = 0.6 v_1 + 4.8$：
$$
54 = 7v_1 - 2(0.6v_1 + 4.8)
$$
$$
54 = 7v_1 - 1.2v_1 - 9.6
$$
$$
54 = 5.8v_1 - 9.6
$$
$$
63.6 = 5.8v_1
$$
$$
v_1 = \frac{63.6}{5.8} = \frac{636}{58} = \frac{318}{29} \approx 10.966\text{ V}
$$

$$
v_3 = 0.6 \times \frac{318}{29} + 4.8 = \frac{190.8}{29} + \frac{139.2}{29} = \frac{330}{29} \approx 11.379\text{ V}
$$

开路时 $v_a = v_3$（$R_5$ 无电流）。所以：
$$
V_{oc} = v_a = v_3 = \frac{330}{29} \approx 11.379\text{ V}
$$

$$
V_{th} = \frac{330}{29}\text{ V}
$$

**求 $R_{th}$（用开路/短路法）。**

求 $I_{sc}$（a、b 短路）。

短路后 a = b = 0。$R_5$ 两端电压 = $v_3 - 0$，$R_6$ 被短路（两端都是 0）。$R_5$ 电流 = $v_3/3$。

KCL at $v_3$（短路后，流入 = 流出）：
$$
\frac{v_1 - v_3}{2} + 2 = \frac{v_3 - 6}{3} + \frac{v_3}{3}
$$

乘 6：
$$
3(v_1 - v_3) + 12 = 2(v_3 - 6) + 2v_3
$$
$$
3v_1 - 3v_3 + 12 = 2v_3 - 12 + 2v_3
$$
$$
3v_1 - 3v_3 + 12 = 4v_3 - 12
$$
$$
3v_1 + 24 = 7v_3
$$
$$
v_3 = \frac{3v_1 + 24}{7} \quad \text{(方程 B)}
$$

KCL at $v_1$ 不变（因为 $R_3$ 的连接没变）：
$$
\frac{12 - v_1}{1} = \frac{v_1 - 6}{4} + \frac{v_1 - v_3}{2}
$$

与之前一样：
$$
54 = 7v_1 - 2v_3
$$

代入 $v_3 = (3v_1 + 24)/7$：
$$
54 = 7v_1 - 2 \cdot \frac{3v_1 + 24}{7}
$$
$$
54 = 7v_1 - \frac{6v_1 + 48}{7}
$$

乘 7：
$$
378 = 49v_1 - 6v_1 - 48
$$
$$
378 = 43v_1 - 48
$$
$$
426 = 43v_1
$$
$$
v_1 = \frac{426}{43} \approx 9.907\text{ V}
$$

$$
v_3 = \frac{3 \times \frac{426}{43} + 24}{7} = \frac{\frac{1278}{43} + \frac{1032}{43}}{7} = \frac{\frac{2310}{43}}{7} = \frac{2310}{301} \approx 7.674\text{ V}
$$

短路电流 $I_{sc}$ = 流过短路线的电流 = 流过 $R_5$ 的电流：
$$
I_{sc} = \frac{v_3}{R_5} = \frac{2310/301}{3} = \frac{2310}{903} = \frac{770}{301} \approx 2.558\text{ A}
$$

所以：
$$
R_{th} = \frac{V_{oc}}{I_{sc}} = \frac{330/29}{770/301} = \frac{330}{29} \times \frac{301}{770}
$$

简化：$\frac{330}{770} = \frac{3}{7}$，所以：
$$
R_{th} = \frac{3}{7} \times \frac{301}{29} = \frac{903}{203} \approx 4.448\text{ Ω}
$$

这就是复杂电路的戴维南等效！$R_{th} = 903/203 \approx 4.45\text{ Ω}$。

## 9. 常见误区

### 误区 1：受控源可以置零

❌ "求 $R_{th}$ 时把所有源都关掉，包括受控源。"

✓ 只有**独立源**可以置零。受控源的电压/电流依赖于电路变量，它必须保留。把受控源置零相当于拆掉了电路的一部分，等效结果必然错误。

**为什么？** 受控源是"活的"，它的值随电路状态变化。把它置零等于假设它永远不工作，这改变了端口的行为。

### 误区 2：戴维南和诺顿的 $R$ 不相等

❌ "戴维南等效的 $R_{th}$ 和诺顿等效的 $R_n$ 不一样，要分别算。"

✓ **$R_{th} = R_n$，永远是同一个数。** 等效电阻不随等效形式而变。如果你用不同方法算出了两个不同的值，说明至少有一个步骤算错了。

### 误区 3：端口接什么会影响 $V_{th}$ 和 $R_{th}$

❌ "如果我在端口接不同的负载，$V_{th}$ 和 $R_{th}$ 也会变。"

✓ **$V_{th}$ 和 $R_{th}$ 只取决于电路内部的元件和结构，与端口外面接什么无关。** 这两个参数是电路的"固有属性"。接不同负载只是利用等效电路计算更方便，$V_{th}$ 和 $R_{th}$ 本身不变。

### 误区 4：忘记电流方向

❌ 求 $V_{oc}$ 和 $I_{sc}$ 时方向随心所欲。

✓ $V_{oc} = V_a - V_b$（a 相对于 b 的电压）。$I_{sc}$ 是从 a 经过短路线流向 b 的电流。这两个方向必须一致：如果 $V_{oc} > 0$（a 比 b 高），则 $I_{sc}$ 的方向是从 a 流向 b（电流从高电位流向低电位）。如果方向搞反，$R_{th} = V_{oc}/I_{sc}$ 会算出负数。

**求 $V_{oc}$**：移除 $R_2$（它是负载）。a、b 开路，没有电流。$R_1$ 上无压降，a 点电压 = $V_s$ 上端电压 = 15 V。b 点 = 0 V（$V_s$ 下端）。

$$
V_{th} = V_{oc} = 15\text{ V}
$$

**求 $R_{th}$**：独立源置零。$V_s$ 短路。从 a、b 看进去，$R_1$ 一端连 a（经过短路的 $V_s$），另一端连 b。

$$
R_{th} = R_1 = 5\text{ Ω}
$$

**等效电路**：15 V 电压源串联 5 Ω 电阻。

验证（接回 $R_2 = 10\text{ Ω}$）：
$$
I_{R2} = \frac{15}{5+10} = 1\text{ A}, \quad V_{R2} = 10\text{ V}
$$
原电路：$I_{原} = 15/(5+10) = 1\text{ A}$，$V_{原} = 10\text{ V}$。一致。

#### 题 4 解答

**求 $V_{oc}$**：a、b 开路。设 b = 0 V 参考点。

$R_3$ 在 $v_1$（$R_1$ 右端、$R_2$ 左端、$R_3$ 上端公共点）和 b 之间。

a 开路，无电流流过 $R_2$，所以 a 点电压 = $v_1$。

KCL at $v_1$：流过 $R_1$ 的电流 = 流过 $R_3$ 的电流（$R_2$ 无电流）

$$
\frac{20 - v_1}{4} = \frac{v_1}{6}
$$

乘 12：
$$
3(20 - v_1) = 2v_1
$$
$$
60 - 3v_1 = 2v_1
$$
$$
60 = 5v_1
$$
$$
v_1 = 12\text{ V}
$$

$$
V_{oc} = v_1 = 12\text{ V} = V_{th}
$$

**求 $R_{th}$**：独立源置零（$V_s$ 短路）。从 a 看：$R_2$ 串联（$R_3$ 并联 $R_1$）。

（因为 $V_s$ 短路后，$R_1$ 左端连 b = 0，右端连 $v_1$。$R_3$ 上端连 $v_1$，下端连 b。$R_1$ 和 $R_3$ 共享 $v_1$ 和 b 两端，所以并联。）

$$
R_1 \parallel R_3 = \frac{4 \times 6}{4+6} = \frac{24}{10} = 2.4\text{ Ω}
$$
$$
R_{th} = R_2 + 2.4 = 8 + 2.4 = 10.4\text{ Ω}
$$

**等效电路**：$V_{th} = 12\text{ V}$，$R_{th} = 10.4\text{ Ω}$。

#### 题 5 解答

**方法 1**（独立源置零）：已在题 4 中完成，$R_{th} = 10.4\text{ Ω}$。

**方法 2**（开路/短路法）：
$V_{oc} = 12\text{ V}$（已求）。

将 a、b 短路。a = b = 0。$R_2$ 两端电压 = $v_1 - 0 = v_1$，$i_{R2} = v_1/8$。

KCL at $v_1$：
$$
\frac{20 - v_1}{4} = \frac{v_1}{6} + \frac{v_1}{8}
$$

乘 24：
$$
6(20 - v_1) = 4v_1 + 3v_1 = 7v_1
$$
$$
120 - 6v_1 = 7v_1
$$
$$
120 = 13v_1
$$
$$
v_1 = \frac{120}{13} \approx 9.231\text{ V}
$$

短路电流 = 流过 $R_2$ 的电流 = $v_1/8 = (120/13)/8 = 15/13 \approx 1.154\text{ A}$。

$$
R_{th} = \frac{V_{oc}}{I_{sc}} = \frac{12}{15/13} = 12 \times \frac{13}{15} = \frac{156}{15} = 10.4\text{ Ω}
$$

一致。

**方法 3**（外加电源法）：
独立源 $V_s$ 短路。在 a、b 之间加测试电压 $V_t$，求 $I_t$。

短路 $V_s$ 后，$R_1$ 和 $R_3$ 并联 = 2.4 Ω，再和 $R_2$ 串联 = 10.4 Ω。

$I_t = V_t / 10.4$，$R_{th} = V_t / I_t = 10.4\text{ Ω}$。

三种方法一致。

#### 题 6 解答

含受控源，用开路/短路法。

**求 $V_{oc}$**（a、b 开路）：

设 b = 0 V。a 开路，无电流流过 $R_2$，所以 $V_{oc} = V_a$ = VCVS 上端电压。

流过 $R_1$ 的电流 $i_x$：
$$
i_x = \frac{12 - V_a}{3}
$$

VCVS 电压 = $0.5 i_x$，上端正极连 a，下端负极连 $R_2$ 上端（设该点为 c）。$R_2$ 下端连 b = 0。

VCVS 的电压关系：$V_a - V_c = 0.5 i_x$

$R_2$ 上电流 = 0（a 开路），所以 $V_c = 0$（下端连 b）。因此 $V_a - 0 = 0.5 i_x$，即 $V_a = 0.5 i_x$。

但 $i_x = (12 - V_a)/3$，所以：
$$
V_a = 0.5 \times \frac{12 - V_a}{3} = \frac{12 - V_a}{6}
$$
$$
6V_a = 12 - V_a
$$
$$
7V_a = 12
$$
$$
V_a = \frac{12}{7} \approx 1.714\text{ V}
$$

所以 $V_{oc} = V_a = 12/7\text{ V} \approx 1.714\text{ V}$。

**求 $I_{sc}$**（a、b 短路）：

a = b = 0。VCVS 上端 = a = 0。VCVS 下端 = c。VCVS 电压：$0 - V_c = 0.5 i_x$ → $V_c = -0.5 i_x$。

$R_2$ 在 c 和 0 之间：$i_{R2} = V_c / R_2 = -0.5 i_x / 2 = -0.25 i_x$。

a、b 短路后 a = b = 0，$R_1$ 右端直接接地，所以 $i_x = (12 - 0)/3 = 4\text{ A}$（$R_1$ 上的电流完全由 $V_s$ 和 $R_1$ 决定，不受 VCVS 影响）。

$V_c = -0.5 i_x = -0.5 \times 4 = -2\text{ V}$。

$i_{R2} = V_c / 2 = -2/2 = -1\text{ A}$（实际方向从 b 流向 c，即向上）。

在 c 节点 KCL：流入 c 的电流只有从 VCVS 来的 $i_{vcvs}$ 和从 $R_2$ 来的 $i_{R2}$（实际从 b 经 $R_2$ 流入 c）。流出 c 的电流为 0（无其他连接）。所以 $i_{vcvs} + i_{R2} = 0$ → $i_{vcvs} = -i_{R2} = 1\text{ A}$。

在 a 节点 KCL：$R_1$ 电流 $i_{R1} = 4\text{ A}$ 流入 a，VCVS 电流 $i_{vcvs} = 1\text{ A}$ 从 a 流出（进入 VCVS 上端），短路电流 $I_{sc}$ 从 a 流出。所以：

$$
4 = 1 + I_{sc}
$$
$$
I_{sc} = 3\text{ A}
$$

所以：
$$
R_{th} = \frac{V_{oc}}{I_{sc}} = \frac{12/7}{3} = \frac{12}{21} = \frac{4}{7} \approx 0.571\text{ Ω}
$$

**等效电路**：$V_{th} = 12/7\text{ V}$，$R_{th} = 4/7\text{ Ω}$。

#### 题 8 解答

对两次测试分别列分压方程：

测试 A（$R_L = 5\text{ Ω}$，$V_L = 10\text{ V}$）：
$$
10 = V_{th} \times \frac{5}{R_{th} + 5} \quad \text{(1)}
$$

测试 B（$R_L = 15\text{ Ω}$，$V_L = 15\text{ V}$）：
$$
15 = V_{th} \times \frac{15}{R_{th} + 15} \quad \text{(2)}
$$

从 (1)：$V_{th} = 10 \times \frac{R_{th} + 5}{5} = 2(R_{th} + 5)$

从 (2)：$V_{th} = 15 \times \frac{R_{th} + 15}{15} = R_{th} + 15$

联立：
$$
2(R_{th} + 5) = R_{th} + 15
$$
$$
2R_{th} + 10 = R_{th} + 15
$$
$$
R_{th} = 5\text{ Ω}
$$

代入 (2)：$V_{th} = 5 + 15 = 20\text{ V}$

验证：
- $R_L = 5\text{ Ω}$：$V_L = 20 \times 5/(5+5) = 20 \times 5/10 = 10\text{ V}$ ✓
- $R_L = 15\text{ Ω}$：$V_L = 20 \times 15/(5+15) = 20 \times 15/20 = 15\text{ V}$ ✓

**等效电路**：$V_{th} = 20\text{ V}$，$R_{th} = 5\text{ Ω}$。

> **本章小结**：戴维南定理和诺顿定理是电路分析的"瑞士军刀"。任何一个复杂的线性电路都可以浓缩成"一个电源 + 一个电阻"的极简模型。$V_{th}$ 是开路电压，$R_{th}$ 是等效内阻。求 $R_{th}$ 有三种方法：独立源置零（最直观）、开路/短路法（最通用）、外加电源法（含受控源必备）。诺顿等效只是戴维南等效的"并联版本"，两者通过源变换自由切换。学完这一章，你已经拥有了化简任意电路的能力。下一章将用这个工具解决一个经典问题：什么时候负载能获得最大功率？

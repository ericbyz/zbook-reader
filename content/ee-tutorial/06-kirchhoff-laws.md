# 第6章：基尔霍夫定律

> **核心问题**：复杂电路中，电流和电压遵循什么"交通规则"？

## 0. 本章导览

前面几章你已经掌握了电路的基本元素：电压、电流、电阻、功率，以及串联和并联。这些够你分析简单的电路，比如一个电池带两个串联的灯泡。但现实中的电路复杂得多：一个芯片上有几十亿个晶体管，一个电源分配网络里有几十个分支。你不可能把每一个都拆成"串联或并联"来分析。

基尔霍夫定律就是解决这个问题的"万能钥匙"。它只有两条规则，简单到你听完就能记住，但这两条规则可以分析**任何电路**，不管它有多复杂。它告诉你电流在节点处怎么分叉、电压在回路里怎么闭合，就像告诉你在一个复杂的城市路网里，车流和海拔变化必须遵守什么规律。掌握了它，你就有了分析一切直流电路的"通关密码"。

**前置知识**：你已学过电压、电流、电阻的基本定义（第2章）、欧姆定律 $V=IR$（第3章）、功率与能量（第4章）、串联与并联电路（第5章）。你需要会解简单的一元一次方程和二元一次方程组。

## 1. 生活例子：两条天下最简单的规则

在学习任何公式之前，先用两个你每天都能见到的场景来"安装"直觉。

### 1a. KCL = 十字路口的车流守恒

想象你站在一个十字路口正中央。车从四个方向来，又往四个方向去。直行的、左转的、右转的，各种车流在路口交汇又分开。

但有一件事永远不会发生：**没有一辆车会在路口正中央凭空冒出来，也没有一辆车会在路口正中央凭空消失。** 所有进入路口的车，必须从路口的某个出口离开。所有离开路口的车，一定是从路口的某个入口进来的。

你站在路口观察一分钟：

```
        ↑ 4 辆（北向出口）
        │
    ────┼────
        │
        
进入路口的车：东 5 辆 + 南 3 辆 + 西 2 辆 = 10 辆
离开路口的车：东 3 辆 + 北 4 辆 + 西 3 辆 = 10 辆

10 辆 = 10 辆 ✓ 没有车凭空消失
```

把"车"换成"电荷"，把"路口"换成"电路中的一个节点"，把"路口的不同方向"换成"连接到节点的不同导线"，这就是基尔霍夫电流定律（KCL）：

> **流进一个节点的总电荷量，一定等于流出这个节点的总电荷量。**

电荷不在节点处凭空产生，也不在节点处凭空消失。进来多少，出去多少。就这么简单。

### 1b. KVL = 爬山绕一圈回到原点

你去爬山。早上从山脚的停车场出发，向上爬了 300 米到半山腰的观景台，然后又向上爬了 200 米到山顶。在山顶休息一会儿，你从另一侧下山，一路下降了 500 米，最后回到了停车场。

你出发时海拔是多少？（假设停车场海拔 0 米）你回到停车场时海拔是多少？还是 0 米。

不管你走哪条路，不管你中间怎么绕、爬了多少坡、下了多少坡，只要你最后回到了出发点，**总的海拔变化一定是零。**

说详细点：上山过程海拔升了 $+300 + (+200) = +500$ 米，下山过程海拔降了 $-500$ 米，总变化 $= 500 + (-500) = 0$。

把"海拔"换成"电势（电压）"，把"爬山路径"换成"电路中的一个闭合回路"，这就是基尔霍夫电压定律（KVL）：

> **沿任意闭合回路走一圈，所有电压升和电压降加在一起，总和为零。**

电势是一个"位置函数"：同一个点只能有一个电势值。你从 A 点出发，沿着导线经过一堆元件，最后回到 A 点，电势必须回到原来的值。所以中途所有的升升降降，总效果一定是零。

## 2. 基尔霍夫电流定律 KCL

### 2a. 直观理解：水管的分叉和汇合

你已经很熟悉水管类比了：电压像水压，电流像水流。现在想象一根粗水管在某个接头处分叉成三根细水管：

```
    粗管（总进水管，每秒 10 升水流）
         │
    ┌────┼────┐
    │    │    │
   细管  细管  细管
   3L/s 4L/s 3L/s
```

不管三根细管多粗多细、延伸到什么地方，有一个事实无法改变：**粗管里流进来的水量（10 L/s），必须等于三根细管加起来流出去的水量（3 + 4 + 3 = 10 L/s）。**

水分子不能在管道的接头处"复制"自己，也不能在接头处"蒸发"。

把"水"换成"电荷"，把"水管接头"换成"电路节点"，这就是 KCL。

### 2b. 形式化定义

**基尔霍夫电流定律（Kirchhoff's Current Law，简称 KCL）**：

> 在电路的任何节点上，任意时刻流进该节点的电流之和，等于流出该节点的电流之和。

用数学公式表达（两种等价写法）：

**写法一（流入 = 流出）**：

$$
\Sigma I_{\text{进}} = \Sigma I_{\text{出}}
$$

**写法二（代数和为零）**：

$$
\Sigma I = 0
$$

第二种写法中，"代数和"的意思是：你把流进节点的电流记为**正**（$+$），流出节点的电流记为**负**（$-$），所有电流加在一起等于零。你反过来规定（流进为负、流出为正）也行，只要统一。

两种写法本质上是同一件事，只是"算账"的方式不同。写法一更直觉；写法二在写程序时更方便，因为你可以统一把所有电流放在等号左边。

### 2c. 物理根源：电荷守恒

KCL 不是一条"人为规定"，它背后的物理原理非常深刻：**电荷守恒定律**。

电荷守恒定律说：在一个孤立系统中，总电荷量永远不会改变。电荷不能被创造，也不能被消灭，只能从一个物体转移到另一个物体。

把这条定律用在电路节点上：节点就是一个导线连接点，它没有"仓库"可以存电荷，也没有"漏洞"可以让电荷漏掉。所以流进节点的电荷和流出节点的电荷必须在每一刻都严格相等。如果流进的比流出的多，节点里的电荷就会越积越多……但这不可能，因为节点没有存储能力。

> **KCL = 电荷守恒在电路节点上的数学表达。**

### 2d. 推广到闭合面

KCL 不仅可以对单个节点使用，还可以对**任意闭合面**使用。

什么叫闭合面？就像你用一个透明的泡泡把一个子电路整个"包起来"。在这个泡泡的表面上，有多少电流流进去，就有多少电流流出来。泡泡里面不管多复杂，表面的"总账"必须平。

```
    ╔══════ 闭合面 ══════╗
    ║                    ║
 I₁ ─┼─→  ┌─R₁──┬─R₂──┐  ║
    ║     │     │     │  ║
    ║     │     │     │  ║
 I₂ ─┼─→  └─────┴─────┘  ║
    ║                    ║
    ╚════════════════════╝
                │
                ↓ I₃

    对闭合面：I₁ + I₂ = I₃
```

这个推广在实际分析中非常有用。当你面对一个复杂的子电路，不想把它内部每个节点都分析一遍时，直接用一个闭合面包住它，只看表面的电流就行了。

### 2e. 手算：KCL 应用题

**题 1（三进两出）**：下图中，$I_1$、$I_2$、$I_3$ 流进节点，$I_4$、$I_5$ 流出节点。已知 $I_1 = 3\ \text{A}$，$I_2 = 4\ \text{A}$，$I_3 = 2\ \text{A}$，$I_4 = 5\ \text{A}$，求 $I_5$。

```
    I₁ → ───┐
    I₂ → ───┤
    I₃ → ───┤ ← 节点
            ├── → I₄
            └── → I₅
```

| 步骤 | 操作 | 结果 |
|------|------|------|
| ① | 列 KCL：流入 = 流出 | $I_1 + I_2 + I_3 = I_4 + I_5$ |
| ② | 代入已知值 | $3 + 4 + 2 = 5 + I_5$ |
| ③ | 合并左边 | $9 = 5 + I_5$ |
| ④ | 移项 | $I_5 = 9 - 5 = 4\ \text{A}$ |

**验证**：流入 $= 3 + 4 + 2 = 9\ \text{A}$，流出 $= 5 + 4 = 9\ \text{A}$。$9 = 9 \ \checkmark$

---

**题 2（方向未知）**：节点上有 $I_1 = 4\ \text{A}$（流进）、$I_2 = 7\ \text{A}$（流进）、$I_3 = 5\ \text{A}$（流出），还有 $I_4$ 方向未知。求 $I_4$ 的大小和方向。

| 步骤 | 操作 | 结果 |
|------|------|------|
| ① | 列 KCL：流入 = 流出 | $I_1 + I_2 = I_3 + I_4$ |
| ② | 代入已知值 | $4 + 7 = 5 + I_4$ |
| ③ | 计算 | $11 = 5 + I_4$ |
| ④ | 移项 | $I_4 = 6\ \text{A}$ |

$I_4$ 算出来是正数，且我们把它放在了"流出"一侧，所以 $I_4$ 实际方向是**流出**节点，大小为 $6\ \text{A}$。

如果用"代数和"版本验证（规定流进为正、流出为负，先假设 $I_4$ 流进）：

$$
+I_1 + I_2 + (-I_3) + I_4 = 0
$$

代入：$+4 + 7 + (-5) + I_4 = 0 \ \rightarrow\  6 + I_4 = 0 \ \rightarrow\  I_4 = -6\ \text{A}$。

负号说明：我假设 $I_4$ 流进，但实际方向是**流出**，大小为 $6\ \text{A}$。和上面结论完全一致。

---

**题 3（含电流源的节点）**：节点连接了四根导线和一个电流源。已知 $I_1 = 8\ \text{A}$（流进），$I_2 = 3\ \text{A}$（流出），电流源 $I_s = 2\ \text{A}$ 方向从节点流出。求 $I_3$。

| 步骤 | 操作 | 结果 |
|------|------|------|
| ① | 列 KCL：流入全部 = 流出全部 | $I_1 = I_2 + I_s + I_3$ |
| ② | 代入 | $8 = 3 + 2 + I_3$ |
| ③ | 计算 | $8 = 5 + I_3$ |
| ④ | 结果 | $I_3 = 3\ \text{A}$（流出） |

**验证**：流进 $8\ \text{A}$，流出 $3 + 2 + 3 = 8\ \text{A}$。$8 = 8 \ \checkmark$

---

**题 4（闭合面应用）**：下面一个子电路被闭合面包住。已知从外部流进闭合面的总电流 $I_{\text{in}} = 15\ \text{A}$，流出闭合面的导线中有 $I_1 = 6\ \text{A}$ 和 $I_2 = 4\ \text{A}$，第三根导线电流 $I_3$ 未知。求 $I_3$。

```
     I_in = 15A
         ↓
   ╔═════╪═════╗
   ║     │     ║  ← 闭合面（泡泡）
   ║  (复杂    ║
   ║   子电路) ║
   ║     │     ║
   ╚═════╪═════╝
      I₁=6A  I₂=4A  I₃=?
       ↓      ↓      ?
```

| 步骤 | 操作 | 结果 |
|------|------|------|
| ① | 对闭合面列 KCL | $I_{\text{in}} = I_1 + I_2 + I_3$ |
| ② | 代入 | $15 = 6 + 4 + I_3$ |
| ③ | 计算 | $15 = 10 + I_3$ |
| ④ | 结果 | $I_3 = 5\ \text{A}$（流出） |

不管闭合面里面的电路长什么样，你只需要看表面的电流进出。这就是闭合面 KCL 的威力。

### 2f. 常见误区

**误区 1：「电流方向必须按实际方向来标，标错了就算错了」**

❌ 做电路分析时，你完全可以**任意假设**电流的方向。算出来的结果是正数，说明你猜对了方向；结果是负数，说明实际方向和你的假设相反，但数值是对的。

✓ 大胆标参考方向！数量级不会错，只是方向可能需要反过来。负号不是"你错了"，而是"反了方向"。这是电路分析里最重要的心态。

**误区 2：「KCL 只对单个节点成立」**

❌ KCL 对任意闭合面同样成立。你可以把一整个子电路看成一个"黑盒子"，只关心盒子表面上的电流进出。在分析多层电路板或芯片内部的供电网络时，这种思路能省掉大量计算。

✓ KCL 的本质是电荷守恒。只要电荷不能在闭合面里凭空产生或消失，KCL 就一定成立，不管闭合面里包了多少个节点和元件。

### 2g. 应用连接

KCL 是分析一切并联电路和节点电路的基础。后面你学到三极管（BJT）时，三个电极的电流就满足 KCL：发射极电流 = 基极电流 + 集电极电流。学到运算放大器时，"虚断"（输入电流 ≈ 0）的概念也是基于 KCL。没有 KCL，你连最简单的电流分配都分析不了。

## 3. 基尔霍夫电压定律 KVL

### 3a. 直观理解：水的势能守恒

回到水管类比。你家楼顶有一个水塔（电池），水从水塔流下来，经过水轮机（负载）做功，然后被水泵重新打回水塔。在这个水循环系统里，有一个基本原则：

> **水每绕系统一圈，降低的能量（水轮机消耗的）必须等于增加的能量（水泵补充的）。**

如果不是这样，水的能量就会越来越多（变成永动机），或者越来越少（越来越没劲）。自然界不允许这种事。电路中完全一样：电荷从电源获得能量（电压升），带着能量走过电阻、灯泡等负载消耗能量（电压降），消耗完能量后回到电源负极。走完一圈，能量收支一定平衡。

### 3b. 形式化定义

**基尔霍夫电压定律（Kirchhoff's Voltage Law，简称 KVL）**：

> 沿电路中的任意闭合回路，所有元件上电压的代数和等于零。

用数学公式表达：

$$
\Sigma V = 0 \quad \text{（沿闭合回路）}
$$

等价地，也可以写成：

$$
\Sigma V_{\text{升}} = \Sigma V_{\text{降}}
$$

翻译：沿一个闭合回路走一圈，所有让你电势升高的元件带来的电压升之和，等于所有让你电势降低的元件带来的电压降之和。

### 3c. 物理根源：能量守恒

KVL 的物理根基是**能量守恒定律**：能量不能凭空产生，也不能凭空消失。

一个电荷携带一定的电势能，在回路中走一圈。如果它回到起点时净获得了能量（电压升 > 电压降），它就能不断加速，输出无限能量，这就是永动机了，不可能。反过来如果净损失了能量，电荷就会越来越"没劲"，最后停住，也不符合实际（实际电路中电流是持续稳定的）。

所以必须有升有降，收支平衡。绕一圈，总 $\Sigma V = 0$。

### 3d. 绕行方向与正负号约定

这是初学者最容易卡住的地方。但规则其实就只有几条，记住就不晕了。

**第 1 步**：选定绕行方向。顺时针也好，逆时针也好，你自己定，但一个回路从头到尾用一个方向。

**第 2 步**：沿绕行方向依次经过每个元件，判断它是让你电势升高还是降低。

具体判断规则：

**对于电阻**：
- 如果你走的绕行方向和电阻上标的电流参考方向**一致**：你是顺着电流走的，电势会降低。电压变化记作 **$-IR$**。
- 如果你走的绕行方向和电流参考方向**相反**：你是逆着电流走的，电势会升高。电压变化记作 **$+IR$**。

为什么？因为电流总是从高电势流向低电势。顺着电流走，就是从高走到低（降）；逆着电流走，就是从低走到高（升）。

**对于电压源（电池）**：
- 如果你从电池的 **$-$** 极进入、从 **$+$** 极出来：经过电池后电势升高了。记作 **$+V_s$**。
- 如果你从电池的 **$+$** 极进入、从 **$-$** 极出来：经过电池后电势降低了。记作 **$-V_s$**。

一个简单的口诀：**沿绕行方向，先碰到 $-$ 就是升，先碰到 $+$ 就是降。**

### 3e. 手算：KVL 应用题

**题 1（单回路，一个电源一个电阻）**：最简单的闭合回路，一个 $12\ \text{V}$ 电池和一个 $4\ \Omega$ 电阻串联。求回路中的电流 $I$。

```
        + 12V -
       ┌──┤├──┐
       │      │
       │      │
       │  R=4Ω│
       │      │
       └──────┘
```

设电流参考方向为顺时针，绕行方向也选顺时针。从电池负极出发：

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $12\ \text{V}$ 电池 | 从 $-$ 到 $+$，升 | $+12$ |
| 2 | $R = 4\ \Omega$ | 绕行方向同电流方向，降 | $-4I$ |

列 KVL 方程：

$$
+12 + (-4I) = 0
$$

$$
12 - 4I = 0 \ \rightarrow\  4I = 12 \ \rightarrow\  I = 3\ \text{A}
$$

用欧姆定律验证：$I = V / R = 12 / 4 = 3\ \text{A}$。一致 ✓

> **小技巧**：选起点的时候，从电池负极出发、沿绕行方向走，能让"升"先出现，不容易搞错正负号。

**题 2（单回路，两个电源两个电阻）**：回路中有两个电池（$V_1 = 10\ \text{V}$，$V_2 = 5\ \text{V}$）和两个电阻（$R_1 = 2\ \Omega$，$R_2 = 3\ \Omega$），排列如下。求回路电流 $I$。

```
        +  10V  -
       ┌──┤├──┬──┐
       │      │  │
    +  │      │  │  +
    R₁=2Ω   R₂=3Ω 5V
    -  │      │  │  -
       │      │  │
       └──────┴──┘
```

**第 1 步**：假设回路电流 $I$ 为顺时针。绕行方向也选顺时针。

**第 2 步**：从 $V_1$ 的负极出发，顺时针依次经过各元件：

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $V_1$（10 V） | 从 $-$ 到 $+$，升 | $+10$ |
| 2 | $R_1$（2 Ω） | 绕行方向同 $I$，降 | $-2I$ |
| 3 | $V_2$（5 V） | 从 $+$ 到 $-$（先遇正极），降 | $-5$ |
| 4 | $R_2$（3 Ω） | 绕行方向同 $I$，降 | $-3I$ |

**第 3 步**：列 KVL 方程：

$$
+10 + (-2I) + (-5) + (-3I) = 0
$$

**第 4 步**：化简：

$$
10 - 2I - 5 - 3I = 0
$$

$$
5 - 5I = 0
$$

**第 5 步**：解方程：

$$
I = 1\ \text{A}
$$

**第 6 步**：验证。"电压升 = 电压降"：

- 电压升：$V_1 = 10\ \text{V}$
- 电压降：$V_{R1} = 2 \times 1 = 2\ \text{V}$，$V_{R2} = 3 \times 1 = 3\ \text{V}$，$V_2 = 5\ \text{V}$
- 总降 $= 2 + 3 + 5 = 10\ \text{V} = V_1 \ \checkmark$

---

**题 3（电池反接）**：把题 2 中 $V_2$ 反接。其他不变。求 $I$。

```
        +  10V  -
       ┌──┤├──┬──┐
       │      │  │
    +  │      │  │  -
    R₁=2Ω   R₂=3Ω 5V
    -  │      │  │  +
       │      │  │
       └──────┴──┘
```

注意：$V_2$ 反接了，正极在右侧。绕行经过 $V_2$ 时，先遇到 $-$、再遇到 $+$，这是**升**！

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $V_1$（10 V） | 从 $-$ 到 $+$，升 | $+10$ |
| 2 | $R_1$（2 Ω） | 降 | $-2I$ |
| 3 | $V_2$（5 V 反接） | 从 $-$ 到 $+$，升 | $+5$ |
| 4 | $R_2$（3 Ω） | 降 | $-3I$ |

KVL 方程：

$$
+10 + (-2I) + (+5) + (-3I) = 0
$$

$$
15 - 5I = 0 \ \rightarrow\  I = 3\ \text{A}
$$

验证：电压升 $= 10 + 5 = 15\ \text{V}$，电压降 $= 2 \times 3 + 3 \times 3 = 6 + 9 = 15\ \text{V}$。$15 = 15 \ \checkmark$

> 注意：$V_2$ 反接后，两个电池"合力"推电流，总电压升从 10 V 变成了 15 V，电流从 1 A 跳到了 3 A。电池的方向极其重要！

---

**题 4（KVL 验证串联分压）**：$V_s = 12\ \text{V}$，$R_1 = 2\ \Omega$，$R_2 = 4\ \Omega$ 串联。$I = 2\ \text{A}$。验证 KVL。

| 步骤 | 操作 | 结果 |
|------|------|------|
| ① | 用欧姆定律求各电阻压降 | $V_{R1} = I \times R_1 = 2 \times 2 = 4\ \text{V}$ |
| ② | | $V_{R2} = I \times R_2 = 2 \times 4 = 8\ \text{V}$ |
| ③ | 验证 KVL | 电压升 $= 12\ \text{V}$，电压降 $= 4 + 8 = 12\ \text{V}$。$12 = 12 \ \checkmark$ |

### 3f. 常见误区

**误区 1：「绕行方向必须和电流方向一致」**

❌ 绕行方向可以任意选，顺时针、逆时针都行。它只是一个"记账"的工具，和电流的实际方向没有必然关系。如果你选的绕行方向和电流方向相反，经过电阻时就是电压升（$+IR$）而不是降（$-IR$）。

✓ 先定绕行方向，再根据绕行方向经过元件时的极性来判断升/降。不要被电流方向"绑架"了绕行方向。

**误区 2：「KVL 中电源永远是电压升」**

❌ 电源是不是升，取决于绕行方向经过电源时的极性顺序。如果绕行方向从电源的正极进、负极出，这个电源在这个回路里就是"降"而不是升（相当于被充电或被反接）。

✓ 不要死记"电源 = 升"。要逐一对每个元件判断：经过它时，电势是升高了还是降低了。

**误区 3：「KVL 只对简单回路成立，复杂回路不行」**

❌ KVL 对**任何一个**闭合回路都成立，不管是只有两个元件的简单回路，还是有几百个元件的复杂回路。你选哪一个回路都可以，只要路径是闭合的。

✓ KVL 是普适的。你可以绕大圈（整个外环），也可以绕小圈（局部的子回路），每个闭合路径的 $\Sigma V$ 都等于零。

### 3g. 应用连接

KVL 和 KCL 是分析一切电路的"双剑"。后面你学到的节点电压法、网孔电流法、戴维南定理、诺顿定理，全部都是 KCL 和 KVL 的"组合应用"。把 KCL 和 KVL 吃透了，后面的所有分析方法你一眼就能看出它们不过是 KCL/KVL 换了种写法而已。

## 4. 支路电流法（Branch Current Method）

### 4a. 直观理解：未知数太多怎么办？

前面我们用 KCL 和 KVL 解的题，都只有一个回路、一个未知电流。但现实中：
- 一个电路可能有 5 条支路、10 条支路，甚至更多
- 每条支路上的电流都是未知的
- 你不能一个个猜，得有一套系统的方法把所有未知电流一块儿解出来

**支路电流法**就是做这件事的：它告诉你，对于一个有 $b$ 条支路、$n$ 个节点的电路，你总能列出一个刚好够用的方程组，解出所有支路电流。

### 4b. 形式化定义

对于一个有 $b$ 条支路、$n$ 个节点的电路：

- **未知数**：$b$ 个支路电流（每条支路一个未知电流）
- **需要的方程总数**：$b$ 个
- **KCL 方程数**：$n-1$ 个（选 $n-1$ 个独立节点来列）
- **KVL 方程数**：$b - n + 1$ 个（选 $b-n+1$ 个独立回路来列）
- **总方程数**：$(n-1) + (b-n+1) = b$ 个，刚好够！

### 4c. 标准步骤

遇到任何电阻电路，按以下步骤操作：

| 步骤 | 操作内容 |
|------|---------|
| **第 1 步** | 数支路数和节点数。$b$ = 支路数，$n$ = 节点数 |
| **第 2 步** | 画出电路图，给每条支路标上电流符号和**参考方向**（箭头），如 $I_1, I_2, I_3, \dots$ |
| **第 3 步** | 选 $n-1$ 个节点，对每个节点列出 KCL 方程（流入 = 流出） |
| **第 4 步** | 选 $b-n+1$ 个独立回路（一般选网孔，即电路图中最小的"洞"），标上绕行方向 |
| **第 5 步** | 对每个回路列出 KVL 方程（$\Sigma V = 0$） |
| **第 6 步** | 联立所有方程，求解 $b$ 个未知电流 |
| **第 7 步** | 验证：把结果代回原方程，看是否满足 |

下面用三个完整的例题来把这个过程练透。

### 4d. 手算示例一：两电源三电阻电路

**题目**：电路如下图所示。$V_1 = 10\ \text{V}$，$V_2 = 6\ \text{V}$，$R_1 = 2\ \Omega$，$R_2 = 1\ \Omega$，$R_3 = 4\ \Omega$。求每条支路的电流 $I_1$、$I_2$、$I_3$。

```
         I₁ ↑        I₃ ↓        I₂ ↑
    ┌────R₁=2Ω──┬───R₃=4Ω───┬──R₂=1Ω────┐
    │           │           │           │
   +│           │           │          +│
   V₁=10V       │           │         V₂=6V
   -│           │           │          -│
    │           │           │           │
    └───────────┴───────────┴───────────┘
        下节点 D                 下节点 D
```

（上节点用 U 表示，左电源下方、中间电阻下方、右电源下方都是同一个节点 D。）

**第 1 步：数支路和节点。**

- 左支路（$V_1 + R_1$，但为了分析把 $V_1$ 和 $R_1$ 看成串联在同一条支路中）：电流 $I_1$
- 右支路（$V_2 + R_2$）：电流 $I_2$
- 中间支路（$R_3$）：电流 $I_3$
- 一共 $b = 3$ 条支路

- 上节点 U（三个电阻的交汇点）
- 下节点 D（三个电源的交汇点）
- $n = 2$ 个节点

**第 2 步：标参考方向。**

- $I_1$：方向设为从下往上（从 D 经过 $V_1$ 和 $R_1$ 到 U）
- $I_2$：方向设为从下往上（从 D 经过 $V_2$ 和 $R_2$ 到 U）
- $I_3$：方向设为从上往下（从 U 经过 $R_3$ 到 D）

**第 3 步：列 KCL 方程。**

$n-1 = 1$ 个 KCL 方程。选上节点 U：

$I_1$ 和 $I_2$ 流进 U，$I_3$ 流出 U：

$$
I_1 + I_2 = I_3 \tag{KCL}
$$

**第 4 步：选独立回路。**

$b - n + 1 = 3 - 2 + 1 = 2$ 个独立回路。选两个网孔：
- 左网孔：D $\to$ $V_1$ $\to$ $R_1$ $\to$ U $\to$ $R_3$ $\to$ D
- 右网孔：D $\to$ $R_3$ $\to$ U $\to$ $R_2$ $\to$ $V_2$ $\to$ D

绕行方向都选**顺时针**。

**第 5 步：列 KVL 方程。**

**左网孔（顺时针）**，从 D 出发：

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $V_1$（10 V） | 从 $-$ 到 $+$，升 | $+10$ |
| 2 | $R_1$（2 Ω） | 绕行方向 D→U，与 $I_1$ 方向（D→U）一致，降 | $-2I_1$ |
| 3 | $R_3$（4 Ω） | 绕行方向 U→D，与 $I_3$ 方向（U→D）一致，降 | $-4I_3$ |

$$
+10 + (-2I_1) + (-4I_3) = 0
$$

$$
10 - 2I_1 - 4I_3 = 0 \tag{KVL1}
$$

整理：$2I_1 + 4I_3 = 10$

**右网孔（顺时针）**，从 D 出发：

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $R_3$（4 Ω） | 绕行方向 D→U，与 $I_3$ 方向（U→D）**相反**，升 | $+4I_3$ |
| 2 | $R_2$（1 Ω） | 绕行方向 U→D，与 $I_2$ 方向（D→U）**相反**，升 | $+1 \cdot I_2 = +I_2$ |
| 3 | $V_2$（6 V） | 从 $+$ 到 $-$（U→D 方向经过 V₂ 时先遇正极），降 | $-6$ |

$$
(+4I_3) + (+I_2) + (-6) = 0
$$

$$
4I_3 + I_2 - 6 = 0 \tag{KVL2}
$$

整理：$I_2 + 4I_3 = 6$

**第 6 步：联立求解。**

方程组：

$$
\begin{cases}
I_1 + I_2 = I_3 & \text{(KCL)} \\
2I_1 + 4I_3 = 10 & \text{(KVL1)} \\
I_2 + 4I_3 = 6 & \text{(KVL2)}
\end{cases}
$$

从 KCL 得 $I_3 = I_1 + I_2$，代入 KVL1 和 KVL2：

代入 KVL1：

$$
2I_1 + 4(I_1 + I_2) = 10
$$

$$
2I_1 + 4I_1 + 4I_2 = 10
$$

$$
6I_1 + 4I_2 = 10 \tag{A}
$$

代入 KVL2：

$$
I_2 + 4(I_1 + I_2) = 6
$$

$$
I_2 + 4I_1 + 4I_2 = 6
$$

$$
4I_1 + 5I_2 = 6 \tag{B}
$$

现在有二元一次方程组：

$$
\begin{cases}
6I_1 + 4I_2 = 10 \\
4I_1 + 5I_2 = 6
\end{cases}
$$

**消元法**：(A)×2，(B)×3：

(A)×2：$12I_1 + 8I_2 = 20$

(B)×3：$12I_1 + 15I_2 = 18$

两式相减（上减下）：

$$
(12I_1 - 12I_1) + (8I_2 - 15I_2) = 20 - 18
$$

$$
-7I_2 = 2
$$

$$
I_2 = -\frac{2}{7} \approx -0.286\ \text{A}
$$

负号！说明 $I_2$ 的实际方向与参考方向（从下往上）**相反**，即实际电流是从上往下流的。

代回 (A)：

$$
6I_1 + 4\left(-\frac{2}{7}\right) = 10
$$

$$
6I_1 - \frac{8}{7} = 10
$$

$$
6I_1 = 10 + \frac{8}{7} = \frac{70}{7} + \frac{8}{7} = \frac{78}{7}
$$

$$
I_1 = \frac{78}{7 \times 6} = \frac{78}{42} = \frac{13}{7} \approx 1.857\ \text{A}
$$

求 $I_3$：

$$
I_3 = I_1 + I_2 = \frac{13}{7} + \left(-\frac{2}{7}\right) = \frac{11}{7} \approx 1.571\ \text{A}
$$

**最终结果**：

| 电流 | 数值 | 实际方向 |
|------|------|----------|
| $I_1$ | $\frac{13}{7} \approx 1.857\ \text{A}$ | 从下往上（与参考方向一致） |
| $I_2$ | $-\frac{2}{7} \approx -0.286\ \text{A}$ | 从上往下（与参考方向相反） |
| $I_3$ | $\frac{11}{7} \approx 1.571\ \text{A}$ | 从上往下（与参考方向一致） |

**第 7 步：验证。**

**验证 KCL（上节点 U）**：

$I_1$ 流进 U（从下往上），$I_2$ 实际方向是从上往下，即从 U **流出**。$I_3$ 从上往下，从 U **流出**。

流入 U：$I_1 = \frac{13}{7}$

流出 U：$|I_2| + I_3 = \frac{2}{7} + \frac{11}{7} = \frac{13}{7}$

流入 = 流出 $\checkmark$

**验证 KVL1（左网孔）**：

$$
10 - 2 \times \frac{13}{7} - 4 \times \frac{11}{7} = 10 - \frac{26}{7} - \frac{44}{7} = 10 - \frac{70}{7} = 10 - 10 = 0 \ \checkmark
$$

**验证 KVL2（右网孔）**：注意 $I_2$ 实际方向是向下（U→D），绕行经过 $R_2$ 时是从 U→D，与 $I_2$ 实际方向一致，是降。$I_2$ 在 $R_2$ 上的电压降 $= 1 \times \frac{2}{7} = \frac{2}{7}\ \text{V}$。$I_3$ 方向是 U→D，绕行经过 $R_3$ 是 D→U（与 $I_3$ 相反），是升。$I_3$ 在 $R_3$ 上的电压变化 $= +4 \times \frac{11}{7} = +\frac{44}{7}\ \text{V}$。

绕行经过各元件：D → $R_3$（升 $+\frac{44}{7}$）→ U → $R_2$（降 $-\frac{2}{7}$）→ $V_2$（从 + 到 −，降 $-6$）。

$$
+\frac{44}{7} - \frac{2}{7} - 6 = \frac{42}{7} - 6 = 6 - 6 = 0 \ \checkmark
$$

全部验证通过！

> **小结**：这个例题演示了支路电流法的完整流程。关键收获：**负电流不是错误，它告诉你实际方向和你的假设相反。** 你只需要放心大胆地设参考方向，算出什么就是什么。数值的绝对值永远是对的。

### 4e. 手算示例二：两电源四电阻电路（两网孔）

**题目**：电路如下。$V_1 = 20\ \text{V}$，$V_2 = 10\ \text{V}$，$R_1 = 5\ \Omega$，$R_2 = 10\ \Omega$，$R_3 = 2\ \Omega$，$R_4 = 8\ \Omega$。

```
        I₁→ R₁=5Ω   I₃↓        I₂→ R₂=10Ω
    A ────/\/\/\─── B ──────── C ────/\/\/\─── D
    │              │  R₃=2Ω    │              │
   +│              │          +│              │
   V₁=20V          │         V₂=10V           │ R₄=8Ω
   -│              │          -│              │
    │              │           │              │
    └──────────────┴───────────┴──────────────┘
    E              F           G              H
```

节点 E、F、G、H 都连接在一起，是同一个节点（地）。所以节点为：A、B、C、D、E(=F=G=H)。$n = 5$ 个节点。

支路：A-E（$V_1$）、A-B（$R_1$）、B-F（$R_3$）、B-C（导线，无元件，可看作 B 和 C 是同一点）、C-D（$R_2$）、C-G（$V_2$）、D-H（$R_4$）。等等，这个电路有点混乱。让我重新画一个更干净的两网孔电路。

---

**重新设计一个干净的两网孔电路**：

```
        I₁→ R₁=3Ω      I₂→ R₂=2Ω
    A ───/\/\/\─── B ───/\/\/\─── C
    │             │              │
   +│             │ I₃↓         +│
   V₁=24V        R₃=6Ω        V₂=12V
   -│             │             -│
    │             │              │
    └─────────────┴──────────────┘
    D             E              F
```

D、E、F 是同一节点（地）。所以 $n = 4$（A、B、C、D(=E=F)）。

支路：D-A（$V_1$）、A-B（$R_1$）、B-E（$R_3$）、B-C（$R_2$）、C-D（$V_2$）。$b = 5$ 条。

**第 1 步**：$b = 5$，$n = 4$。

**第 2 步**：标参考方向。

- $I_1$：A → B（经过 $R_1$）
- $I_2$：B → C（经过 $R_2$）
- $I_3$：B → E（经过 $R_3$）

电源支路电流用 KCL 在节点 A 和 C 处表示：
- 节点 A：$I_{V1} = I_1$（$V_1$ 支路电流 = $R_1$ 支路电流）
- 节点 C：$I_{V2} = I_2$（$V_2$ 支路电流 = $R_2$ 支路电流）

所以只需要 3 个未知数：$I_1$、$I_2$、$I_3$。

**第 3 步：KCL 方程**（$n-1 = 3$ 个）。

节点 A：$I_{V1} = I_1$ → 引入 $I_{V1}$，但可消去

节点 B：$I_1$ 流入，$I_2$ 流出，$I_3$ 流出

$$
I_1 = I_2 + I_3 \tag{KCL-B}
$$

节点 C：$I_2 = I_{V2}$（$I_2$ 流入，$I_{V2}$ 流出）→ 定义 $I_{V2}$

节点 D（地）：省略（冗余）。

有效 KCL 方程只有 1 个（对三个未知数）。

**第 4 步：KVL 方程**（$b-n+1 = 5-4+1 = 2$ 个）。

选两个网孔，顺时针绕行。

**左网孔**：D → $V_1$ → A → $R_1$ → B → $R_3$ → D

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $V_1$（24 V） | 从 $-$ 到 $+$，升 | $+24$ |
| 2 | $R_1$（3 Ω） | 绕行方向同 $I_1$，降 | $-3I_1$ |
| 3 | $R_3$（6 Ω） | 绕行方向同 $I_3$（都是 B→D 向下），降 | $-6I_3$ |

$$
24 - 3I_1 - 6I_3 = 0 \tag{KVL1}
$$

整理：$3I_1 + 6I_3 = 24$，即 $I_1 + 2I_3 = 8$

**右网孔**：D → $R_3$ → B → $R_2$ → C → $V_2$ → D

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $R_3$（6 Ω） | 绕行方向 D→B，与 $I_3$（B→D）**相反**，升 | $+6I_3$ |
| 2 | $R_2$（2 Ω） | 绕行方向 B→C，同 $I_2$，降 | $-2I_2$ |
| 3 | $V_2$（12 V） | 从 $+$ 到 $-$（先遇正极），降 | $-12$ |

$$
6I_3 - 2I_2 - 12 = 0 \tag{KVL2}
$$

整理：$-2I_2 + 6I_3 = 12$，即 $-I_2 + 3I_3 = 6$

**第 5 步：联立求解。**

方程组：

$$
\begin{cases}
I_1 = I_2 + I_3 & \text{(KCL)} \\
I_1 + 2I_3 = 8 & \text{(KVL1)} \\
-I_2 + 3I_3 = 6 & \text{(KVL2)}
\end{cases}
$$

从 KCL 代入 KVL1：

$$
(I_2 + I_3) + 2I_3 = 8
$$

$$
I_2 + 3I_3 = 8 \tag{A}
$$

KVL2：

$$
-I_2 + 3I_3 = 6 \tag{B}
$$

(A) + (B)：

$$
(I_2 - I_2) + (3I_3 + 3I_3) = 8 + 6
$$

$$
6I_3 = 14
$$

$$
I_3 = \frac{14}{6} = \frac{7}{3} \approx 2.333\ \text{A}
$$

代回 (B)：

$$
-I_2 + 3 \times \frac{7}{3} = 6
$$

$$
-I_2 + 7 = 6
$$

$$
I_2 = 1\ \text{A}
$$

代回 KCL：

$$
I_1 = 1 + \frac{7}{3} = \frac{10}{3} \approx 3.333\ \text{A}
$$

**结果**：

| 电流 | 数值 | 方向 |
|------|------|------|
| $I_1$ | $\frac{10}{3} \approx 3.333\ \text{A}$ | A → B |
| $I_2$ | $1\ \text{A}$ | B → C |
| $I_3$ | $\frac{7}{3} \approx 2.333\ \text{A}$ | B → D |

**验证 KVL1**：$24 - 3 \times \frac{10}{3} - 6 \times \frac{7}{3} = 24 - 10 - 14 = 0 \ \checkmark$

**验证 KVL2**：$6 \times \frac{7}{3} - 2 \times 1 - 12 = 14 - 2 - 12 = 0 \ \checkmark$

**验证 KCL（节点 B）**：$I_1 = \frac{10}{3}$，$I_2 + I_3 = 1 + \frac{7}{3} = \frac{10}{3}$。$\frac{10}{3} = \frac{10}{3} \ \checkmark$

全部通过！

### 4f. 手算示例三：含三个电源和五个电阻的复杂电路

**题目**：电路如下，含 2 个电压源、5 个电阻。下方还有一个附加电源（通过一个电阻连接到中间节点）。

```
        I₁→ R₁=3Ω    I₂→ R₂=2Ω
    A ───/\/\/\── B ───/\/\/\── C
    │             │             │
   +│        I₄↓  │ R₃=4Ω      │
   V₁=15V         │             │ R₄=5Ω
   -│             │             │
    │             │             │
    └─────────────┴─────────────┘
    D             E             F
                  │
                  │ R₅=1Ω
                  │
                 +│
                 V₂=10V
                 -│
                  │
                  G(=D=F 地)
```

D、F、G 是同一节点（地）。E 是中间节点。

**第 1 步**：数支路和节点。

节点：A、B、C、D(=F=G)、E → $n = 5$

支路：D-A（$V_1$）、A-B（$R_1$）、B-C（$R_2$）、C-D（$R_4$）、B-E（$R_3$）、E-D（$R_5 + V_2$）→ $b = 6$

**第 2 步**：标参考方向。

- $I_1$：A → B（经过 $R_1$）
- $I_2$：B → C（经过 $R_2$）
- $I_3$：B → E（经过 $R_3$）
- $I_4$：C → D（经过 $R_4$）
- $I_5$：E → D（经过 $R_5 + V_2$）

**第 3 步：KCL 方程**（$n-1 = 4$ 个）。

节点 A：$I_{V1} = I_1$（仅定义 $I_{V1}$）

节点 B：$I_1$ 流入，$I_2$ 和 $I_3$ 流出

$$
I_1 = I_2 + I_3 \tag{KCL-B}
$$

节点 C：$I_2$ 流入，$I_4$ 流出

$$
I_2 = I_4 \tag{KCL-C}
$$

节点 E：$I_3$ 流入（从 B 经 $R_3$），$I_5$ 流出（经 $R_5+V_2$ 到 D）

$$
I_3 = I_5 \tag{KCL-E}
$$

节点 D（地）：省略（冗余）。

从 KCL 得出：$I_4 = I_2$，$I_5 = I_3$。独立的未知数只有 $I_1$、$I_2$、$I_3$ 三个。

**第 4 步：KVL 方程**（$b-n+1 = 6-5+1 = 2$ 个）。

选两个网孔，顺时针绕行。

**网孔 1（左网孔）**：D → $V_1$ → A → $R_1$ → B → $R_3$ → E → ($R_5 + V_2$) → D

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $V_1$（15 V） | 从 $-$ 到 $+$，升 | $+15$ |
| 2 | $R_1$（3 Ω） | 同 $I_1$ 方向，降 | $-3I_1$ |
| 3 | $R_3$（4 Ω） | 同 $I_3$ 方向，降 | $-4I_3$ |
| 4 | $R_5$（1 Ω） | 同 $I_5$ 方向（$I_5 = I_3$），降 | $-1 \cdot I_3 = -I_3$ |
| 5 | $V_2$（10 V） | 从 $-$ 到 $+$（$V_2$ 负极在上、正极在下，绕行 E→D 方向先遇负极），升 | $+10$ |

$$
+15 + (-3I_1) + (-4I_3) + (-I_3) + (+10) = 0
$$

$$
25 - 3I_1 - 5I_3 = 0 \tag{KVL1}
$$

整理：$3I_1 + 5I_3 = 25$

**网孔 2（右网孔）**：E → B → $R_2$ → C → $R_4$ → D → ($R_5+V_2$) → E

注意：这个网孔从 E 出发，顺时针走。但经过 $R_3$ 是从 E→B，与 $I_3$ 方向（B→E）相反。

| 顺序 | 经过元件 | 判断 | 电压变化 |
|------|---------|------|----------|
| 1 | $R_3$（4 Ω） | 绕行 E→B，与 $I_3$（B→E）**相反**，升 | $+4I_3$ |
| 2 | $R_2$（2 Ω） | 绕行 B→C，同 $I_2$，降 | $-2I_2$ |
| 3 | $R_4$（5 Ω） | 绕行 C→D，同 $I_4$（$=I_2$），降 | $-5I_2$ |
| 4 | $R_5$（1 Ω） | 绕行 D→E，与 $I_5$（E→D，$I_5 = I_3$）**相反**，升 | $+1 \cdot I_3 = +I_3$ |
| 5 | $V_2$（10 V） | 绕行 D→E 经过 V₂ 从 $+$ 到 $-$（先遇正极），降 | $-10$ |

$$
+4I_3 + (-2I_2) + (-5I_2) + (+I_3) + (-10) = 0
$$

$$
4I_3 - 7I_2 + I_3 - 10 = 0
$$

$$
-7I_2 + 5I_3 - 10 = 0 \tag{KVL2}
$$

整理：$-7I_2 + 5I_3 = 10$

**第 5 步：联立求解。**

方程组：

$$
\begin{cases}
I_1 = I_2 + I_3 & \text{(KCL)} \\
3I_1 + 5I_3 = 25 & \text{(KVL1)} \\
-7I_2 + 5I_3 = 10 & \text{(KVL2)}
\end{cases}
$$

从 KCL：$I_1 = I_2 + I_3$，代入 KVL1：

$$
3(I_2 + I_3) + 5I_3 = 25
$$

$$
3I_2 + 3I_3 + 5I_3 = 25
$$

$$
3I_2 + 8I_3 = 25 \tag{A}
$$

KVL2：

$$
-7I_2 + 5I_3 = 10 \tag{B}
$$

从 (B) 解 $I_3$：

$$
5I_3 = 7I_2 + 10
$$

$$
I_3 = \frac{7}{5}I_2 + 2
$$

代入 (A)：

$$
3I_2 + 8\left(\frac{7}{5}I_2 + 2\right) = 25
$$

$$
3I_2 + \frac{56}{5}I_2 + 16 = 25
$$

$$
\left(3 + \frac{56}{5}\right)I_2 = 9
$$

$$
\frac{15 + 56}{5}I_2 = 9
$$

$$
\frac{71}{5}I_2 = 9
$$

$$
I_2 = \frac{45}{71} \approx 0.634\ \text{A}
$$

求 $I_3$：

$$
I_3 = \frac{7}{5} \times \frac{45}{71} + 2 = \frac{315}{355} + 2 = \frac{63}{71} + \frac{142}{71} = \frac{205}{71} \approx 2.887\ \text{A}
$$

求 $I_1$：

$$
I_1 = I_2 + I_3 = \frac{45}{71} + \frac{205}{71} = \frac{250}{71} \approx 3.521\ \text{A}
$$

**最终结果**：

| 电流 | 数值 | 实际方向 |
|------|------|----------|
| $I_1$ | $\frac{250}{71} \approx 3.521\ \text{A}$ | A → B |
| $I_2$ | $\frac{45}{71} \approx 0.634\ \text{A}$ | B → C |
| $I_3$ | $\frac{205}{71} \approx 2.887\ \text{A}$ | B → E |
| $I_4$ | $= I_2 = 0.634\ \text{A}$ | C → D |
| $I_5$ | $= I_3 = 2.887\ \text{A}$ | E → D |

**验证 KVL1**：

$$
25 - 3 \times \frac{250}{71} - 5 \times \frac{205}{71} = 25 - \frac{750}{71} - \frac{1025}{71} = 25 - \frac{1775}{71} = 25 - 25 = 0 \ \checkmark
$$

**验证 KVL2**：

$$
-7 \times \frac{45}{71} + 5 \times \frac{205}{71} = -\frac{315}{71} + \frac{1025}{71} = \frac{710}{71} = 10 \ \checkmark
$$

**验证 KCL（节点 B）**：$I_1 = \frac{250}{71}$，$I_2 + I_3 = \frac{45}{71} + \frac{205}{71} = \frac{250}{71} \ \checkmark$

全部通过！

> **小结**：这个例题有 2 个电压源、5 个电阻、5 个节点、6 条支路。用支路电流法，我们列了 4 个 KCL 和 2 个 KVL，通过 KCL 把未知数从 6 个压缩到 3 个，然后用 KVL 联立求解。整个过程非常有条理，没有任何"猜"的成分。

## 5. 独立方程的选择

### 5a. 直观理解：为什么 $n-1$ 个 KCL 就够了？

想象一个水管网络，有 $n$ 个接头（节点）。每个接头上，你都能写一个"流进 = 流出"的方程。但最后一个接头的方程是"多余"的，它已经隐含在前面 $n-1$ 个方程里了。

为什么？因为**所有节点的流入总量等于流出总量**。如果你已经确认了前 $n-1$ 个节点上"流进 = 流出"，那么所有水流的总账已经结清了，第 $n$ 个节点自动平衡。再写它的方程不会提供任何新信息。

用数学语言说：$n$ 个 KCL 方程里，只有 $n-1$ 个是线性独立的。第 $n$ 个可以由前 $n-1$ 个推导出来。

### 5b. 独立 KCL 方程：选哪些节点？

规则极其简单：**任选 $n-1$ 个节点。** 选哪 $n-1$ 个都可以，结果一样。一般选最方便写方程的那 $n-1$ 个。

比如前面 5 节点电路（A、B、C、D、E），我选了 A、B、C、E 写 KCL，省略了 D。如果改选 B、C、D、E 写 KCL，省略 A，得到的最终结果完全一样（只是方程的写法不同）。

### 5c. 独立 KVL 方程：选哪些回路？

KVL 对**任何闭合回路**都成立。但如果你写的 KVL 方程太多，它们之间会线性相关（有些方程是其他方程的"组合"），浪费你的时间。

你需要选出恰好 $b-n+1$ 个**独立回路**。最常用的方法：**选网孔（Mesh）**。

网孔是电路图中最小的"洞"。把电路图画在纸上，网孔就是那些被导线围成的、内部没有其他导线的区域。每个网孔定义一个独立的 KVL 方程，这些方程之间不会互相包含。

```
    ┌──R₁──┬──R₂──┐
    │      │      │
    V₁     R₃     V₂
    │      │      │
    └──────┴──────┘
    
    网孔1（左洞）  网孔2（右洞）
```

两个网孔就是两个独立的 KVL 方程。你还可以列"外环"的 KVL（绕过最外面一圈），但外环方程 = 网孔1 + 网孔2，不是独立的。

**验证**：对于示例三的电路，

KVL1：$25 - 3I_1 - 5I_3 = 0$

KVL2：$-7I_2 + 5I_3 - 10 = 0$

外环（D → $V_1$ → A → $R_1$ → B → $R_2$ → C → $R_4$ → D）：

外环 KVL = $15 - 3I_1 - 2I_2 - 5I_4 = 15 - 3I_1 - 2I_2 - 5I_2 = 15 - 3I_1 - 7I_2 = 0$

将 KVL1 和 KVL2 相加：$(25 - 3I_1 - 5I_3) + (-7I_2 + 5I_3 - 10) = 15 - 3I_1 - 7I_2 = 0$

确实等于外环方程！所以外环方程是多余的。

> **规则总结**：对于一个可以画在纸上没有导线交叉的电路（平面电路），选所有网孔作为独立回路，个数恰好是 $b-n+1$。

## 6. 用矩阵方法解方程组（Python）

### 6a. 为什么需要矩阵？

前面两个未知数的方程组你可以手算。但如果有 5 个、10 个、甚至 100 个未知数呢？手算不仅慢，还极其容易出错。这时候你需要计算机。

所有线性方程组都可以写成矩阵形式：

$$
A\mathbf{x} = \mathbf{b}
$$

其中：
- $A$ 是系数矩阵（方程左边的系数，$n \times n$）
- $\mathbf{x}$ 是未知向量（你要解的未知数，$n \times 1$）
- $\mathbf{b}$ 是常数向量（方程右边的常数，$n \times 1$）

然后用 `numpy.linalg.solve(A, b)` 一行代码搞定。

### 6b. 从方程组到矩阵

以 4d 节的两电源三电阻例题为例，方程组是：

$$
\begin{cases}
I_1 + I_2 - I_3 = 0 \\
2I_1 + 4I_3 = 10 \\
I_2 + 4I_3 = 6
\end{cases}
$$

把所有未知数移到左边、常数移到右边（KCL 中 $I_1 + I_2 = I_3$ 变为 $I_1 + I_2 - I_3 = 0$）：

$$
\begin{bmatrix}
1 & 1 & -1 \\
2 & 0 & 4 \\
0 & 1 & 4
\end{bmatrix}
\begin{bmatrix}
I_1 \\ I_2 \\ I_3
\end{bmatrix}
=
\begin{bmatrix}
0 \\ 10 \\ 6
\end{bmatrix}
$$

- $A$ 的每一行对应一个方程，每一列对应一个未知数
- 第一行（KCL）：$I_1$ 系数 1，$I_2$ 系数 1，$I_3$ 系数 $-1$，右边常数 $0$
- 第二行（KVL1）：$I_1$ 系数 2，$I_2$ 系数 0，$I_3$ 系数 4，右边常数 $10$
- 第三行（KVL2）：$I_1$ 系数 0，$I_2$ 系数 1，$I_3$ 系数 4，右边常数 $6$

### 6c. Python 实现

```python
# 基尔霍夫定律：用矩阵方法求解电路方程组
import numpy as np

# 系数矩阵 A（每个方程左边的系数）
A = np.array([
    [1,  1, -1],   # KCL:  I1 + I2 - I3 = 0
    [2,  0,  4],   # KVL1: 2I1 + 0I2 + 4I3 = 10
    [0,  1,  4]    # KVL2: 0I1 + I2 + 4I3 = 6
])

# 常数向量 b（每个方程右边的值）
b = np.array([0, 10, 6])

# 求解 Ax = b
I = np.linalg.solve(A, b)

# 输出结果
print("各支路电流：")
print(f"  I1 = {I[0]:.4f} A")
print(f"  I2 = {I[1]:.4f} A")
print(f"  I3 = {I[2]:.4f} A")

# 验证：计算 A·I 应该等于 b
b_check = A @ I
print(f"\n验证 A·I = b:")
print(f"  方程1: {b_check[0]:.6f} (应 ≈ {b[0]})")
print(f"  方程2: {b_check[1]:.6f} (应 ≈ {b[1]})")
print(f"  方程3: {b_check[2]:.6f} (应 ≈ {b[2]})")
```

**预期输出**：

```
各支路电流：
  I1 = 1.8571 A
  I2 = -0.2857 A
  I3 = 1.5714 A

验证 A·I = b:
  方程1: 0.000000 (应 ≈ 0)
  方程2: 10.000000 (应 ≈ 10)
  方程3: 6.000000 (应 ≈ 6)
```

和手算结果 $\frac{13}{7}$、$-\frac{2}{7}$、$\frac{11}{7}$ 完全一致。

### 6d. 通用电路求解器

下面是一个更通用的函数，给定 $A$ 和 $\mathbf{b}$ 就能解出所有电流，并自动验证结果。

```python
# 通用线性电路求解器
import numpy as np

def solve_circuit(A, b, current_names=None):
    """
    求解电路线性方程组 Ax = b
    
    参数:
        A: 系数矩阵 (n×n)
        b: 常数向量 (n,)
        current_names: 电流变量名列表 (可选)
    
    返回:
        I: 解向量
    """
    # 求解
    I = np.linalg.solve(A, b)
    
    # 输出结果
    if current_names is None:
        current_names = [f"I{i+1}" for i in range(len(I))]
    
    print("=" * 50)
    print("电路分析结果")
    print("=" * 50)
    for name, val in zip(current_names, I):
        if val >= 0:
            print(f"  {name} = {val:+.4f} A  (与参考方向一致)")
        else:
            print(f"  {name} = {val:+.4f} A  (与参考方向相反)")
    
    # 验证
    residuals = A @ I - b
    max_error = np.max(np.abs(residuals))
    print(f"\n最大残差: {max_error:.2e}")
    if max_error < 1e-10:
        print("验证通过！所有方程均满足。")
    else:
        print("验证失败，请检查方程。")
    
    return I


# === 使用示例：解示例三的复杂电路 ===
A = np.array([
    [1, -1, -1,  0,  0],   # KCL-B:  I1 - I2 - I3 = 0
    [0,  1,  0, -1,  0],   # KCL-C:  I2 - I4 = 0
    [0,  0,  1,  0, -1],   # KCL-E:  I3 - I5 = 0
    [3,  0,  5,  0,  0],   # KVL1:  3I1 + 5I3 = 25
    [0, -2,  4, -5,  1]    # KVL2:  -2I2 + 4I3 -5I4 + I5 = 10
    # 注意：这里 KVL2 需要用 I4 和 I5 代入，完整形式涉及所有 5 个电流
    # 实际上 KVL2: -7I2 + 5I3 = 10 (I4=I2, I5=I3 代入后)
])
b = np.array([0, 0, 0, 25, 10])

names = ["I1", "I2", "I3", "I4", "I5"]
I = solve_circuit(A, b, names)
```

> **注意**：上面示例中的 KVL2 用了简化后的形式。实际使用时，你可以把压缩后的方程组（比如只有 3 个未知数 $I_1$、$I_2$、$I_3$）喂给求解器，也可以把完整的 5 元方程组（带 $I_4$、$I_5$）连同 KCL 关系一起写进去。两种方式结果一样，矩阵方法不挑。

**预期输出**：

```
==================================================
电路分析结果
==================================================
  I1 = +3.5211 A  (与参考方向一致)
  I2 = +0.6338 A  (与参考方向一致)
  I3 = +2.8873 A  (与参考方向一致)
  I4 = +0.6338 A  (与参考方向一致)
  I5 = +2.8873 A  (与参考方向一致)

最大残差: 1.42e-14
验证通过！所有方程均满足。
```

## 7. 完整实战：独立分析一个含多源多阻的复杂电路

### 7a. 题目

分析下面这个电路。它包含 3 个电压源、5 个电阻。请独立完成从识别节点和支路、列方程、求解到验证的全过程。

```
         I₁→ R₁=4Ω    I₂→ R₂=2Ω    I₃→ R₃=1Ω
    A ────/\/\/\── B ───/\/\/\── C ───/\/\/\── D
    │              │             │              │
   +│         I₆↓  │ R₄=3Ω      │ R₅=6Ω   I₄↓ +│
   V₁=30V          │             │              V₃=15V
   -│              │             │             -│
    │              │             │              │
    └──────────────┴─────────────┴──────────────┘
    E              F             G              H
                   │
                  +│
                  V₂=12V
                  -│
                   │
                   I(=E=H 地)
```

E、H、I 是同一节点（地），F、G 连接到 I（地）。所以：
- 节点：A、B、C、D、地（E=F=G=H=I）。$n = 5$
- 支路：E-A（$V_1$）、A-B（$R_1$）、B-F（$R_4$）、F-I（$V_2$）、B-C（$R_2$）、C-G（$R_5$）、C-D（$R_3$）、D-H（$V_3$）。$b = 8$

等等，$V_2$ 在 F-I 之间，$R_4$ 在 B-F 之间，所以 B-F-I 这条路径包含 $R_4$ 和 $V_2$。但如果 F 和 I 之间有 $V_2$，而且 F 和 I 连接……这实际上是一个支路 B → $R_4$ → F → $V_2$ → I（地）。这是一个串联支路，可以把 $R_4$ 和 $V_2$ 看作同一条支路。

同样，C-G-I 是 C → $R_5$ → G → I（地）。G 和 I 之间没有额外元件，$R_5$ 直接到地。

简化后支路：
1. E→A：$V_1$
2. A→B：$R_1$
3. B→I：$R_4 + V_2$（串联）
4. B→C：$R_2$
5. C→I：$R_5$
6. C→D：$R_3$
7. D→H：$V_3$

但 E 和 H 也是地……让我简化：E、F(=I?)、G(=I?)、H(=I)、I 都是地。那 $R_4$ 上端接 B，下端接 F。F 在那儿？如果 F 不接地，那 F 是一个悬空的节点。除非 $V_2$ 把它连到地。

好，我简化设计：B 的下方有 $R_4$ 串联 $V_2$ 到地。C 的下方有 $R_5$ 到地。

节点重新数：A、B、C、D、地 → $n = 5$。

支路：$V_1$（地→A）、$R_1$（A→B）、$(R_4+V_2)$（B→地）、$R_2$（B→C）、$R_5$（C→地）、$R_3$（C→D）、$V_3$（地→D，从地到 D 经过 V₃ 的正极到负极…不对，V₃ 负极接地，正极接 D）。所以 V₃ 支路是 地→D（经过 V₃ 的内部从 - 到 +）。

$b = 7$ 条支路。

KCL 方程：$n-1 = 4$ 个
KVL 方程：$b-n+1 = 7-5+1 = 3$ 个
总计：$4 + 3 = 7 = b \ \checkmark$

电流参考方向：
- $I_1$：A → B（经 $R_1$）
- $I_2$：B → C（经 $R_2$）
- $I_3$：C → D（经 $R_3$）

### 7c. 列 KCL 方程（$n-1 = 4$ 个）

**节点 A**：$I_{V1}$ 流入 A（从地来），$I_1$ 流出 A。$I_{V1} = I_1$。

**节点 B**：$I_1$ 流入，$I_2$ 和 $I_4$ 流出。

$$
I_1 = I_2 + I_4 \tag{KCL-B}
$$

**节点 C**：$I_2$ 流入（从 B），$I_3$ 和 $I_5$ 流出。

$$
I_2 = I_3 + I_5 \tag{KCL-C}
$$

**节点 D**：$I_3$ 流入（从 C），$I_{V3}$ 流出（到地）。$I_3 = I_{V3}$。

有效 KCL（电阻电流之间的关系）：KCL-B 和 KCL-C，共 2 个方程。

### 7d. 列 KVL 方程（$b-n+1 = 3$ 个）

选 3 个网孔，顺时针绕行。

**网孔 1（左）**：地 → $V_1$ → A → $R_1$ → B → ($R_4 + V_2$) → 地

| 顺序 | 元件 | 判断 | 变化 |
|------|------|------|------|
| 1 | $V_1$（30 V） | − 到 +，升 | $+30$ |
| 2 | $R_1$（4 Ω） | 同 $I_1$，降 | $-4I_1$ |
| 3 | $R_4$（3 Ω） | 同 $I_4$，降 | $-3I_4$ |
| 4 | $V_2$（12 V） | 绕行 B→地 经过 $V_2$ 从 + 到 −（B 端是正极），降 | $-12$ |

$$
30 - 4I_1 - 3I_4 - 12 = 0
$$

$$
18 - 4I_1 - 3I_4 = 0 \tag{KVL1}
$$

整理：$4I_1 + 3I_4 = 18$

**网孔 2（中）**：地 → ($R_4+V_2$) → B → $R_2$ → C → $R_5$ → 地

| 顺序 | 元件 | 判断 | 变化 |
|------|------|------|------|
| 1 | $R_4$（3 Ω） | 绕行 地→B，与 $I_4$（B→地）**相反**，升 | $+3I_4$ |
| 2 | $V_2$（12 V） | 绕行 地→B 经过 $V_2$ 从 − 到 +，升 | $+12$ |
| 3 | $R_2$（2 Ω） | 同 $I_2$，降 | $-2I_2$ |
| 4 | $R_5$（6 Ω） | 同 $I_5$，降 | $-6I_5$ |

$$
3I_4 + 12 - 2I_2 - 6I_5 = 0 \tag{KVL2}
$$

整理：$-2I_2 + 3I_4 - 6I_5 = -12$

**网孔 3（右）**：地 → $R_5$ → C → $R_3$ → D → $V_3$ → 地

| 顺序 | 元件 | 判断 | 变化 |
|------|------|------|------|
| 1 | $R_5$（6 Ω） | 绕行 地→C，与 $I_5$（C→地）**相反**，升 | $+6I_5$ |
| 2 | $R_3$（1 Ω） | 同 $I_3$，降 | $-1 \cdot I_3 = -I_3$ |
| 3 | $V_3$（15 V） | 绕行 D→地 经过 $V_3$ 从 + 到 −（D 端是正极），降 | $-15$ |

$$
6I_5 - I_3 - 15 = 0 \tag{KVL3}
$$

整理：$-I_3 + 6I_5 = 15$

### 7e. 联立求解

我们有 5 个未知数（$I_1, I_2, I_3, I_4, I_5$），5 个方程：

$$
\begin{cases}
I_1 = I_2 + I_4 & \text{(KCL-B)} \\
I_2 = I_3 + I_5 & \text{(KCL-C)} \\
4I_1 + 3I_4 = 18 & \text{(KVL1)} \\
-2I_2 + 3I_4 - 6I_5 = -12 & \text{(KVL2)} \\
-I_3 + 6I_5 = 15 & \text{(KVL3)}
\end{cases}
$$

用消元法逐步求解。

从 KCL-C：$I_2 = I_3 + I_5$

从 KCL-B：$I_1 = I_2 + I_4 = I_3 + I_5 + I_4$

代入 KVL1：

$$
4(I_3 + I_5 + I_4) + 3I_4 = 18
$$

$$
4I_3 + 4I_5 + 4I_4 + 3I_4 = 18
$$

$$
4I_3 + 4I_5 + 7I_4 = 18 \tag{A}
$$

从 KVL3：$I_3 = 6I_5 - 15$

代入 (A)：

$$
4(6I_5 - 15) + 4I_5 + 7I_4 = 18
$$

$$
24I_5 - 60 + 4I_5 + 7I_4 = 18
$$

$$
28I_5 + 7I_4 = 78
$$

$$
4I_5 + I_4 = \frac{78}{7} \tag{B}
$$

现在处理 KVL2。将 $I_2 = I_3 + I_5$ 代入：

$$
-2(I_3 + I_5) + 3I_4 - 6I_5 = -12
$$

$$
-2I_3 - 2I_5 + 3I_4 - 6I_5 = -12
$$

$$
-2I_3 - 8I_5 + 3I_4 = -12
$$

代入 $I_3 = 6I_5 - 15$：

$$
-2(6I_5 - 15) - 8I_5 + 3I_4 = -12
$$

$$
-12I_5 + 30 - 8I_5 + 3I_4 = -12
$$

$$
-20I_5 + 3I_4 = -42 \tag{C}
$$

现在有 (B) 和 (C) 两个方程：

$$
\begin{cases}
4I_5 + I_4 = \frac{78}{7} \\\\
-20I_5 + 3I_4 = -42
\end{cases}
$$

从 (B)：$I_4 = \frac{78}{7} - 4I_5$。代入 (C)：

$$
-20I_5 + 3\left(\frac{78}{7} - 4I_5\right) = -42
$$

$$
-20I_5 + \frac{234}{7} - 12I_5 = -42
$$

$$
-32I_5 = -42 - \frac{234}{7} = -\frac{294}{7} - \frac{234}{7} = -\frac{528}{7}
$$

$$
I_5 = \frac{528}{7 \times 32} = \frac{528}{224} = \frac{33}{14} \approx 2.357\ \text{A}
$$

求 $I_4$：

$$
I_4 = \frac{78}{7} - 4 \times \frac{33}{14} = \frac{78}{7} - \frac{132}{14} = \frac{156}{14} - \frac{132}{14} = \frac{24}{14} = \frac{12}{7} \approx 1.714\ \text{A}
$$

求 $I_3$：

$$
I_3 = 6I_5 - 15 = 6 \times \frac{33}{14} - 15 = \frac{198}{14} - 15 = \frac{99}{7} - \frac{105}{7} = -\frac{6}{7} \approx -0.857\ \text{A}
$$

负号！$I_3$ 的实际方向与参考方向（C→D）**相反**，即电流从 D 流向 C。

求 $I_2$：

$$
I_2 = I_3 + I_5 = -\frac{6}{7} + \frac{33}{14} = -\frac{12}{14} + \frac{33}{14} = \frac{21}{14} = \frac{3}{2} = 1.5\ \text{A}
$$

求 $I_1$：

$$
I_1 = I_2 + I_4 = \frac{3}{2} + \frac{12}{7} = \frac{21}{14} + \frac{24}{14} = \frac{45}{14} \approx 3.214\ \text{A}
$$

### 7f. 最终结果

| 电流 | 数值 | 实际方向 |
|------|------|----------|
| $I_1$ | $\frac{45}{14} \approx 3.214\ \text{A}$ | A → B |
| $I_2$ | $1.5\ \text{A}$ | B → C |
| $I_3$ | $-\frac{6}{7} \approx -0.857\ \text{A}$ | D → C（与参考相反） |
| $I_4$ | $\frac{12}{7} \approx 1.714\ \text{A}$ | B → 地 |
| $I_5$ | $\frac{33}{14} \approx 2.357\ \text{A}$ | C → 地 |

**验证 KVL1**：

$$
18 - 4 \times \frac{45}{14} - 3 \times \frac{12}{7} = 18 - \frac{180}{14} - \frac{36}{7} = 18 - \frac{90}{7} - \frac{36}{7}
$$

$$
= 18 - \frac{126}{7} = 18 - 18 = 0 \ \checkmark
$$

**验证 KVL2**（注意 $I_2$、$I_4$、$I_5$ 的实际方向）：

$$
3 \times \frac{12}{7} + 12 - 2 \times 1.5 - 6 \times \frac{33}{14} = \frac{36}{7} + 12 - 3 - \frac{198}{14}
$$

$$
= \frac{72}{14} + 12 - 3 - \frac{198}{14} = 9 - \frac{126}{14} = 9 - 9 = 0 \ \checkmark
$$

**验证 KVL3**（$I_3$ 实际从 D→C，绕行经过 $R_3$ 是 C→D，与 $I_3$ 实际方向相反，是升）：

$$
6 \times \frac{33}{14} - \left(-\frac{6}{7}\right) - 15 = \frac{198}{14} + \frac{6}{7} - 15 = \frac{99}{7} + \frac{6}{7} - 15
$$

$$
= \frac{105}{7} - 15 = 15 - 15 = 0 \ \checkmark
$$

全部通过！

## 8. 常见误区

**误区 1：KCL 和 KVL 混淆**

❌ 很多初学者会搞混：以为 KCL 是"电压的守恒"，KVL 是"电流的守恒"。这是完全反过来的！

✓ KCL（电流定律）管的是**节点**上的**电流**（流进 = 流出）。KVL（电压定律）管的是**回路**上的**电压**（升 = 降）。记住：**节**点和**电**流都带"节/电"这个音？没有好记的口诀。那就死记：C = Current（电流），V = Voltage（电压）。KCL 是电流定律，KVL 是电压定律。

**误区 2：参考方向设错了怎么办？**

❌ 初学者常常因为不确定电流方向而不敢下手标参考方向，怕标错了全盘算错。

✓ 任意标！算出来的结果是正的，说明你猜对了方向。算出来是负的，说明实际方向和你的假设相反。**数值的绝对值永远是对的。** 方向反了，把箭头反过来就行。这是电路分析中最基本的操作，不是错误。

**误区 3：方程数量凑不够或太多了**

❌ 有人在列方程时，把所有能列的 KCL 和 KVL 全列了一遍，结果发现方程数量大于未知数数量，解不出来（线性相关导致无穷多解或无解），于是怀疑自己哪里写错了。

✓ 严格按照公式：$(n-1)$ 个 KCL + $(b-n+1)$ 个 KVL = $b$ 个方程。不要多列，多了也是线性相关的（没用）。列少了当然更不行。先数清楚 $b$ 和 $n$，再动手列方程。

**误区 4：KVL 中绕行方向和正负号搞反**

❌ 有些教程教你"经过电阻时绕行方向与电流方向一致则 $+IR$"，有些教你"一致则 $-IR$"。初学者被两种说法搞晕。

✓ 两种说法都对，只是把"电压降"记为 $+$ 还是 $-$ 的约定不同。本书的约定是：**$\Sigma V = 0$，经过电阻时如果绕行方向与电流方向一致，是电压降，记为 $-IR$。** 你只要选定一种约定并坚持到底，结果就是对的。

#### 题 1 解答

已知 $I_1 = 5\ \text{A}$ 流进，$I_2 = 2\ \text{A}$ 流出。

列 KCL（流入 = 流出）：$I_1 = I_2 + I_3$

如果假设 $I_3$ 流出节点：$5 = 2 + I_3$，$I_3 = 3\ \text{A}$（流出）。

如果假设 $I_3$ 流进节点（用代数和方法验证）：$+5 + (-2) + I_3 = 0$，$I_3 = -3\ \text{A}$。负号说明 $I_3$ 实际是流出。

答案：$I_3 = 3\ \text{A}$，方向为**流出**节点。

---

#### 题 2 解答

**电路**：$V_s = 9\ \text{V}$，$R_1 = 2\ \Omega$，$R_2 = 3\ \Omega$，$R_3 = 4\ \Omega$ 全部串联。

**Step 1**：总电阻 $R_{\text{总}} = 2 + 3 + 4 = 9\ \Omega$。

**Step 2**：用欧姆定律：$I = \frac{V_s}{R_{\text{总}}} = \frac{9}{9} = 1\ \text{A}$。

**Step 3**：KVL 验证。绕行方向顺时针，从电源负极出发：

电压升：$+9\ \text{V}$（$V_s$）

电压降：$-2 \times 1 = -2\ \text{V}$（$R_1$）、$-3 \times 1 = -3\ \text{V}$（$R_2$）、$-4 \times 1 = -4\ \text{V}$（$R_3$）

总：$9 - 2 - 3 - 4 = 0 \ \checkmark$

各电阻压降之和 $= 2 + 3 + 4 = 9\ \text{V} = V_s$。

#### 题 3 解答

$V_1 = 10\ \text{V}$，$V_2 = 4\ \text{V}$（反接），$R = 3\ \Omega$ 串联。

两个电池反接意味着它们的极性"对着干"。$V_1$ 的正极接 $V_2$ 的正极（或负极？反接指方向相反）。

设回路电流 $I$ 为顺时针。$V_1$ 为升（$+10$），$V_2$ 反接后也是升（$+4$）。但等一下，"反接"具体怎么接的？

#### 题 9 解答

这个说法**不对**。

问题 1：$n$ 个 KCL 方程不是全部独立的。只有 $n-1$ 个是独立的，第 $n$ 个可以由前 $n-1$ 个推出。如果列了 $n$ 个 KCL，多出来的那个是冗余的，不提供新信息。

问题 2：同样，$b$ 个回路也不全是独立的。只有 $b-n+1$ 个独立回路（对于平面电路就是网孔数）。如果你随便选了 $b$ 个回路写 KVL，其中很多会是其他回路的线性组合，不提供新信息。

正确做法：$(n-1)$ 个 KCL + $(b-n+1)$ 个 KVL = $b$ 个独立方程，刚好解出 $b$ 个未知电流。

如果硬要列 $n+b$ 个方程，你会发现其中 $n+b-b = n$ 个是冗余的。多余的方程会导致系数矩阵不是满秩的，线性代数求解时会出问题。

#### 题 10 解答

"电流为 $-2\ \text{A}$" 不是电流"真的为负"，而是**实际电流方向与你事先标的参考方向相反，大小为 $2\ \text{A}$。**

参考方向是你自己选的一个"假设"，就像你猜一个未知数大概是多少。算出负号不表示"错了"，而是告诉你：你的假设方向反了，把箭头掉个头就行。

用 KCL 验证：如果你在某个节点上，$I_1 = 5\ \text{A}$（流进），$I_2 = -2\ \text{A}$（你标的参考方向是流进），那实际流出是 $2\ \text{A}$。$I_3 = 3\ \text{A}$（流出）。KCL：$5 - 2 - 3 = 0$，成立。说明数值完全正确，只是 $I_2$ 的实际方向和参考方向相反而已。

负电流不是数学上的"负数"，而是物理上的"方向反转"。这在电路分析中每天都发生，完全正常。

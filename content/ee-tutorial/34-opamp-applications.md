## 1. 有源滤波器——运放+RC 的选频魔法

### 1a. 生活例子：你每天在用滤波器

你打电话时，手机里的滤波器在干活。你听歌时，音响里的分频器也是滤波器——它把低频送给低音炮，把高频送给高音喇叭。你的眼球看不清紫外线也听不到超声波——大自然给你装好了带通滤波器。

滤波器做的事情很简单：**让某些频率通过，挡住另外一些频率。** 就像火车站检票口——有票的过，没票的拦下。只不过滤波器的"票"是信号的频率。

在第18章你学过了无源滤波器——纯 R、L、C 搭的 RC 低通、RL 高通、RLC 带通。那有源滤波器呢？答：**在无源滤波器的基础上加运放**。"有源"（Active）指电路里含有需要供电的放大元件。

### 1b. 直观理解：为什么加运放就厉害了？

纯 RC 低通滤波器（电阻串联、电容对地）有两个天生缺陷：

**缺陷 1：没有增益。** 在低频通带内，输出振幅最多等于输入振幅（$|H| \le 1$）。事实上纯 RC 无源滤波器只能衰减，不能放大。如果在通带内你需要一点增益——对不起，无源做不到。

**缺陷 2：带负载能力差。** 无源 RC 滤波器的输出直接来自中间节点，输出阻抗是 $R$ 的量级（kΩ 级）。如果后面接一个低阻抗的负载，负载会"拖"偏滤波特性——截止频率偏移、通带增益下降。

**有源滤波器怎么解决？** 把运放接在无源 RC 网络的后面，做成一个**缓冲+可能的增益**。运放输入阻抗极高（虚断），几乎不取 RC 网络上的电流，所以 RC 网络不会被加载。运放输出阻抗极低，能驱动任何合理的负载。如果用的是同相放大器配置，还能顺便提供增益。一句概括：**无源部分负责选频，运放负责隔离和增益。**

### 1c. 形式化定义：一阶有源低通滤波器

最简单的有源低通滤波器（Active Low-Pass Filter,简称 LPF）由两级组成：

- **第一级**：RC 无源低通（$R$ 串联、$C$ 对地），截止频率 $\omega_c = 1/(RC)$。
- **第二级**：同相放大器（运放 + $R_1$, $R_f$），增益 $A_v = 1 + R_f / R_1$。

```
        R
Vin ──[====]────┬────┐
                │    │
                C    │   ┌─┐
                │    └───┤+├──┬── Vout
               GND       │  │  │
                    ┌────┤-├──┘
                    │    └─┘
                   GND   R1
                         │
                    ┌──[===]──┐
                    │         │
                    │     Rf  │
                    └───[===]─┘
```

传递函数是两级传递函数的乘积：

$$
H(j\omega) = \underbrace{\frac{1}{1 + j\omega RC}}_{\text{RC 低通}} \times \underbrace{\left(1 + \frac{R_f}{R_1}\right)}_{\text{同相放大器增益}}
$$

在 $\omega \ll \omega_c$（低频通带）时，$|H| \approx 1 \times A_v$——输出振幅是输入的 $A_v$ 倍。在 $\omega \gg \omega_c$（高频阻带）时，$|H| \to 0$——信号被滤掉。

对比纯无源 RC 低通（$|H| \le 1$），有源版本多了 $A_v$ 这个系数。对于需要微弱信号滤波后再放大的场景（如传感器输出中有高频噪声），一级搞定两者，不需要分开做——太方便了。

> **关键公式**（一阶有源低通）：
> $$
> f_c = \frac{1}{2\pi RC}
> $$
> $$
> |H(j\omega)| = A_v \cdot \frac{1}{\sqrt{1 + (\omega / \omega_c)^2}}
> $$
> 其中 $\omega_c = 1/(RC)$，$A_v = 1 + R_f / R_1$。

### 1d. 一阶有源高通滤波器

把 RC 网络中 $R$ 和 $C$ 的位置互换（$C$ 串联、$R$ 对地），后接同相放大器，就是**有源高通滤波器**（Active High-Pass Filter, HPF）：

```
        C
Vin ───||───┬────┐
             │    │
             R    │   ┌─┐
             │    └───┤+├──┬── Vout
            GND       │  │  │
                 ┌────┤-├──┘
                 │    └─┘
                GND   R1
                      │
                 ┌──[===]──┐
                 │         │
                 │     Rf  │
                 └───[===]─┘
```

传递函数：

$$
H(j\omega) = \underbrace{\frac{j\omega RC}{1 + j\omega RC}}_{\text{RC 高通}} \times \underbrace{\left(1 + \frac{R_f}{R_1}\right)}_{\text{增益}}
$$

截止频率同样是 $f_c = 1/(2\pi RC)$。低于 $f_c$ 的信号被抑制（电容对低频的阻抗大），高于 $f_c$ 的信号通过并被放大。

### 1e. 有源带通滤波器——低通+高通的组合

把低通滤波器和高通滤波器**串联**——高通先滤掉低频，低通再滤掉高频，剩下的当然是中间的一段频率。这就是**带通滤波器**（Band-Pass Filter, BPF）。

如果高通截止频率 $f_L$ 低于低通截止频率 $f_H$（即 $f_L < f_H$），则电路允许 $f_L$ 到 $f_H$ 之间的频率通过：

- $f < f_L$：被高通滤掉
- $f_L < f < f_H$：通过（带通）
- $f > f_H$：被低通滤掉

通常 $f_L$ 称为下截止频率，$f_H$ 称为上截止频率。带宽 $BW = f_H - f_L$。中心频率 $f_0 = \sqrt{f_L \cdot f_H}$（对宽带滤波器通常用几何平均）。

### 1f. 手算示例

**例题 1：有源低通滤波器设计**

设计要求：通带增益 5 倍（约 14 dB），截止频率 $f_c = 1\ \text{kHz}$。

**第一步**：选 RC 值来确定 $f_c$。

$f_c = 1/(2\pi RC) = 1000$ Hz。

选 $C = 0.01\ \mu\text{F} = 10^{-8}$ F（常用标准值）。则：

$$
R = \frac{1}{2\pi f_c C} = \frac{1}{2\pi \times 1000 \times 10^{-8}} = \frac{1}{6.283 \times 10^{-5}} \approx 15,915\ \Omega
$$

最接近的标准电阻值为 16 kΩ。用 $R = 16\ \text{k}\Omega$，实际截止频率：

$$
f_c = \frac{1}{2\pi \times 16000 \times 10^{-8}} = \frac{1}{0.0010053} \approx 995\ \text{Hz}
$$

和设计目标 1 kHz 差了 5 Hz（0.5%），工程上完全可以接受。

**第二步**：选增益电阻。

$A_v = 1 + R_f / R_1 = 5$。所以 $R_f / R_1 = 4$。

选 $R_1 = 10\ \text{k}\Omega$，则 $R_f = 40\ \text{k}\Omega$（标准值 39 kΩ + 1 kΩ 串联得到，或者直接用 39 kΩ 得到增益 $1 + 39/10 = 4.9 \approx 5$）。

或用 $R_1 = 2.2\ \text{k}\Omega$，$R_f = 8.8\ \text{k}\Omega$（标准值 9.1 kΩ，增益 $1 + 9.1/2.2 = 5.14$）。

设计完成！总元件清单：运放 1 颗，$R = 16\ \text{k}\Omega$，$C = 0.01\ \mu\text{F}$，$R_1 = 2.2\ \text{k}\Omega$，$R_f = 9.1\ \text{k}\Omega$。

**检验**：给电路输入 1 kHz、0.2 V 峰值的正弦波。在截止频率处，RC 网络衰减为输入的 $1/\sqrt{2}$ 倍（约 0.707）。同相放大器增益 5.14。所以输出振幅：

$$
V_{out} = 0.2 \times 0.707 \times 5.14 \approx 0.727\ \text{V 峰值}
$$

比通带内的 $0.2 \times 5.14 = 1.028$ V 下降到了约 0.7 倍——正是 -3 dB 截止的定义。

**例题 2：带通滤波器设计**

设计一个带通滤波器，下截止频率 $f_L = 100\ \text{Hz}$，上截止频率 $f_H = 10\ \text{kHz}$，通带增益 2。

思路：高通（100 Hz）+ 低通（10 kHz）串联。

**高通级**：$f_L = 100$ Hz。选 $C_1 = 0.1\ \mu\text{F}$。

$$
R_1 = \frac{1}{2\pi \times 100 \times 10^{-7}} = \frac{1}{6.283 \times 10^{-6}} = 159,155\ \Omega \approx 160\ \text{k}\Omega
$$

**低通级**：$f_H = 10$ kHz。选 $C_2 = 0.001\ \mu\text{F}$（1 nF）。

$$
R_2 = \frac{1}{2\pi \times 10000 \times 10^{-9}} = \frac{1}{6.283 \times 10^{-5}} = 15,915\ \Omega \approx 16\ \text{k}\Omega
$$

**增益分配**：总增益 2。可以让高通级增益为 1（电压跟随器，$R_f = 0$, $R_1 = \infty$，或直接短路），低通级增益为 2。或两级各增益 $\sqrt{2} \approx 1.414$ 分配噪优化。简单起见，高通用跟随器（$A_{v1} = 1$），低通用同相放大器 $A_{v2} = 2$。

低通级增益电阻：$A_{v2} = 1 + R_f / R_1 = 2$ → $R_f = R_1$。选 $R_1 = R_f = 10\ \text{k}\Omega$。

高通级直接用电压跟随器（$R_f$ 短路，不需要 $R_1$）。

电路需要两颗运放（或用双运放芯片如 TL072）。

### 1g. Python 验证：有源滤波器的频率响应

```python
# ============================================================
# 有源低通频率响应仿真
# 验证手算的截止频率 f_c = 995 Hz, 增益 A_v ≈ 5
# ============================================================
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ---------- 电路参数 ----------
R = 16e3          # 16 kΩ
C = 0.01e-6       # 0.01 μF
R1 = 2.2e3        # 2.2 kΩ
Rf = 9.1e3        # 9.1 kΩ

Av = 1 + Rf / R1  # 同相放大器增益
fc = 1 / (2 * np.pi * R * C)  # 截止频率

print(f"设计参数：")
print(f"  截止频率 f_c = {fc:.0f} Hz (设计目标 1000 Hz)")
print(f"  通带增益 A_v = {Av:.2f} (设计目标 5)")

# ---------- 频率轴（对数刻度）----------
f = np.logspace(1, 5, 2000)  # 10 Hz ~ 100 kHz
omega = 2 * np.pi * f

# ---------- RC 低通传递函数 ----------
H_RC = 1 / (1 + 1j * omega * R * C)

# ---------- 有源低通传递函数 = RC × Av ----------
H_active = H_RC * Av

# ---------- 画图 ----------
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8))

# 幅频响应（dB）
mag_db = 20 * np.log10(np.abs(H_active))
ax1.semilogx(f, mag_db, 'b-', linewidth=2, label='有源低通')
ax1.axvline(x=fc, color='r', linestyle='--', linewidth=1.5, label=f'f_c = {fc:.0f} Hz')
ax1.axhline(y=20*np.log10(Av) - 3, color='g', linestyle=':', linewidth=1, label=f'{20*np.log10(Av)-3:.1f} dB (-3dB点)')
ax1.set_xlabel('频率 (Hz)', fontsize=12)
ax1.set_ylabel('幅度 (dB)', fontsize=12)
ax1.set_title(f'有源低通滤波器频率响应 (f_c≈{fc:.0f}Hz, A_v≈{Av:.2f})', fontsize=14)
ax1.grid(True, alpha=0.3, which='both')
ax1.legend(fontsize=10)

# 相频响应
phase = np.angle(H_active, deg=True)
ax2.semilogx(f, phase, 'r-', linewidth=2)
ax2.axvline(x=fc, color='r', linestyle='--', linewidth=1.5)
ax2.axhline(y=-45, color='g', linestyle=':', linewidth=1, label='-45° (截止频率处)')
ax2.set_xlabel('频率 (Hz)', fontsize=12)
ax2.set_ylabel('相位 (度)', fontsize=12)
ax2.set_title('相频响应', fontsize=14)
ax2.grid(True, alpha=0.3, which='both')
ax2.legend(fontsize=10)

plt.tight_layout()
plt.show()

# 验证 -3 dB 频率
idx_fc = np.argmin(np.abs(mag_db - (20*np.log10(Av) - 3)))
print(f"\n验证：在 f = {f[idx_fc]:.1f} Hz 处幅度 = {mag_db[idx_fc]:.2f} dB")
print(f"通带增益 (dB): {20*np.log10(Av):.2f} dB")
print(f"-3 dB 频率误差: {f[idx_fc] - fc:.1f} Hz ({(f[idx_fc]-fc)/fc*100:.2f}%)")
```

**预期输出**：幅频曲线在 995 Hz 附近下降 3 dB，低频通带增益约为 14 dB（5 倍）。相频曲线在截止频率处为 -45°。验证显示 -3 dB 点非常接近设计值 995 Hz。

**带通滤波器仿真**：

```python
# ============================================================
# 带通滤波器频率响应（高通 100 Hz + 低通 10 kHz 串联）
# ============================================================
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ---------- 高通参数 (f_L = 100 Hz) ----------
R_hp, C_hp = 160e3, 0.1e-6
fc_low = 1 / (2 * np.pi * R_hp * C_hp)

# ---------- 低通参数 (f_H = 10 kHz), 增益 Av = 2 ----------
R_lp, C_lp = 16e3, 1e-9
fc_high = 1 / (2 * np.pi * R_lp * C_lp)
Av_lp = 2.0

print(f"高通截止频率: {fc_low:.0f} Hz (设计 100 Hz)")
print(f"低通截止频率: {fc_high:.0f} Hz (设计 10000 Hz)")
print(f"通带中心频率: {np.sqrt(fc_low*fc_high):.0f} Hz")
print(f"带宽: {fc_high - fc_low:.0f} Hz")

# ---------- 频率轴 ----------
f = np.logspace(0, 5, 3000)  # 1 Hz ~ 100 kHz
omega = 2 * np.pi * f

# ---------- 传递函数 ----------
H_hp = (1j * omega * R_hp * C_hp) / (1 + 1j * omega * R_hp * C_hp)  # 高通
H_lp = Av_lp / (1 + 1j * omega * R_lp * C_lp)                         # 低通（带增益）
H_bp = H_hp * H_lp                                                     # 带通 = 高通 × 低通

# ---------- 画图 ----------
plt.figure(figsize=(10, 5))
plt.semilogx(f, 20*np.log10(np.abs(H_bp)), 'b-', linewidth=2)
plt.axvline(x=fc_low, color='orange', linestyle='--', linewidth=1.5, label=f'f_L = {fc_low:.0f} Hz')
plt.axvline(x=fc_high, color='red', linestyle='--', linewidth=1.5, label=f'f_H = {fc_high:.0f} Hz')

# 标记 -3 dB 点（相对于通带最大增益）
peak_db = np.max(20*np.log10(np.abs(H_bp)))
plt.axhline(y=peak_db - 3, color='green', linestyle=':', linewidth=1, label=f'通带峰值-3dB ({peak_db-3:.1f} dB)')

plt.xlabel('频率 (Hz)', fontsize=12)
plt.ylabel('幅度 (dB)', fontsize=12)
plt.title(f'有源带通滤波器频率响应 (f_L≈{fc_low:.0f}Hz, f_H≈{fc_high:.0f}Hz, Av={Av_lp})', fontsize=14)
plt.grid(True, alpha=0.3, which='both')
plt.legend(fontsize=10)
plt.tight_layout()
plt.show()

# 验证 -3 dB 带宽
mag_bp = np.abs(H_bp)
idx_upper = np.argmin(np.abs(mag_bp - (np.max(mag_bp)/np.sqrt(2))))
# 在实际应用中，需要分别找左右两侧的 -3 dB 点
f_3dB = f[idx_upper]
print(f"\n从峰值下降 3 dB 对应频率约: {f_3dB:.0f} Hz")
```

**预期输出**：一条钟形曲线，峰值在约 1 kHz 处，增益约 6 dB（×2）。低频端随频率降低而滚降（高通特性），高频端随频率升高而滚降（低通特性）。

### 1h. 常见误区

**❌ 误区 1：有源滤波器和无源滤波器没什么区别，加个运放只是多此一举。**

✓ 区别巨大。无源滤波器不能提供增益（输出 ≤ 输入），且输出阻抗高、带负载能力差。有源滤波器可以提供增益、输出阻抗低（可以驱动重负载）、各级之间不会相互加载。多级级联时，有源滤波器的优势更加明显。

**❌ 误区 2：滤波器的截止频率就是"信号完全被挡住"的频率。**

✓ 截止频率是输出功率降到一半（电压降到 $1/\sqrt{2} \approx 0.707$）的频率点，不是信号突然消失的点。在截止频率以上/以下，信号是**逐渐**衰减的——一阶滤波器每十倍频程衰减约 20 dB，二阶约 40 dB。

**❌ 误区 3：运放截止频率需要比信号最高频率高很多倍，滤波器也不例外。**

✓ 滤波器本来就是要在高频处衰减的。运放只要在**通带最高频率**处有足够增益即可。比如低通截止 10 kHz，运放 GBW 有 100 kHz（在 10 kHz 处开环增益 ≥ 10 倍）就够了，不需要 GBW 远高于阻带频率。

### 1i. 应用连接

有源滤波器无处不在：医疗心电图机（ECG）用带通滤波器提取 0.05-150 Hz 的心电信号同时滤除 50 Hz 工频干扰；音频均衡器用多个带通滤波器把频谱分成低音/中音/高音分别调节；射频接收机的中频滤波器；数据采集系统前级的抗混叠低通滤波器（防止高频噪声在 ADC 采样时折叠回低频）。

在后续的[传感器信号调理](#6-仪表放大器高精度差分信号调理)中，你还会看到滤波和仪表放大器的组合使用——先放大微弱差分信号，再滤除噪声，最后送入 ADC。

## 2. 比较器——运放"裸奔"时的样子

### 2a. 生活例子：自动售货机的硬币检测

自动售货机怎么知道你是不是投了一个真的一元硬币？投币口里有一组光电传感器和一套机械检测。硬币经过时，传感器输出一个脉冲电压。这个电压有大有小——真硬币大，假硬币（比如游戏币）小。电路里有一个**比较器**：设定一个阈值电压。传感器的脉冲超过阈值 → 判断为真硬币，打开出货机构。低于阈值 → 判断为假，退回硬币。

这个"判断大于还是小于"的电路就是比较器。它做的是最简单的决策：**大还是小？**

### 2b. 直观理解：运放开环 = 比较器

在第33章你学过公式：

$$
V_{out} = A_{OL} \times (V_+ - V_-)
$$

其中 $A_{OL}$ 是开环增益，巨大（10 万 ~ 100 万倍）。

当运放**不加负反馈**、直接开环工作时：

- 如果 $V_+ > V_-$，哪怕只大一丁点（比如 1 μV），$V_{out}$ 就会往正电源方向飙，输出饱和在 $+V_{sat}$（接近正电源电压）。
- 如果 $V_+ < V_-$，输出饱和在 $-V_{sat}$（接近负电源电压或 GND）。
- 如果 $V_+ \approx V_-$，这是一个极窄的过渡区（大约 ±几十微伏），输出在这个区间内线性变化。但稍一偏离就立即进入饱和状态。

**这跟"二值输出"的状态一模一样——只有高电平和低电平两种结果。** 这就是一个理想的比较器。

### 2c. 形式化定义：比较器的传输特性

比较器（Comparator）有一个**阈值电压 $V_{ref}$**（参考电压）接在反相输入端，同相输入端接待测信号 $V_{in}$：

```
         ┌─┐
Vin  ────┤+├─── Vout
    ┌────┤-├──┐
    │    └─┘  │
   Vref      GND
```

传输特性是一个"阶梯"：

$$
V_{out} = \begin{cases}
+V_{sat} & \text{当 } V_{in} > V_{ref} \\
-V_{sat} & \text{当 } V_{in} < V_{ref}
\end{cases}
$$

这就是我们常说的"**过零比较器**"（当 $V_{ref} = 0$ 时）或"**阈值比较器**"。

> **本质**：比较器 = 运放在开环模式下工作。开环增益极高 → 输入差值稍微偏离零，输出立即饱和。

顺便说一句：有**专门的比较器芯片**（如 LM393、LM339），它们针对"快速从饱和状态恢复"优化过，比普通运放做比较器速度更快。普通运放（如 LM741）做比较器也能用，但切换速度慢（受限于运放的压摆率和内部频率补偿电容）。对速度要求不高的场合，普通运放假扮比较器完全没问题。

### 2d. 手算示例

**例题 3：电池欠压检测**

设计要求：用比较器检测 3.7 V 锂电池是否低于 3.2 V。低于 3.2 V 时输出低电平（红灯亮），高于 3.2 V 时输出高电平（绿灯亮）。电源 5 V 供电，运放用轨到轨输出（输出能到 5 V 和 0 V）。

**第一步**：产生 3.2 V 的参考电压。用两个电阻分压：$V_{ref} = 5 \times R_2/(R_1 + R_2) = 3.2$ V。

选 $R_2 = 10\ \text{k}\Omega$：

$$
R_1 = \frac{5 - 3.2}{3.2} \times 10\ \text{k}\Omega = \frac{1.8}{3.2} \times 10\ \text{k}\Omega = 5.625\ \text{k}\Omega
$$

标准值用 5.6 kΩ 或 5.62 kΩ。

**第二步**：将 $V_{ref} = 3.2$ V 接入反相输入端（$-$）。锂电池电压 $V_{bat}$ 通过分压器衰减后接入同相输入端（$+$）。因为电池最高 4.2 V，而运放用 5 V 供电，可以直接接（只要不超过 $V_{CC}$ 即可）。

直接接不需要分压衰减（因为电池电压 3.0~4.2 V 在运放输入共模范围 0~5 V 内）。将电池正极直接连到同相输入端。

**第三步**：输出判断。

- $V_{bat} > 3.2$ V → $V_{out} \approx 5$ V（高电平，绿灯亮）
- $V_{bat} < 3.2$ V → $V_{out} \approx 0$ V（低电平，红灯亮）

如果需要在 3.2 V 附近不反复跳变（防止电池电压在阈值附近抖动造成指示灯闪烁），就要加入**滞回特性**——这引出下一节：施密特触发器。

### 2e. 常见误区

**❌ 误区：比较器和运放的黄金法则（虚短）可以同时适用。**

✓ 比较器工作在**开环状态**（无负反馈），运放输出几乎总是饱和的。虚短成立的前提是有负反馈且运放工作在线性区。比较器中 $V_+$ 和 $V_-$ 通常是不相等的（差很多），虚短不成立。虚断仍然成立（输入端不取电流），可以用来分析输入分压网络。

## 3. 施密特触发器——给比较器加上"定力"

### 3a. 生活例子：恒温器的回差

你家里的空调恒温器设定在 25°C。当室温升到 25.1°C 时，压缩机启动制冷。当室温降到 24.9°C 时，压缩机关闭恢复。

但如果恒温器没有"回差"（Hysteresis），室温在 25.0°C 附近波动时会发生什么？25.01°C → 启动！24.99°C → 关闭！25.00°C → 又启动！压缩机在一分钟内启停几十次，很快就烧坏了。

恒温器聪明地做了个"**回差**"（也叫"滞回"）：升温到 25.5°C 才启动制冷，降温到 24.5°C 才关闭。中间有 1°C 的**死区**——在这个区间内，压缩机保持上一个状态不变。这样就不会在阈值附近反复跳变了。

**施密特触发器（Schmitt Trigger）就是给比较器加上"回差"。** 它有两个阈值：上升阈值 $V_{TH+}$ 和下降阈值 $V_{TH-}$。信号从低往高走，超过 $V_{TH+}$ 才能让输出翻转。信号从高往低走，低于 $V_{TH-}$ 才能让输出翻转。$V_{TH+} - V_{TH-}$ 叫做**回差电压**（或滞回宽度）。

### 3b. 直观理解：正反馈的妙用

普通的比较器，阈值电压是固定的（比如上一节例 3 的 3.2 V）。输入信号只要越过这个固定阈值，输出就翻转。如果输入信号上有噪声（比如纹波 ±50 mV），在阈值附近会来回穿越，造成输出反复跳变。

施密特触发器怎么解决？它把**正反馈**（Positive Feedback）引入电路。输出端通过一个电阻网络连回同相输入端。这样一来，参考电压不再是一个固定值，而是**取决于输出当前的状态**。

- 当输出为高电平（$+V_{sat}$）时，同相输入端被拉到较高电压。信号必须上升到这个较高的电压，才能让输出翻转为低。
- 一旦输出变为低电平（$-V_{sat}$），同相输入端被拉到较低电压。信号必须下降到更低的电压，才能让输出翻回高。

输出自己会"记住"当前状态，并影响触发的门槛——这就是滞回。**施密特触发器 = 比较器 + 正反馈。**

### 3c. 形式化定义：施密特触发器的阈值计算

经典的反相施密特触发器电路：

```
        R1        R2
Vin ──[====]───┬───[====]──┬── Vout
               │           │
               │   ┌─┐     │
               └───┤+├─────┘
               ┌───┤-├── Vref
               │   └─┘
              GND
```

$V_{in}$ 通过 $R_1$ 接同相输入端。$V_{out}$ 通过 $R_2$ 也接同相输入端。反相输入端接固定的参考电压 $V_{ref}$（通常为 0,即 GND）。

当 $V_{out} = +V_{sat}$ 时，同相输入端电压为 $R_1$ 和 $R_2$ 对 $V_{in}$ 和 $+V_{sat}$ 的分压（用叠加原理）：

$$
V_+ = V_{in} \cdot \frac{R_2}{R_1 + R_2} + (+V_{sat}) \cdot \frac{R_1}{R_1 + R_2}
$$

翻转条件：$V_+ = V_{ref}$（和反相输入端相等时触发翻转）。代入后解出 $V_{in}$：

$$
V_{TH+} = V_{ref} \cdot \frac{R_1 + R_2}{R_2} - V_{sat} \cdot \frac{R_1}{R_2}
$$

类似地，当 $V_{out} = -V_{sat}$ 时，翻转条件对应的 $V_{in}$ 为：

$$
V_{TH-} = V_{ref} \cdot \frac{R_1 + R_2}{R_2} + V_{sat} \cdot \frac{R_1}{R_2}
$$

（这里假设 $V_{sat}$ 取正值。如果 $-V_{sat} = -V_{sat}$，负号已经包含在上式加减中。）

**回差电压**：

$$
\Delta V = V_{TH+} - V_{TH-} = 2V_{sat} \cdot \frac{R_1}{R_2}
$$

调整 $R_1/R_2$ 的比值，就能任意设置回差大小。比值越大 → 回差越大（抗噪声能力强，但灵敏度下降）。

当 $V_{ref} = 0$（反相端直接接地）时，公式简化为：

$$
V_{TH+} = +V_{sat} \cdot \frac{R_1}{R_1 + R_2}
$$

$$
V_{TH-} = -V_{sat} \cdot \frac{R_1}{R_1 + R_2}
$$

回差 $\Delta V = 2V_{sat} \cdot R_1/(R_1 + R_2)$。

### 3d. 手算示例

**例题 4：带抗噪声的过零检测施密特触发器**

设计一个施密特触发器，用于检测正弦波过零点。电源 ±10 V，运放输出饱和电压 $V_{sat} \approx \pm 9$ V。输入信号有约 ±200 mV 的噪声。要求回差电压至少 500 mV（确保噪声不会触发误翻转）。

$V_{ref} = 0$（过零检测）。回差：

$$
\Delta V = 2 \times 9 \times \frac{R_1}{R_1 + R_2} \ge 0.5\ \text{V}
$$

$$
\frac{R_1}{R_1 + R_2} \ge \frac{0.5}{18} = 0.0278
$$

设 $R_1 = 1\ \text{k}\Omega$：

$$
\frac{1000}{1000 + R_2} \ge 0.0278 \quad\Rightarrow\quad 1000 \ge 27.8 + 0.0278 R_2
$$

$$
R_2 \le \frac{1000 - 27.8}{0.0278} \approx 34,970\ \Omega
$$

取 $R_2 = 33\ \text{k}\Omega$（标准值），回差：

$$
\Delta V = 2 \times 9 \times \frac{1}{1 + 33} = 18 \times \frac{1}{34} \approx 0.529\ \text{V} = 529\ \text{mV}
$$

满足 > 500 mV 要求。阈值：

$$
V_{TH+} = +9 \times \frac{1}{34} \approx +0.265\ \text{V}
$$
$$
V_{TH-} = -0.265\ \text{V}
$$

解释：信号从负向正穿越时，必须在约 +0.265 V 以上才能让输出从 $-9$ V 跳到 $+9$ V。信号从正向负穿越时，必须低于 -0.265 V 才能让输出从 $+9$ V 跳回 $-9$ V。±200 mV 的噪声在 ±0.265 V 的阈值范围内，不会误触发。完美！

### 3e. Python 仿真：施密特触发器的滞回曲线

```python
# ============================================================
# 施密特触发器滞回曲线仿真
# R1=1kΩ, R2=33kΩ, V_sat=±9V, V_ref=0
# ============================================================
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ---------- 参数 ----------
R1, R2 = 1e3, 33e3
Vsat = 9.0  # 饱和电压

# 回差和阈值
Vth_plus  =  Vsat * R1 / (R1 + R2)
Vth_minus = -Vsat * R1 / (R1 + R2)
delta_V   = Vth_plus - Vth_minus

print(f"施密特触发器参数：")
print(f"  上升阈值 V_TH+ = {Vth_plus:.3f} V")
print(f"  下降阈值 V_TH- = {Vth_minus:.3f} V")
print(f"  回差电压 ΔV   = {delta_V:.3f} V ({delta_V*1000:.0f} mV)")

# ---------- 模拟输入信号（带噪声的正弦波上升沿）----------
t = np.linspace(0, 20e-3, 5000)  # 0~20 ms
f_sig = 500                       # 500 Hz 正弦波
V_in = 1.0 * np.sin(2 * np.pi * f_sig * t)  # 1V 峰值
noise = 0.05 * np.random.randn(len(t))       # 50 mV RMS 噪声
V_in_noisy = V_in + noise

# ---------- 施密特触发器行为仿真 ----------
V_out = np.zeros(len(t))
state = -Vsat  # 初始状态低
for i in range(len(t)):
    if V_in_noisy[i] > Vth_plus:
        state = +Vsat
    elif V_in_noisy[i] < Vth_minus:
        state = -Vsat
    # 在回差区间内保持上一状态
    V_out[i] = state

# ---------- 画滞回曲线（V_in vs V_out，去掉噪声看纯净曲线）----------
fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 左图：时域波形
axes[0].plot(t * 1000, V_in_noisy, 'b-', linewidth=0.8, alpha=0.7, label='输入+噪声')
axes[0].plot(t * 1000, V_out, 'r-', linewidth=2, label='施密特输出')
axes[0].axhline(y=Vth_plus, color='orange', linestyle='--', linewidth=1, label=f'V_TH+ = {Vth_plus:.2f}V')
axes[0].axhline(y=Vth_minus, color='green', linestyle='--', linewidth=1, label=f'V_TH- = {Vth_minus:.2f}V')
axes[0].set_xlabel('时间 (ms)', fontsize=12)
axes[0].set_ylabel('电压 (V)', fontsize=12)
axes[0].set_title('施密特触发器时域波形', fontsize=14)
axes[0].grid(True, alpha=0.3)
axes[0].legend(fontsize=9, loc='upper right')

# 右图：滞回曲线（传输特性）
# 用无噪声的输入画清晰的滞回曲线
t_ramp = np.linspace(0, 1, 1000)
V_ramp = -1.5 + 3.0 * t_ramp  # 从 -1.5V 缓慢上升到 +1.5V
V_ramp_down = 1.5 - 3.0 * t_ramp  # 从 +1.5V 回落到 -1.5V

V_out_up = np.where(V_ramp > Vth_plus, +Vsat, -Vsat)
V_out_down = np.where(V_ramp_down < Vth_minus, -Vsat, +Vsat)

axes[1].plot(V_ramp, V_out_up, 'b-', linewidth=2, label='上升沿')
axes[1].plot(V_ramp_down, V_out_down, 'r-', linewidth=2, label='下降沿')
axes[1].axvline(x=Vth_plus, color='orange', linestyle='--', linewidth=1.5)
axes[1].axvline(x=Vth_minus, color='green', linestyle='--', linewidth=1.5)
axes[1].annotate(f'V_TH+={Vth_plus:.2f}', xy=(Vth_plus+0.05, -6), fontsize=10, color='orange')
axes[1].annotate(f'V_TH-={Vth_minus:.2f}', xy=(Vth_minus-0.6, 6), fontsize=10, color='green')
axes[1].set_xlabel('输入电压 V_in (V)', fontsize=12)
axes[1].set_ylabel('输出电压 V_out (V)', fontsize=12)
axes[1].set_title(f'施密特滞回曲线 (ΔV={delta_V:.2f}V)', fontsize=14)
axes[1].grid(True, alpha=0.3)
axes[1].legend(fontsize=10)

plt.tight_layout()
plt.show()

print(f"\n观察：")
print(f"1. 滞回曲线的形状像一个矩形环——输入输出关系不是单值函数")
print(f"2. 在 V_TH- 和 V_TH+ 之间，输出取决于历史（从哪边走进来的）")
print(f"3. ±50mV 噪声在 529mV 回差内，不会造成误触发")
```

**预期输出**：左图显示有噪声的输入正弦波和干净的输出方波（没有在过零点附近抖动）。右图显示经典的施密特滞回矩形环——上升沿在 +0.265 V 处跳变，下降沿在 -0.265 V 处跳变。

### 3f. 常见误区

**❌ 误区 1：正反馈和负反馈差不多，都是"反馈"，没什么区别。**

✓ 完全相反。负反馈让运放**稳定在线性区**，是放大器的核心机制。正反馈让运放**迅速脱离线性区进入饱和**，是振荡器和施密特触发器的核心机制。正反馈越大，电路越"不稳定"（往两个极端跑）。

**❌ 误区 2：回差电压越大越好，抗噪声能力更强。**

✓ 回差大了虽然抗噪声强，但也牺牲了灵敏度。如果回差 500 mV 但你要检测 100 mV 的小信号，电路根本不会触发。回差应该略大于噪声幅值即可，不是越大越好。

### 3g. 应用连接

施密特触发器最重要的应用之一是**数字信号的"整形"**。比如一个方波信号经过长线传输后波形变差（上升沿变缓、加上噪声），在进入数字电路之前用施密特触发器整形，恢复出干净规整的方波。几乎所有数字芯片的输入引脚都内置了施密特触发器（如 74HC14 六反相施密特触发器）。在微控制器（MCU）的 GPIO 引脚中，也常可选择启用施密特输入模式。

此外，电源监控芯片中也有施密特特性——检测电压跌落时产生复位信号，回差防止复位信号在阈值附近抖动。

## 4. 振荡器——用运放自己产生波形

### 4a. 生活例子：弹吉他时的"啸叫"

你弹电吉他，把音量开大，吉他离音箱太近。突然，"呜——"的一声尖锐啸叫，所有人都捂住耳朵。怎么回事？

吉他的拾音器把弦振动变成电信号，音箱把这个信号放大播出声音。声音又被拾音器收进去，再放大，再播出……形成正反馈循环（声音→电→声音→电…）。任何一个频率只要满足"回路增益大于1"的条件，就会在循环中越滚越大，直到音箱功率极限——这就是自激振荡。

**振荡器（Oscillator）本质上就是被"驯服"的自激振荡。** 我们故意设计一个正反馈回路，让它在一个特定频率上满足起振条件，稳定地输出那个频率的正弦波——这就是正弦波振荡器。

### 4b. 直观理解：文氏桥振荡器的"选频"思想

要让振荡器只在一个频率振荡（而不是像吉他啸叫那样乱叫），需要一个**选频网络**——它只对某一个特定频率的信号"友好"（相位偏移 0° 且衰减最小），对其他频率的信号要么衰减大、要么相位不对。

**文氏桥（Wien Bridge）**就是这样一个选频网络。它由两个 RC 对组成——一个串联 RC（$R$ 和 $C$ 串联）、一个并联 RC（$R$ 和 $C$ 并联）。串联臂和并联臂首尾相连，像一个桥。

```
      串联RC           并联RC
 ───[=R=]──||C──┬───┐
                │   R
                ├──[=]──┐
                │       │
                │      === C
                │       │
               Vout    GND
```

这个网络的传递函数（分压输出 / 输入）为：

$$
\beta(j\omega) = \frac{1}{3 + j(\omega RC - \frac{1}{\omega RC})}
$$

在 $\omega = \omega_0 = 1/(RC)$ 时，分母中的虚部 $j(\omega RC - 1/(\omega RC)) = 0$，分母变成纯实数 3。此时：

- $|\beta| = 1/3$（衰减到输入的三分之一）
- $\angle \beta = 0°$（输出和输入同相位）

对于其他频率，要么衰减更大，要么有相位偏移。

### 4c. 形式化定义：文氏桥振荡器的起振条件和振荡频率

把运放的同相放大器接在选频网络后面，形成完整的正反馈回路：

```
           R1
     ┌───[====]───┐
     │          ┌─┤-├── Vout
     │          │ └─┘  │
     │   ┌──────┤+├────┘
     │   │      └─┘
     │   │    选频网络
     │   ├──[=R=]──||C──┬── Vout
     │   │              │
     │   │              R
     │   │             [=]
     │   │              │
     │   │             === C
     │   │              │
     │   └─────────────GND
     │
     └── R2 ── GND
```

**选频网络**把输出的一部分反馈到同相输入端（正反馈）。同相放大器提供增益 $A_v = 1 + R_2 / R_1$（这是负反馈部分？不——文氏桥电路中，同相放大器 R1/R2 是负反馈路径，选频网络是正反馈路径。两条反馈同时在作用，最终的反馈性质取决于正反馈和负反馈的"竞赛"结果）。

在 $\omega_0$ 处：
- 选频网络衰减为 1/3（正反馈路径增益 $\beta = 1/3$）
- 运放增益为 $A_v$

为了维持振荡（不衰减），从输出经过选频网络回到输入端、再被放大出来的信号幅度必须不小于原始信号——即**环路增益 $A_v \times \beta \ge 1$**。这就是**巴克豪森准则（Barkhausen Criterion）**。

因为 $\beta(\omega_0) = 1/3$，所以需要 $A_v \ge 3$。即：

$$
1 + \frac{R_2}{R_1} \ge 3 \quad\Rightarrow\quad R_2 \ge 2R_1
$$

实际设计中，取 $R_2$ 略大于 $2R_1$（比如 $R_2 = 2.1 R_1$），让环路增益略大于 1，确保可靠起振。但增益也不能太大，否则输出会因运放饱和而失真（变成方波而不是正弦波）。

> **文氏桥振荡器关键公式**：
> $$
> f_0 = \frac{1}{2\pi RC}
> $$
> 起振条件：$R_2 \ge 2R_1$（同相放大器增益 $A_v \ge 3$）

### 4d. 手算示例

**例题 5：1 kHz 文氏桥振荡器设计**

设计一个输出频率为 1 kHz 的文氏桥正弦波振荡器。

**第一步**：选 RC。

$f_0 = 1/(2\pi RC) = 1000$ Hz。

选 $C = 0.01\ \mu\text{F}$，则：

$$
R = \frac{1}{2\pi \times 1000 \times 10^{-8}} = \frac{1}{6.283 \times 10^{-5}} = 15,915\ \Omega \approx 16\ \text{k}\Omega
$$

**第二步**：选增益电阻。

$A_v = 3$ 是维持振荡的理论最小值。为了可靠起振，取 $A_v = 3.1$（略大于 3）。

$1 + R_2 / R_1 = 3.1$ → $R_2 / R_1 = 2.1$。

选 $R_1 = 10\ \text{k}\Omega$，则 $R_2 = 21\ \text{k}\Omega$（标准值 22 kΩ 得到增益 $1 + 22/10 = 3.2$）。

如果增益过大（如 $A_v = 5$），输出波形会严重失真——波峰被削平，接近方波。工程上常在 $R_2$ 支路中串联一个微调电位器，或者用稳压管/二极管做自动增益控制（AGC），让增益在起振后自动降低到恰好 3，从而输出纯正弦波。

### 4e. 晶振——当频率精度要求极高时

文氏桥振荡器的频率稳定性受限于 RC 元件的温度系数和老化。RC 的精度做到 1% 就了不起了，而且随温度变化大。

如果你需要极高的频率精度——比如手表的 32.768 kHz 晶振，每天误差不超过 1 秒（精度约 30 ppm,即 百万分之三十），或者 GPS 接收机中的 10 MHz 高稳晶振——RC 振荡器就完全不够用了。

**石英晶体（Crystal）**利用压电效应（机械振动 → 电信号）产生极其稳定的谐振频率。它的等效 Q 值可以达到数万甚至数十万（而 RC 文氏桥的 Q 值只有几）。Q 值越高，谐振峰越尖锐，频率越不容易被外界因素"拉偏"。

一块 32.768 kHz 的晶振，配上两个小电容（典型 12.5 pF）和一个反相器（或运放搭成的振荡电路），就能产生极其精准的时钟信号。它之所以是 32.768 kHz 这个"奇怪"的数字：$2^{15} = 32768$，用 15 级二分频刚好得到 1 Hz（1 秒）的脉冲——完美用于计时。

在实际工程中，对频率精度要求不高的场合用 RC 振荡器（便宜），要求中等用陶瓷谐振器，要求高精度则用石英晶振。

## 5. 精密整流——消除二极管的 0.7 V "罚款"

### 5a. 生活例子：高速公路收费站

普通二极管整流就像高速公路收费站——每过一次都要"缴费"0.7 V。如果你的信号只有 50 mV 峰值，0.7 V 的"过路费"比信号本身还大十几倍，信号根本过不去！小信号直接被二极管"吞"了。

精密整流器（Precision Rectifier）用运放来"垫付"这个过路费。运放监测输入信号，自动在输出端"预加"0.7 V，补偿掉二极管的正向压降。结果是**二极管看起来像理想元件——0 V 正向压降**。

### 5b. 直观理解：半波精密整流电路

经典电路——运放 + 两个二极管做半波精密整流：

```
          R
    ┌───[====]───┐
    │          ┌─┤-├───────┬──── Vout
    │          │ └─┘       │
Vin─┴─────────┤+├──── D1   │
    │          └─┘    ▼    │
    │                  │   │
    │            D2    ├───┘
    │             ▲    │
    │             │    │
    └─────────────┴────┘
```

工作原理（简化理解）：当 $V_{in} > 0$ 时，运放输出端会驱动 $V_-$ 试图跟上 $V_+ = V_{in}$。由于 D1 需要约 0.7 V 才能导通，运放的输出会自动"抬高"约 0.7 V，使得 D1 刚好导通。此时 $V_{out}$ 精确地跟随 $V_{in}$（D1 的压降被运放补偿了）。当 $V_{in} < 0$ 时，D1 截止，$V_{out} = 0$（通过 D2 和反馈路径确定）。

最终效果：$V_{out} = \max(V_{in}, 0)$，精确到毫伏级别。毫伏级的小信号也能被"整流"。

### 5c. 全波精密整流

将半波精密整流和一个加法器组合，可以实现全波精密整流——输出 $V_{out} = |V_{in}|$（绝对值电路）。具体实现通常用两颗运放（一颗半波整流 + 一颗加法器/减法器），这里不展开电路细节。

精密整流器在**仪器仪表**（如数字万用表的 AC-DC 转换）、**音频电平检测**（VU 表驱动）、**AM 包络检波**等领域广泛应用。

## 6. 仪表放大器——高精度差分信号调理

### 6a. 生活例子：在嘈杂的工厂里听一个人说话

你在一个轰隆隆的工厂车间里，想听清对面工友说话。车间里各种机器的噪声有 90 dB（相当于 3.16 V RMS），而工友的说话声只有 60 dB（约 0.02 V RMS）。两个声音混在一起到达你的耳朵。

你的耳朵有一个神奇的能力——它能分辨出两个人的声音方向不同，从而在混响中"提取"出你想听的那个。但如果两个声音从完全相同的方向传来（比如共用一个喇叭放出来），就很难分开了。

**仪表放大器（Instrumentation Amplifier，简称 INA）做的就是这件事：从巨大的共模噪声中提取出微弱的差分信号。** 它的输入不是单个电压，而是一对**差分信号**（两个电压，$V_1$ 和 $V_2$）。输出正比于 $V_1 - V_2$，而对 $(V_1 + V_2)/2$（共模信号）几乎不响应。

### 6b. 直观理解：三运放仪表放大器结构

经典的三运放仪表放大器由三个运放组成：

- **前两级**：两个同相放大器（运放 A1 和 A2），各自分别接差分信号的两个输入端。这两级不取输入电流（虚断），输入阻抗极高——适合直接接高阻抗传感器。两个同相放大器的增益由中间一颗电阻 $R_G$ 同时控制。

- **后一级**：减法器（运放 A3），把前两级的差分输出转换成单端输出，同时进一步抑制共模信号。

整个结构的关键优势：
1. **超高输入阻抗**：两个同相端做输入，不加载传感器。
2. **高共模抑制比（CMRR）**：前级对称结构对共模信号等量放大，再在减法器中被抵消。
3. **增益由一颗电阻设定**：$G = 1 + 2R_1 / R_G$（$R_1$ 是前级反馈电阻，$R_G$ 是跨接在两个反相端之间的增益设定电阻）。

### 6c. 形式化定义：仪表放大器增益公式

三运放仪表放大器总增益（经过推导）：

$$
V_{out} = \left(1 + \frac{2R_1}{R_G}\right) \times \frac{R_3}{R_2} \times (V_{in+} - V_{in-})
$$

通常让第二级（减法器）增益为 1（即 $R_2 = R_3$），此时总增益：

$$
G = 1 + \frac{2R_1}{R_G}
$$

只需改变 $R_G$ 就能调整整体增益。$R_G$ 越小 → 增益越大。商用仪表放大器芯片（如 AD620、INA128）内部已经做好电阻匹配，用户只需要外接一颗 $R_G$ 来设定增益。

### 6d. 手算示例

**例题 6：传感器信号调理**

一个压力传感器输出差分信号：共模电压 2.5 V，差分信号在 -20 mV 到 +20 mV 之间（满量程 40 mV）。需要将差分信号放大到 0-2 V 范围（供 3.3 V ADC 读取）。

需要的差分增益：$G = 2\ \text{V} / 0.04\ \text{V} = 50$。

用仪表放大器，设第二级减法器增益为 1（$R_2 = R_3 = 10\ \text{k}\Omega$）。

前级：$G = 1 + 2R_1 / R_G = 50$ → $2R_1 / R_G = 49$。

选 $R_G = 1\ \text{k}\Omega$（常用值），则 $R_1 = 49 \times 1\ \text{k}\Omega / 2 = 24.5\ \text{k}\Omega$，标准值用 24 kΩ（增益 $1 + 2 \times 24/1 \approx 49$）或 24.9 kΩ（增益接近 50.8）。

输出电压：$V_{out} = 50 \times (V_{in+} - V_{in-})$。当差分信号为 +20 mV 时，$V_{out} = 1\ \text{V}$。配合仪表放大器的参考电压引脚（$V_{ref}$），可以将输出偏置到 $V_{ref}$ 水平——比如设置 $V_{ref} = 1\ \text{V}$，则输出范围变成 1 V ± 1 V = 0 到 2 V，正好对应 ADC 的 0-2 V 输入范围。

实际工程中，也可以直接用 AD620 芯片：$G = 1 + 49.4\ \text{k}\Omega / R_G$。取 $R_G \approx 1\ \text{k}\Omega$，增益约 50.4。一步到位，不需要自己搭三运放。

## 7. V-I/I-V转换器——电压和电流互相转换

### 7a. 生活例子：4-20 mA 工业传感器

在工厂里，传感器和控制室之间的距离可能长达几百米。如果传感器输出的是电压信号（比如 0-5 V），长导线的电阻会分压，线上的电磁干扰也会直接叠加在电压上——到了控制室，信号可能已经面目全非了。

工业界有一个聪明的解决方案：**用电流代替电压来传输信号。** 传感器输出的不是电压而是电流，典型标准是 4-20 mA 电流环。4 mA 代表测量范围的最小值（比如 0°C），20 mA 代表最大值（比如 100°C）。0 mA 代表断线故障——极其方便做故障诊断。

电流信号的好处：导线的电阻不会影响电流（串联回路中电流处处相等），电磁干扰对电流环的影响也比对电压信号小得多。

这就需要两种转换电路：
- **V-I 转换器**：把传感器输出的电压信号（如 0-1 V）转成 4-20 mA 电流输出。
- **I-V 转换器**：在接收端把 4-20 mA 的电流信号转回电压信号（如 0-5 V），供 ADC 采样。

### 7b. V-I 转换器（电压→电流）

用运放实现的一个简单 V-I 转换器——**改进的 Howland 电流源**或简单的浮动负载电路。

最基本的 V-I 转换器：运放 + 一颗检流电阻 $R_{sense}$。负载接在反馈回路中：

```
        R
Vin ──[====]──┬────┐
              │  ┌─┤-├───┐
              └──┤ │  │   │
                 └─┘  │   │
              ┌───────┤+  │
              │       └─┘  │
              │            │
              │         R_load（负载）
              │            │
              └────────────┼── GND (或 Vref)
                          GND
```

虚短保证 $V_- = V_+ = V_{in}$。因为运放不取电流（虚断），流入负载 $R_{load}$ 的电流等于流过 $R$ 的电流：

$$
I_{out} = I_R = \frac{V_{in}}{R}
$$

输出电流 $I_{out}$ 仅由 $V_{in}$ 和 $R$ 决定，与 $R_{load}$ 无关。只要 $V_{in}$ 稳定，$I_{out}$ 就稳定。

### 7c. I-V 转换器（电流→电压）

最简单的 I-V 转换器——**跨阻放大器（Transimpedance Amplifier, TIA）**。常用于光电二极管（输出是微弱电流 nA~μA 级别）的信号放大：

```
        Rf
    ┌──[====]──┐
    │        ┌─┤-├── Vout
I_in────────┤ │  │
            └─┘  │
         ┌───────┤+ 
         │       └─┘
        GND
```

同相端接地（$V_+ = 0$），虚短确保 $V_- = 0$（虚地）。输入电流 $I_{in}$ 必须全部流过 $R_f$（虚断，运放不取电流）：

$$
V_{out} = -I_{in} \times R_f
$$

输入电流 1 μA，$R_f = 1\ \text{M}\Omega$ → 输出 1 V。一个巨大电阻就把微小电流变成了可测量的电压。

## 8. 运放选型指南——茫茫芯片海中找对那颗

学完了前面那么多电路，你可能已经在想："那我要做具体的项目，买哪个运放？" 运放不是通用螺丝刀——不同的应用场景对参数的要求差异极大。这一节给你一个实用的选型框架。

### 8a. 按应用场景分类

| 应用场景 | 关注参数 | 典型型号 |
|---------|---------|---------|
| **通用低频**（音频、传感器） | 价格、供货 | LM358（双运放，低端通用），TL072（JFET 输入，低噪声） |
| **精密测量**（仪器、ADC 前级） | 失调电压 $V_{os}$ < 100 μV，低漂移 | OPA277，OPA227，AD8628（零漂移） |
| **高速信号**（视频、射频 IF） | GBW > 50 MHz，SR > 100 V/μs | AD8055，THS4631，LM7171 |
| **低功耗/电池供电** | 静态电流 < 1 mA，轨到轨输入输出 | MCP6002，TLV2371，OPA369 |
| **低噪声**（微弱信号前置放大） | 电压噪声密度 < 5 nV/√Hz | NE5532（音频），AD797（超低噪声），LT1028 |
| **高输出电流**（耳机驱动、线驱动） | 输出电流 > 50 mA | LM386（音频功放），BUF634（缓冲器），OPA551 |

### 8b. 选型四步曲

**第 1 步：确定信号带宽。** 信号最高频率是多少？需要的闭环增益是多少？算出最小 GBW = 增益 × 最高频率。然后乘以 5-10 倍的余量。

**第 2 步：检查输入/输出摆幅。** 输入信号的电压范围在运放的共模输入范围内吗？输出需要的摆幅（峰峰值）在电源电压减去饱和压降后够吗？如果是单电源供电，运放支持轨到轨（Rail-to-Rail）输入/输出吗？

**第 3 步：评估精度需求。** 需要多高的直流精度？失调电压 $V_{os}$ 会不会被放大到不可接受的水平？温漂在目标温度范围内（如 -40°C 到 85°C）会引入多大误差？

**第 4 步：考虑噪声和功耗约束。** 是电池供电（功耗关键）还是市电供电（功耗不敏感）？噪声要求严格吗（如音频前端需要 < 5 nV/√Hz）？

### 8c. 常见场景典型选择

- **学习/实验/面包板**：LM358（便宜到可以烧坏不心疼）、TL072/TL074（音频级别且便宜）
- **音频前放/混音**：NE5532（低噪声+便宜，音频黄金标准）、OPA2134（Hi-Fi 级）
- **传感器信号调理**：INA128（仪表放大器，一颗搞定差分放大）、AD620（经典款）
- **高精度数据采集**：OPA277（精密）、AD8628（自稳零零漂移，$V_{os}$ < 1 μV）

## 9. 完整实战：设计一个电容式触摸检测电路

**场景**：你要设计一个触摸按键——手指触摸时，感应电容约 10 pF 的变化。需要在 100 ms 内检测到触摸事件，用 LED 指示。

**方案**：用运放搭一个**电容-频率转换器**（振荡器，频率随电容变化）。手指不触摸 → 振荡频率 $f_1$ 较低。手指触摸 → 附加电容使频率降到 $f_2$。单片机测量频率的变化，判断是否有触摸。

这里我们只专注运放振荡器部分。

**电路设计**：用施密特触发器 + RC 充放电做张弛振荡器。

```
        R
    ┌──[====]──┐
    │        ┌─┤-├── Vout
    │   ┌────┤ │  │
    │   │    └─┘  │
    │   │         │
    │   │    RF   │
    │   └───[===]─┘
    │
    C_sense（触摸感应电容，未触摸 20 pF，触摸后 30 pF）
    │
   GND
```

这是经典的运放张弛振荡器，振荡频率取决于 RC：

$$
f \approx \frac{1}{2RC \times \ln\! \left(\frac{1+\beta}{1-\beta}\right)}, \quad \beta = \frac{R_1}{R_1 + R_2}
$$

（$R_1$, $R_2$ 是施密特正反馈分压电阻）

取 $R = 1\ \text{M}\Omega$，$C = 20\ \text{pF}$（未触摸）。分压比 $\beta = 0.5$（$R_1 = R_2 = 10\ \text{k}\Omega$）。

$$
f \approx \frac{1}{2 \times 10^6 \times 20 \times 10^{-12} \times \ln(3)} \approx \frac{1}{4 \times 10^{-5} \times 1.099} \approx 22,750\ \text{Hz} \approx 22.8\ \text{kHz}
$$

触摸后 $C = 30\ \text{pF}$，频率降至约 $15.2\ \text{kHz}$。频率变化约 7.6 kHz，足够被单片机轻松检测。

## 10. Python 综合仿真：有源滤波器 + 施密特 + 振荡器

前面已经分节给出了滤波器和施密特的 Python 代码。这里补一个**文氏桥振荡器起振过程**的时域仿真：

```python
# ============================================================
# 文氏桥振荡器起振过程仿真
# 用一阶低通近似运放带宽，模拟从噪声中"长"出正弦波
# ============================================================
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ---------- 参数 ----------
R, C = 16e3, 0.01e-6
f0 = 1 / (2 * np.pi * R * C)
print(f"文氏桥固有频率 f0 = {f0:.0f} Hz")

# 运放增益（略大于3以起振）
Av = 3.2
# 选频网络在 f0 处的衰减 = 1/3
beta = 1.0 / 3.0

# ---------- 时间轴 ----------
fs = 500e3  # 采样率 500 kHz
t_end = 0.05  # 50 ms
t = np.linspace(0, t_end, int(t_end * fs))
dt = t[1] - t[0]

# ---------- 运放一阶模型（有限GBW）----------
GBW = 1e6  # 1 MHz GBW
tau_opamp = 1 / (2 * np.pi * GBW / Av)  # 闭环时间常数

# ---------- 仿真振荡器起振 ----------
# 用数值方法解电路的微分方程
# 简化模型：Vout 通过选频网络反馈到通相端，和反相端比较
v_out = np.zeros(len(t))
v_plus = np.zeros(len(t))
# 加微小初始扰动模拟噪声触发起振
v_out[0] = 0.001  # 1 mV 初始扰动

# RC 选频网络状态（串联电容电压和并联电容电压）
v_c_series = np.zeros(len(t))
v_c_parallel = np.zeros(len(t))

for i in range(1, len(t)):
    # 简化的离散时域仿真
    # 选频网络输出（同相端电压）≈ 前一时刻的 v_out 经过 RC 网络
    # 用一阶近似
    v_plus[i] = v_plus[i-1] + dt * (v_out[i-1] * beta - v_plus[i-1]) / (R * C)
    
    # 同相放大器（反相端接分压反馈）
    v_minus = v_out[i-1] * (1.0 / Av)  # 负反馈分压
    v_diff = v_plus[i] - v_minus
    v_out[i] = v_out[i-1] + dt * (Av * v_diff - v_out[i-1]) / tau_opamp
    
    # 防止饱和（真实运放输出有限）
    v_out[i] = np.clip(v_out[i], -12, 12)

# ---------- 画图 ----------
fig, axes = plt.subplots(2, 1, figsize=(12, 7))

# 时域波形
axes[0].plot(t * 1000, v_out, 'b-', linewidth=1)
axes[0].set_xlabel('时间 (ms)', fontsize=12)
axes[0].set_ylabel('V_out (V)', fontsize=12)
axes[0].set_title(f'文氏桥振荡器起振过程 (f0≈{f0:.0f}Hz, Av={Av:.1f})', fontsize=14)
axes[0].grid(True, alpha=0.3)
axes[0].set_xlim(0, 20)

# 稳态局部放大
axes[1].plot(t * 1000, v_out, 'b-', linewidth=1.5)
axes[1].set_xlabel('时间 (ms)', fontsize=12)
axes[1].set_ylabel('V_out (V)', fontsize=12)
axes[1].set_title(f'稳态振荡波形 (40-50 ms 局部放大)', fontsize=14)
axes[1].grid(True, alpha=0.3)
axes[1].set_xlim(40, 50)

plt.tight_layout()
plt.show()

# 用 FFT 验证振荡频率
from scipy.fft import fft, fftfreq
n = len(v_out)
v_fft = np.abs(fft(v_out[-int(fs/f0*10):]))  # 取最后10个周期
freqs = fftfreq(len(v_fft), 1/fs)

idx_peak = np.argmax(v_fft[:len(v_fft)//2])
f_measured = freqs[idx_peak]
print(f"\nFFT 实测振荡频率: {f_measured:.1f} Hz (理论 {f0:.0f} Hz)")
print(f"频率误差: {abs(f_measured - f0)/f0*100:.2f}%")
```

**预期输出**：时域波形显示从小扰动（1 mV）逐渐放大、最终进入稳态的正弦振荡（约 ±10 V 峰峰值）。FFT 验证实测频率非常接近 1 kHz。

## 11. 常见误区总结

除了前面各节中已穿插的误区，这里再补充几个跨章节的常见陷阱：

**❌ 误区 1：运放做比较器时，输出可以用一个电阻上拉到任意逻辑电平。**

✓ 只有当运放是**开漏输出（Open-Drain）**或**开集输出（Open-Collector）**时才能做电平转换（用一个上拉电阻接到目标逻辑电压）。大多数通用运放是推挽输出（Push-Pull），输出电平由电源电压决定，不能上拉到超过电源。专用比较器（如 LM393）是开集输出，适合电平转换。

**❌ 误区 2：有源滤波器不需要考虑运放的 GBW。**

✓ 虽然滤波器在阻带衰减大，但运放仍需在**通带最高频率**处有足够的开环增益。如果 GBW 不够，高频性能会恶化（通带增益下降、截止频率偏移）。经验：GBW ≥ 100 × 通带最高频率 × 增益。

**❌ 误区 3：仪表放大器的输入可以随便接。**

✓ 仪表放大器的两个输入端虽然阻抗高，但**共模电压必须在芯片规定范围内**。比如 INA128 用 ±15 V 供电时，共模输入范围大约是 ±13 V。如果共模电压超出这个范围（比如接了一个 24 V 电源上的电流采样电阻），仪表放大器不会正常工作。此外，两个输入端都不能悬空——必须有到地的直流偏置路径。

**❌ 误区 4：振荡器起振后振幅会无限增长，所以必须加保护电路。**

✓ 在真实电路中，运放输出饱和会自然限制振幅（输出被"夹"在 ±$V_{sat}$ 之间），不需要额外保护。但如果想要纯净的正弦波，饱和失真反而是问题——需要用 AGC（自动增益控制）电路在起振后自动降低增益到恰好 3，维持稳定且不失真的正弦输出。

**题 1 解答**：

三个优势：
1. **有增益**：有源滤波器可以提供增益（$A_v > 1$），而无源滤波器输出 ≤ 输入。这对于需要滤波+放大的场景（如微弱传感器信号）非常关键。
2. **带负载能力强**：运放输出阻抗极低，驱动后级负载不会改变滤波特性。无源滤波器的输出阻抗大，负载变动会影响截止频率和通带增益。
3. **级联不相互加载**：运放的高输入阻抗使各级之间隔离得很好。多级无源滤波器级联时，后级会加载前级，使得整体响应不是各级响应的简单乘积。有源滤波器各级独立，总传递函数就是各级乘积。

---

**题 2 解答**：

- **虚短不成立**。比较器工作在开环状态（无负反馈），运放输出几乎总是饱和在电源电压附近。输入端的电压差 $(V_+ - V_-) = V_{out}/A_{OL}$。虽然 $A_{OL}$ 很大，但 $V_{out}$ 也很大（饱和值），所以 $V_+ - V_-$ 可能达到几十毫伏甚至几伏——和"近似零"差距巨大。
- **虚断仍成立**。运放的输入阻抗极高这一物理特性不依赖于是否有负反馈。无论开环还是闭环，流入运放输入端的电流都可以忽略（pA~nA 级别）。

**题 3 解答**：

频率由选频网络中的 $R$ 和 $C$ 决定：$f_0 = 1/(2\pi RC)$。

起振条件由巴克豪森准则给出：环路增益 $A_v \times \beta \ge 1$。$\beta(\omega_0) = 1/3$，所以 $A_v \ge 3$，即 $1 + R_2/R_1 \ge 3$ 或 $R_2 \ge 2R_1$。

注意：$A_v = 3$ 是理论上的维持振荡条件。实际设计中通常取 $A_v$ 略大于 3（如 3.1~3.2）以确保可靠起振，但不能太大（否则输出会因饱和失真变成方波而非正弦波）。

**题 4 解答**：

给定 $C = 0.022\ \mu\text{F} = 2.2 \times 10^{-8}$ F，$f_c = 500$ Hz。

**第一步**：算 $R$。
$$
R = \frac{1}{2\pi f_c C} = \frac{1}{2\pi \times 500 \times 2.2 \times 10^{-8}} = \frac{1}{6.912 \times 10^{-5}} = 14,468\ \Omega
$$

标准电阻值选择：14.7 kΩ（E24 系列中最接近的）。如果要求更精确，可以用 14.3 kΩ + 220 Ω 串联得到 14.52 kΩ。

用 14.7 kΩ 时实际截止频率：
$$
f_c = \frac{1}{2\pi \times 14700 \times 2.2 \times 10^{-8}} \approx 492\ \text{Hz}
$$
误差约 1.6%，可以接受。

**第二步**：算增益电阻。
$A_v = 1 + R_f/R_1 = 10$ → $R_f/R_1 = 9$。

选 $R_1 = 2.2\ \text{k}\Omega$，则 $R_f = 9 \times 2.2 = 19.8\ \text{k}\Omega$。标准值 20 kΩ 给出增益 $1 + 20/2.2 = 10.09$。

设计总结：$R = 14.7\ \text{k}\Omega$，$C = 0.022\ \mu\text{F}$，$R_1 = 2.2\ \text{k}\Omega$，$R_f = 20\ \text{k}\Omega$。

**题 5 解答**：

公式：$V_{TH+} = V_{sat} \cdot R_1/(R_1 + R_2)$，$V_{TH-} = -V_{sat} \cdot R_1/(R_1 + R_2)$。

分压比：
$$
\frac{R_1}{R_1 + R_2} = \frac{2.2}{2.2 + 47} = \frac{2.2}{49.2} = 0.04472
$$

$$
V_{TH+} = 12 \times 0.04472 = +0.537\ \text{V}
$$

$$
V_{TH-} = -12 \times 0.04472 = -0.537\ \text{V}
$$

$$
\Delta V = 0.537 - (-0.537) = 1.074\ \text{V}

噪声峰峰值 200 mV，远小于回差 1.074 V（约 5.4 倍余量），**完全够用**。只有当噪声峰峰值超过 1.074 V 时才会误触发。

差分输入范围 0~100 mV，需要输出 0~2.5 V。

G = \frac{2.5\ \text{V}}{0.1\ \text{V}} = 25
$$

后级减法器增益为 1，所以总增益 = 前级增益。

G = 1 + \frac{2R_1}{R_G} = 25

\frac{2R_1}{R_G} = 24

R_G = \frac{2 \times 10}{24} = \frac{20}{24} = 0.833\ \text{k}\Omega = 833\ \Omega

**题 7 解答**：

**电路设计**：

用 LM358（单电源 5 V 供电，轨到轨输出）。

**1. 参考电压产生**：用电阻分压把 5 V 分出一个约 3.3 V 的参考。但 3.3 V 太高（电池可能在 3.3 V 上下），不容易做精确分压且容易受电源纹波影响。更好的办法是用 TL431（2.5 V 精密基准源）产生稳定的参考电压，然后分压得到所需阈值。但为了简单，直接用电阻分压 5 V → 获得 1.65 V（5 V 的一半作为施密特的参考点），将电池电压也相应分压。

**更简洁的方案**：用两个电阻分压采样电池电压，和一个固定的参考电压比较。设电池电压经 $R_a$ 和 $R_b$ 分压后，在 3.3 V 时等于 1 V 参考电压（可用分压从 5 V 电源获取或用稳压管）。

为了简化，用电阻分压产生约 2.5 V 参考电压：$R_{ref1} = R_{ref2} = 10\ \text{k}\Omega$ 分压 5 V 得到 2.5 V。

电池电压经 $R_3 = 10\ \text{k}\Omega$ 和 $R_4 = 10\ \text{k}\Omega$ 分压（衰减一半）。当电池 3.3 V 时，分压点电压 = $3.3 \times 10/(10+10) = 1.65$ V。

比较器参考设为 1.65 V。这个电压从 5 V 经分压得到：$5 \times R_b/(R_a + R_b) = 1.65$。选 $R_a = 20\ \text{k}\Omega$，$R_b = 10\ \text{k}\Omega$，则 $5 \times 10/30 = 1.667$ V ≈ 1.65 V（误差可接受）。

**2. 施密特回差**：需要约 100 mV 的回差。在电池分压点观察，100 mV 的电池电压变化在分压点体现为 50 mV（分压比为 0.5）。

施密特回差公式（$V_{ref} = 1.65$ V，$V_{sat} = 5$ V，$V_{sat-} = 0$ V 单电源）：

$$
V_{TH+} = V_{ref} \cdot \frac{R_1+R_2}{R_2} - V_{sat} \cdot \frac{R_1}{R_2}
$$
$$
V_{TH-} = V_{ref} \cdot \frac{R_1+R_2}{R_2} - 0
$$

回差 $\Delta V = V_{TH+} - V_{TH-} = V_{sat} \cdot R_1/R_2 = 5 \times R_1/R_2$。

要求在电池端体现 100 mV 回差。分压后回差需要 50 mV：

$$
5 \times \frac{R_1}{R_2} = 0.05
$$

$$
\frac{R_1}{R_2} = 0.01
$$

选 $R_2 = 100\ \text{k}\Omega$，$R_1 = 1\ \text{k}\Omega$。

**3. 输出驱动 LED**：LM358 输出可直接驱动 LED（串一个限流电阻约 220 Ω 到 1 kΩ）。

绿灯（电池 OK）：运放输出高 → 绿灯亮。红灯（电池低）：运放输出低 → 比较器输出经一个反相器（或另一个 LM358 通道接成反相器）驱动红灯。

或者更简单：$V_{bat}$ 到同相端，$V_{ref}$ 到反相端。
- 正常（$V_{bat} > 3.3$ V）→ 输出高 → 绿灯亮、红灯灭。
- 低压（$V_{bat} < 3.2$ V）→ 输出低 → 绿灯灭、红灯亮（红灯串在 VCC 到输出之间，输出低时红灯有电流）。

**电路总结**：
- 电池分压：$R_3 = R_4 = 10\ \text{k}\Omega$，分压点接运放同相端。
- 参考分压：$R_a = 20\ \text{k}\Omega$，$R_b = 10\ \text{k}\Omega$ 从 5 V 分得约 1.67 V，接运放反相端。
- 正反馈施密特电阻：$R_1 = 1\ \text{k}\Omega$（从同相端到输出），$R_2 = 100\ \text{k}\Omega$（从同相端到参考点，或到地视电路拓扑而定——这里用经典施密特结构需要仔细画图。简化用同相端→输出接 $R_1 = 1\ \text{k}\Omega$，反相端到参考，回差约 50 mV 在分压点，对应电池端 100 mV）。
- LED：绿灯（输出高亮）串 470 Ω 到 GND。红灯（输出低亮）阳极接 5 V，阴极串 470 Ω 到运放输出。

**题 8 解答**：

RC 有源滤波器占据主导地位的主要原因是**电感在实际电路中的缺陷**：
1. **体积和重量**：低频（如音频 20 Hz~20 kHz）需要的电感量在 H（亨利）级别，这种电感又大又重又贵。而 RC 滤波器用的小电容（μF~nF 级别）和小电阻（kΩ 级别）又小又轻又便宜。
2. **非理想特性**：实际电感有串联电阻（铜损）、并联电容（绕组间分布电容）、磁芯饱和等非理想效应，使得滤波特性偏离理论。高 Q 值电感的实现尤其困难。
3. **电磁干扰**：电感会产生漏磁场，干扰周围电路，也容易被外界磁场干扰。

有源 RC 滤波器用运放的增益和反馈来模拟电感的效果（基于 Sallen-Key 或 Multiple Feedback 等拓扑），完全避开了物理电感的缺点。

**RLC 有源滤波器什么时候用？** 在以下场景：
- **极高频率**（> 1 MHz）：此时运放的 GBW 和 SR 成为瓶颈，无源 RLC 滤波器可能更优。
- **高功率**：滤波器要处理的信号功率很大（如发射机的输出滤波），运放承受不住大功率，必须用无源 LC 或 RLC。
- **需要尖锐的谐振特性**：在一些特殊场合（如射频调谐），LC 谐振回路的天然高 Q 值直接利用可能比用运放模拟更简单。

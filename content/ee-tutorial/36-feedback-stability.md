## 0. 本章导览

第33章我们学会了用运放搭放大器。你可能会有一个困惑：运放本身的开环增益 $A_{OL}$ 高达 100,000 甚至 1,000,000（100-120 dB），但我们实际做出来的放大器增益却只有 10 倍、20 倍，而且非常精确。这中间到底发生了什么魔法？

答案就是**负反馈**（Negative Feedback）。它是模拟电路中最重要、最聪明的设计思想之一。没有负反馈，运放几乎没法用——增益太大、漂移太大、失真太大、带宽太窄。加上负反馈后，增益由外部电阻精确决定，失真降低，带宽展宽，整个放大器变得"又准又稳"。

但负反馈也有代价——处理不好就会**振荡**。本章后半部分会讲清楚为什么会振荡、如何判断稳定性、以及怎么设计补偿来"稳住"你的电路。

## 1. 负反馈的直觉——从生活开始

### 1a. 生活例子：骑自行车

你骑过自行车吗？想象你正骑在一条有点窄的小路上。你是怎么保持在路中间、不倒下去的？

你骑车的全过程就是一个**负反馈系统**：

1. **设定目标**：你的大脑想要沿着路的中间骑（这是"期望值"）。
2. **持续观察**：你的眼睛不断看路，发现车头偏左了（这是"检测误差"）。
3. **修正动作**：你的手臂轻轻往右拐，把车头拉回来（这是"反馈控制"）。
4. **重新比较**：拉回来之后继续观察，如果偏右了再往左修正。
5. **反复循环**：这个过程每秒发生几十次，所以最终看起来你骑得很直很稳。

如果把骑自行车画成信号流程图，大概长这样：

```
        ┌──────────┐
期望路径─┤ 大脑(减法)├──误差──→[手臂肌肉]──→[自行车]──┬──→实际路径
        └──────────┘                               │
               ↑                                   │
               └────────── 眼睛(反馈) ──────────────┘
```

**大脑不断"减去"实际位置和期望位置的差异**——这就是"负"反馈（负 = 做减法）。如果差异被放大而不是被减掉（比如你看到偏左了反而更往左拐），那就是正反馈——你很快就会摔倒了。

### 1b. 直观理解：放大器的"自动驾驶"

回到放大器。运放加负反馈就像给放大器装上了"自动驾驶"：

- 运放的 $V_{out} = A_{OL} \times (V_+ - V_-)$。开环增益 $A_{OL}$ 极大。
- 我们把输出端的一部分信号通过电阻分压**接回**反相输入端 $V_-$。
- 由于 $A_{OL}$ 极大，$V_{out}$ 只要有一点点不对，$V_-$ 就会跟着变，进而拉大或拉小 $V_+ - V_-$ 的差值。
- 系统自动稳定到一个状态：$V_+ \approx V_-$（虚短），输出刚好等于输入乘以外部电阻决定的倍数。

用更直白的话说：**运放自己"太猛了"——增益一百万，稍微一丁点输入变化输出就撞墙。负反馈像一个缰绳，把这匹烈马驯服成一匹听话的马，你想让它走多快（增益多大）它就多快，而且很稳定。**

另一个经典类比是**淋浴调水温**。你把水龙头往"热"方向拧一点，水太烫了，你就往回拧一点；太凉了再往热拧一点……你在不断地比较"实际水温"和"期望水温"，然后反向调节。这就是负反馈。如果你觉得水凉了反而把热水关小——这才叫正反馈，你会冻死或烫伤。

### 1c. 形式化定义：正反馈与负反馈

在电路中，反馈就是把输出信号（电压或电流）的一部分送回到输入端。

**负反馈**（Negative Feedback）：反馈信号与输入信号**相位相反**（相差 180°），两者相减。数学上：
$$
x_f = \beta \cdot x_o, \quad x_i' = x_i - x_f
$$
其中 $x_i$ 是原始输入，$x_o$ 是输出，$\beta$ 是反馈系数（反馈网络的比例），$x_f$ 是反馈信号，$x_i'$ 是送入放大器本身的净输入。

**正反馈**（Positive Feedback）：反馈信号与输入信号**相位相同**（同相），两者相加。数学上：
$$
x_i' = x_i + x_f
$$
正反馈让信号越滚越大，最终导致振荡（就像话筒对着音箱发出的"啸叫"声）。正反馈并非总是不好——下一章的振荡器就靠正反馈来产生信号。但在放大器中，正反馈通常是要避免的。

本章全部围绕**负反馈放大器**展开。

### 1d. 手算示例：反相放大器的反馈分析

以第33章学过的反相放大器为例，用反馈的角度重新分析：

```
         R2 (反馈电阻)
    ┌─────[̅̅R2̅̅]─────┐
    │                │
Vi──[R1]──┬──(-)运放(+)──GND──→ Vo
          │
          └──────────────┘
```

- 这是**电压并联负反馈**（后面会详细讲四种类型）。
- 反馈网络由 $R_1$ 和 $R_2$ 组成，反馈系数 $\beta$ 是多少？

反馈系数 $\beta$ 定义为反馈电压（或电流）与输出电压之比。在反相放大器中，反馈电压 $V_f$ 是 $R_1$ 和 $R_2$ 分压的结果（假设运放输入电流为零）：

$$
V_f = V_o \cdot \frac{R_1}{R_1 + R_2}
$$

所以 $\beta = \dfrac{R_1}{R_1 + R_2}$。

设 $R_1 = 1\ \text{kΩ}$，$R_2 = 10\ \text{kΩ}$，则：
$$
\beta = \frac{1}{1+10} = \frac{1}{11} \approx 0.0909
$$

闭环增益（假设 $A_{OL} \to \infty$，即深度负反馈）：
$$
A_f = \frac{1}{\beta} = 11
$$

但反相放大器的增益是 $-R_2/R_1 = -10$，为什么这里是 $+11$？**注意**：这里的 $A_f = 1/\beta$ 是从**同相**信号角度算的，而反相放大器的负号来源于信号方式的差异。在深度负反馈下 $A_f \approx 1/\beta$ 是近似公式，具体拓扑需要具体分析。我们马上在第4节详细推导精确公式。

### 1e. Python 验证：看看负反馈怎样"驯服"增益

```python
# 负反馈如何让增益变稳定——开环漂移 ±50%，闭环几乎不变
import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

A_nominal = 100000
A_variations = np.linspace(50000, 150000, 1000)  # 模拟 ±50% 漂移
beta = 0.1
Af = A_variations / (1 + A_variations * beta)
Af_ideal = 1 / beta

print(f"开环 A 变化: {A_variations[0]:.0f} → {A_variations[-1]:.0f}  (±50%)")
print(f"闭环 Af 变化: {Af[0]:.4f} → {Af[-1]:.4f}")
print(f"最大偏差: {(Af[-1]-Af_ideal)/Af_ideal*100:.4f}%")
print(f"反馈深度 1+Aβ = {1+A_nominal*beta:.0f}")

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))
ax1.plot(A_variations/1000, Af, 'b-', linewidth=2)
ax1.axhline(Af_ideal, color='r', linestyle='--', label=f'理想 1/β={Af_ideal:.1f}')
ax1.set_xlabel('开环增益 A (×1000)'); ax1.set_ylabel('闭环增益 Af')
ax1.set_title('开环漂移 ±50%，闭环几乎不变'); ax1.legend(); ax1.grid(alpha=0.3)

ax2.plot(A_variations/1000, Af, 'b-', linewidth=2)
ax2.axhline(Af_ideal, color='r', linestyle='--')
ax2.set_ylim(Af_ideal-0.01, Af_ideal+0.01)  # 放大纵轴
ax2.set_xlabel('开环增益 A (×1000)'); ax2.set_ylabel('闭环增益 Af')
ax2.set_title('纵轴放大——变化极小'); ax2.grid(alpha=0.3)
plt.tight_layout(); plt.show()
```

预期输出：开环变化 ±50%，闭环仅变化约 ±0.005%。稳定性提升约 10,000 倍（$1+A\beta \approx 10001$）。

### 1f. 常见误区

❌ **误区1："负反馈让输出变小，所以信号变弱了。"**
✓ 负反馈让**增益**变小，但不是让信号变弱。你可以通过加大输入信号来获得需要的输出幅度。代价是增益降低，换来的是精度、稳定性和带宽的全面提升。

❌ **误区2："任何把输出接到输入的就是负反馈。"**
✓ 关键是**相位**。接回去的信号如果和输入反相才是负反馈。如果把输出接到同相输入端，或者反馈网络引入了额外相移，可能变成正反馈而导致振荡。反馈是"负"还是"正"取决于信号频率。

❌ **误区3："反馈系数 $\beta$ 越大越好（反馈越深越好）。"**
✓ $\beta$ 越大，反馈越深，精度越高，但系统越容易振荡。深度负反馈需要频率补偿来保证稳定性。设计上永远在精度和稳定性之间取平衡。

### 1g. 应用连接

负反馈无处不在。你手机里的音频放大器、WiFi接收器中的低噪声放大器、心电图仪的前置放大器、电源管理芯片中的稳压器……几乎任何一个需要"精确放大"的地方都用了负反馈。第37章要讲的稳压电源，本质上就是一个深度负反馈系统——输出电压和参考电压比较，误差放大后去控制调整管。

## 2. 四种负反馈拓扑——从信号类型分类

### 2a. 生活例子：四种"监督"方式

想象你是老师，想确保学生考试不作弊。有几种监督方式：

- **盯着他们的试卷看**（采样输出电压——电压反馈）
- **站在他们背后，看有没有小动作**（采样输出电流——电流反馈）
- **把监督结果告诉每个学生本人**（串联叠加到输入——串联反馈）
- **把监督结果贴在黑板上让大家看**（并联叠加到输入——并联反馈）

放大器中，反馈网络**采样什么信号**（电压还是电流）和**怎样叠加回输入端**（串联还是并联），决定了负反馈的四种基本拓扑。

### 2b. 直观理解：四张方框图

反馈放大器抽象为两个方块：基本放大器 $A$、反馈网络 $\beta$，以及求和点 ($\Sigma$)：输入 $X_i$ 减去反馈 $X_f$ 得净输入 $X_i'$。

```
Xi ──→(Σ)──→ 基本放大器 A ├──→ Xo
      ↑                    │
      └── 反馈网络 β ←─────┘
```

根据 $X_i$、$X_o$、$X_f$ 是电压还是电流，产生四种组合：

| 类型 | 采样（输出端） | 叠加（输入端） | 俗称 | 典型例子 |
|------|:-----------:|:-----------:|------|---------|
| 电压串联 | 电压 $V_o$ | 电压串联 | 电压放大器 | 同相放大器 |
| 电压并联 | 电压 $V_o$ | 电流并联 | 跨阻放大器 | 反相放大器 |
| 电流串联 | 电流 $I_o$ | 电压串联 | 跨导放大器 | 射极电阻负反馈 |
| 电流并联 | 电流 $I_o$ | 电流并联 | 电流放大器 | 两级电流反馈 |

记忆：采样端——并联取电压（如电压表），串联取电流（如电流表）。叠加端——串联用 KVL（电压相减），并联用 KCL（电流相减）。

### 2c. 形式化定义：两种采样 + 两种叠加

**输入端的叠加方式：**

- **串联叠加**：反馈信号以电压形式 $V_f$ 与输入电压 $V_i$ 串联。净输入电压 $V_i' = V_i - V_f$。这要求反馈网络的输出**串联**在输入回路中。
- **并联叠加**：反馈信号以电流形式 $I_f$ 与输入电流 $I_i$ 并联。净输入电流 $I_i' = I_i - I_f$。这要求反馈网络的输出**并联**在输入回路中。

**输出端的采样方式：**

- **采样电压**：反馈网络并联在输出端，取输出电压的一部分。反馈网络的输入阻抗应很大，以免影响输出电压。
- **采样电流**：反馈网络串联在输出回路中，取输出电流流过它产生的电压。反馈网络的输入阻抗应很小，以免影响输出电流。

四种组合：

**① 电压串联** — 采样 $V_o$，电压串联叠加 → 闭环电压增益 $A_{vf}$。代表：同相放大器。

```
Vi'─┤基本放大器(A_v)├─┬→Vo     β网络←Vo(采样), →Vf串联回输入端
    └──────────────┘ │
    ┌──────┐←───────┘
    │  β   │→Vf
    └──────┘
```

**② 电压并联** — 采样 $V_o$，电流并联叠加 → 闭环跨阻增益 $R_{mf}$。代表：反相放大器。

```
Ii'─┤基本放大器(R_m)├─┬→Vo     β网络←Vo(采样), →If并联回输入端
    └──────────────┘ │
    ┌──────┐←───────┘
    │  β   │→If
    └──────┘
```

**③ 电流串联** — 采样 $I_o$，电压串联叠加 → 闭环跨导增益 $G_{mf}$。代表：BJT 射极负反馈。

```
Vi'─┤基本放大器(G_m)├→Io→负载    β网络←Io(采样), →Vf串联回输入端
    └──────────────┘   │
    ┌──────┐←─────────┘
    │  β   │→Vf
    └──────┘
```

**④ 电流并联** — 采样 $I_o$，电流并联叠加 → 闭环电流增益 $A_{if}$。代表：两级电流反馈。

```
Ii'─┤基本放大器(A_i)├→Io→负载    β网络←Io(采样), →If并联回输入端
    └──────────────┘   │
    ┌──────┐←─────────┘
    │  β   │→If
    └──────┘
```

### 2d. 手算示例：判断反相放大器的反馈类型

题目：判断经典反相放大器属于哪种负反馈类型。

```
         R2
    ┌────[̅̅R2̅̅]────┐
    │              │
Vi──[R1]──┬──(-)运放──→ Vo
          │    (+)
          │     │
          └─────┘
              GND
```

分析步骤：

**步骤1：看输出端——采样电压还是电流？**
$R_2$ 跨接在输出和反相输入端之间。反馈网络（$R_1$、$R_2$）并联在输出端，取的是 $V_o$ 的分压。这说明**采样的是输出电压**。

**步骤2：看输入端——串联叠加还是并联叠加？**
输入信号 $V_i$ 通过 $R_1$ 注入反相输入端，反馈信号也通过 $R_2$ 注入同一个节点。两个信号在这个节点**以电流形式叠加**（并联）。反相输入端的节点满足 KCL：$I_{R1} + I_{R2} = I_- \approx 0$。所以 $I_f \approx -I_i$。

**结论：电压并联负反馈**。这正是"Shunt-Shunt"拓扑——输入端并联采样电流，输出端并联采样电压。

### 2e. Python 验证：四种反馈拓扑的增益对比

```python
# 四种负反馈拓扑——统一公式 Af = A/(1+Aβ) 的演示
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

A, Ri, Ro, beta = 10000, 100e3, 100, 0.1

# 四种拓扑闭环增益（各自量纲）
Avf  = A / (1 + A * beta)                          # 电压串联
Rmf  = Ri / beta                                   # 电压并联（深度负反馈近似）
Gmf  = (A/Ro) / (1 + (A/Ro) * beta * Ro)          # 电流串联
Aif  = (A*Ri/Ro) / (1 + (A*Ri/Ro) * beta)         # 电流并联

print(f"① 电压串联 Avf = {Avf:.2f} V/V  (理想 1/β = {1/beta:.1f})")
print(f"② 电压并联 Rmf = {Rmf:.0f} Ω")
print(f"③ 电流串联 Gmf = {Gmf:.3f} S")
print(f"④ 电流并联 Aif = {Aif:.2f} A/A")
print(f"反馈深度 1+Aβ = {1+A*beta:.0f}")

# 柱状图对比
fig, ax = plt.subplots(figsize=(9, 5))
labels = ['① 电压串联\n$A_{vf}$', '② 电压并联\n$R_{mf}$', 
          '③ 电流串联\n$G_{mf}$', '④ 电流并联\n$A_{if}$']
vals = [Avf, Rmf/1e5, Gmf, Aif/1e3]
bars = ax.bar(labels, vals, color=['#3498db','#e74c3c','#2ecc71','#f39c12'])
for b, v in zip(bars, [f'{Avf:.1f}', f'{Rmf/1e3:.0f}kΩ', f'{Gmf:.3f}S', f'{Aif:.0f}']):
    ax.text(b.get_x()+b.get_width()/2, b.get_height()+0.2, v, ha='center', fontweight='bold')
ax.set_title('四种拓扑——$A_f=A/(1+A\\beta)$ 统一公式', fontsize=13)
ax.grid(True, alpha=0.3, axis='y'); plt.tight_layout(); plt.show()
```

### 2f. 常见误区

❌ **误区1："电压串联和电压并联只是名字不同，实际一样。"**
✓ 完全不同。电压串联反馈的同相放大器输入阻抗很高（理想无穷大），电压并联反馈的反相放大器输入阻抗等于 $R_1$（通常不高）。选哪种拓扑取决于信号源是电压源还是电流源。

❌ **误区2："四种拓扑可以随便混用。"**
✓ 采样和叠加方式必须匹配。如果采样电压却试图串联叠加电流，反馈网络没法正常工作，电路要么不放大，要么振荡。初学者应先从经典电路（同相、反相、跟随器）出发理解反馈，再尝试自己设计。

### 2g. 应用连接

- **电压串联**：音频前置放大器（高输入阻抗、电压增益精确）
- **电压并联**：光电流放大器（TIA，把光电二极管电流变成电压）
- **电流串联**：LED 驱动（控制电流而非电压）
- **电流并联**：磁头放大器、高精度电流镜

## 3. 闭环增益公式——负反馈的数学核心

### 3a. 生活例子：空调恒温器

你家空调是怎么维持 26°C 的？

- 你设定温度为 26°C（参考输入 $T_{ref}$）
- 温度传感器测出房间实际温度 $T_{room}$（反馈信号）
- 空调控制器计算误差 $e = T_{ref} - T_{room}$
- 误差越大，压缩机功率越大；误差越小，功率越小；误差为零，停机
- 最终稳定在 $T_{room} \approx T_{ref}$

如果把空调的开环能力（把冷气灌进房间的能力）记为 $A$，把温度传感器的反馈比例记为 $\beta$（这里 $\beta=1$，传感器直接读出温度），那么：

实际温度 = $\dfrac{A}{1 + A \cdot 1} \times$ 设定温度

当 $A$ 很大（空调很猛），实际温度 ≈ 设定温度。这就是 $A_f \approx 1/\beta$ 的直觉——跟电路中深度负反馈下增益只由 $\beta$ 决定完全一致。

### 3b. 直观理解：误差驱动的自我纠偏

回到放大器。负反馈的工作过程分四步：

1. **比较**：输入 $V_i$ 和反馈信号 $V_f$ 相减，得到**误差信号** $V_e = V_i - V_f$
2. **放大**：误差信号被基本放大器放大 $A$ 倍：$V_o = A \cdot V_e$
3. **反馈**：输出 $V_o$ 经反馈网络衰减 $\beta$ 倍送回：$V_f = \beta \cdot V_o$
4. **循环**：$V_o$ 变大 → $V_f$ 变大 → $V_e$ 变小 → $V_o$ 回落……直到平衡

把四个方程联立：

$$
\begin{aligned}
V_e &= V_i - V_f \\
V_o &= A \cdot V_e \\
V_f &= \beta \cdot V_o
\end{aligned}
$$

代入消元：

$$
V_o = A(V_i - \beta V_o)
$$

$$
V_o = A V_i - A \beta V_o
$$

$$
V_o + A \beta V_o = A V_i
$$

$$
V_o (1 + A \beta) = A V_i
$$

$$
A_f = \frac{V_o}{V_i} = \frac{A}{1 + A \beta}
$$

**这就是闭环增益公式，负反馈中最重要的一行数学。**

### 3c. 形式化定义：闭环增益公式

对于任何负反馈系统（不仅限于电压放大器），只要满足：

- 前向增益为 $A$（基本放大器的放大倍数）
- 反馈系数为 $\beta$（反馈网络的传输比例）
- 反馈信号与输入信号相减（负反馈）

闭环增益 $A_f$（即整个反馈放大器的增益）为：

$$
\boxed{A_f = \frac{A}{1 + A\beta}}
$$

其中：

| 符号 | 名称 | 含义 |
|------|------|------|
| $A$ | 开环增益（Open-Loop Gain） | 没有反馈时基本放大器的增益 |
| $\beta$ | 反馈系数（Feedback Factor） | 反馈网络将输出送回输入的比例，$0 < \beta \leq 1$ |
| $A\beta$ | 回路增益（Loop Gain） | 信号绕整个环路一周的增益 |
| $1 + A\beta$ | 反馈深度（Amount of Feedback / Desensitivity） | 反馈"有多深"；越大精度越高 |
| $A_f$ | 闭环增益（Closed-Loop Gain） | 加上反馈后整个放大器的增益 |

**深度负反馈**（$A\beta \gg 1$，通常 $A\beta > 100$ 就算深）时的近似：

$$
A_f = \frac{A}{1 + A\beta} \approx \frac{A}{A\beta} = \boxed{\frac{1}{\beta}}
$$

这个近似是模拟电路设计的**超级武器**。它告诉你：**只要 $A$ 足够大，闭环增益就由 $\beta$ 决定，而 $\beta$ 由无源元件（电阻）决定，非常精确和稳定。**

运放的开环增益 $A_{OL}$ 可能随温度漂移 ±30%，但外部电阻 $R_1$ 和 $R_2$ 的精度可以达到 ±1% 甚至 ±0.1%，所以闭环增益的精度远高于开环增益。这就是负反馈最核心的价值。

### 3d. 手算示例：反相放大器精确增益计算

题目：反相放大器，$R_1 = 2\ \text{kΩ}$，$R_2 = 20\ \text{kΩ}$，运放开环增益 $A_{OL} = 80\ \text{dB}$（即 10000 倍）。计算精确闭环增益和深度负反馈近似值的误差。

**步骤1：计算反馈系数 $\beta$**

在反相放大器中，反馈网络是 $R_1$ 和 $R_2$ 的分压。但注意，这里是电压并联反馈，直接用电压串联的 $A_f = A/(1+A\beta)$ 公式需要小心。

对于反相放大器（电压并联），闭环增益为：
$$
A_{vf} = -\frac{R_2}{R_1} \cdot \frac{1}{1 + \frac{1}{A_{OL}}\left(1 + \frac{R_2}{R_1}\right)}
$$

推导较为复杂。我们换一种更直观的方法：利用 $V_+ \approx V_-$（虚短）的精度。

**步骤2：虚短分析**

$V_+ = 0$（接地）。设 $V_-$ 为 $V_x$。
- 流过 $R_1$ 的电流：$I_1 = (V_i - V_x) / R_1$
- 流过 $R_2$ 的电流：$I_2 = (V_o - V_x) / R_2$
- KCL：$I_1 + I_2 = 0$（运放输入电流为零）
- $V_o = -A_{OL} \cdot V_x$

代入：
$$
\frac{V_i - V_x}{R_1} + \frac{V_o - V_x}{R_2} = 0
$$

$$
\frac{V_i}{R_1} - \frac{V_x}{R_1} + \frac{V_o}{R_2} - \frac{V_x}{R_2} = 0
$$

$$
\frac{V_i}{R_1} + \frac{V_o}{R_2} = V_x \left(\frac{1}{R_1} + \frac{1}{R_2}\right)
$$

代入 $V_x = -V_o / A_{OL}$：

$$
\frac{V_i}{R_1} + \frac{V_o}{R_2} = -\frac{V_o}{A_{OL}} \left(\frac{1}{R_1} + \frac{1}{R_2}\right)
$$

$$
\frac{V_i}{R_1} = -V_o \left[ \frac{1}{R_2} + \frac{1}{A_{OL}} \left(\frac{1}{R_1} + \frac{1}{R_2}\right) \right]
$$

$$
A_{vf} = \frac{V_o}{V_i} = -\frac{\frac{1}{R_1}}{\frac{1}{R_2} + \frac{1}{A_{OL}}\left(\frac{1}{R_1} + \frac{1}{R_2}\right)}
$$

**步骤3：代入数值**

$R_1 = 2000\ \Omega$，$R_2 = 20000\ \Omega$，$A_{OL} = 10000$

$$
\frac{1}{R_1} = 5 \times 10^{-4}\ \text{S},\quad \frac{1}{R_2} = 5 \times 10^{-5}\ \text{S}
$$

$$
\frac{1}{R_1} + \frac{1}{R_2} = 5.5 \times 10^{-4}\ \text{S}
$$

$$
A_{vf} = -\frac{5 \times 10^{-4}}{5 \times 10^{-5} + \frac{1}{10000} \times 5.5 \times 10^{-4}}
= -\frac{5 \times 10^{-4}}{5 \times 10^{-5} + 5.5 \times 10^{-8}}
$$

$$
\text{分母} = 5.0 \times 10^{-5} + 0.0055 \times 10^{-5} = 5.0055 \times 10^{-5}
$$

$$
A_{vf} = -\frac{5 \times 10^{-4}}{5.0055 \times 10^{-5}} = -9.989
$$

**步骤4：与理想值比较**

理想值（$A_{OL} \to \infty$）：$A_{vf,ideal} = -R_2/R_1 = -20/2 = -10$

| 项目 | 值 |
|------|-----|
| 理想增益 | -10.000 |
| 精确增益 | -9.989 |
| 误差 | 0.11% |

即使 $A_{OL}$ 只有 10000（80dB），误差也仅约 0.1%。如果是精密运放 $A_{OL} = 10^6$（120dB），误差将小到 0.001% 级别。这就是负反馈的魔力。

### 3e. Python 验证：$A_f$ 公式的精度验证

```python

import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# 参数
R1, R2 = 2000, 20000  # Ω
beta = R1 / (R1 + R2)  # 分压反馈系数
A_ideal_gain = R2 / R1  # 理想反相增益（绝对值）

# 扫 A_OL 从 100 到 1e7
A_range = np.logspace(2, 7, 1000)

# 精确公式：对于反相放大器

Af_exact = -A_ideal_gain / (1 + (1 + A_ideal_gain) / A_range)
Af_approx = -A_ideal_gain  # 深度负反馈近似值

# 画图
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(13, 5))

ax1.set_ylabel('闭环增益 $|A_f|$', fontsize=12)
ax1.set_title('闭环增益随 $A_{OL}$ 的变化', fontsize=13)
ax1.legend(fontsize=10)
ax1.grid(True, alpha=0.3, which='both')

ax2.loglog(A_range, error_pct, 'r-', linewidth=2)
ax2.axhline(y=1, color='gray', linestyle='--', alpha=0.5, label='1% 误差线')

ax2.set_xlabel('开环增益 $A_{OL}$', fontsize=12)
ax2.set_ylabel('相对误差 (%)', fontsize=12)
ax2.set_title('近似值 $A_f ≈ R_2/R_1$ 的误差', fontsize=13)
ax2.legend(fontsize=9)
ax2.grid(True, alpha=0.3, which='both')

# 标注关键点
A_80dB = 10000
idx = np.argmin(np.abs(A_range - A_80dB))
ax2.annotate(f'$A_{{OL}}=10^4$\n误差={error_pct[idx]:.2f}%', 
             xy=(A_80dB, error_pct[idx]),

             arrowprops=dict(arrowstyle='->', color='black'),

plt.tight_layout()
plt.show()

print(f"A_OL = 100 时误差: {error_pct[0]:.4f}%")

print(f"A_OL = 10000 (80dB) 时误差: {error_pct[np.argmin(np.abs(A_range - 10000))]:.4f}%")
print(f"A_OL = 100000 (100dB) 时误差: {error_pct[np.argmin(np.abs(A_range - 100000))]:.4f}%")
```

### 3f. 常见误区

❌ **误区1："$A_f = A/(1+A\beta)$ 对所有放大器都直接适用。"**
✓ 这个公式是通用形式，但 $A$ 和 $\beta$ 的量纲要匹配：电压串联反馈中 $A$ 是电压增益，$\beta$ 无量纲；电压并联反馈中 $A$ 是跨阻（Ω），$\beta$ 是跨导（S），$A\beta$ 才无量纲。用错量纲算出来的结果毫无意义。

❌ **误区2："深度负反馈下 $A_f = 1/\beta$，跟 $A$ 完全无关。"**
✓ 是近似无关，但前提是 $A\beta \gg 1$。如果 $A$ 不够大（比如在接近带宽上限的频率处 $A$ 已经下降），这个近似就不成立了。高频下还是要用完整公式。

### 3g. 应用连接

闭环增益公式是分析任何运放电路的起点。同相放大器 $A_f = 1 + R_2/R_1$，反相放大器 $A_f = -R_2/R_1$，这些公式都是 $A_f \approx 1/\beta$ 在不同 $\beta$ 网络下的特例。

## 4. 负反馈的四大好处

### 4a. 生活例子：把一把烂吉他变成好吉他

假设你有一把很差劲的吉他——音准不稳定，同一根弦每次弹的音高都略有飘移，弹久了音色还变差。你没法换一把好吉他，但你可以怎么办？

你可以在吉他的琴桥上装一个**压电拾音器**，把振动变成电信号，然后：
1. 用一个频率检测电路测出实际音高
2. 和标准音高比较，算出误差
3. 用一个小马达自动调节弦的张力，把音高拉回来

这就是一个**自动调音系统**——本质上就是负反馈。通过它，一把烂吉他的音准变得稳定可靠。

放大器中，负反馈也扮演了完全相同的角色：它把"先天不足"的开环放大器变成了"表现优异"的闭环放大器。具体来说有四大好处。

### 4b. 直观理解：四个维度的改善

**好处①：稳定增益（Gain Desensitization）**

没有反馈时，增益 $A$ 随温度、电源电压、元件老化而变化。加上负反馈后：
- 闭环增益 $A_f \approx 1/\beta$，只由外部无源元件决定
- $\beta$ 由精密电阻分压实现（电阻温度系数可低至 ±25 ppm/°C）
- 增益稳定性提升 $1 + A\beta$ 倍

**好处②：展宽带宽（Bandwidth Extension）**

放大器在高频时增益下降。但负反馈"借"了一部分低频增益去换带宽，使得增益-带宽乘积（GBW）保持不变。闭环后的 -3dB 带宽是开环带宽的 $1 + A\beta$ 倍：

$$
f_{Hf} = f_H (1 + A\beta)
$$

**好处③：减小非线性失真（Distortion Reduction）**

放大器本身有非线性失真（比如晶体管输入小信号时和弦波，大信号时产生谐波）。负反馈像"误差放大器"——它感知到输出有失真（与输入形状不一致），就反向修正。失真降低 $1 + A\beta$ 倍。

**好处④：改变输入/输出阻抗**

- 串联反馈 → 增大输入阻抗（$R_{if} = R_i (1 + A\beta)$）
- 并联反馈 → 减小输入阻抗（$R_{if} = R_i / (1 + A\beta)$）
- 电压采样 → 减小输出阻抗（$R_{of} = R_o / (1 + A\beta)$）
- 电流采样 → 增大输出阻抗（$R_{of} = R_o (1 + A\beta)$）

阻抗的改变让我们可以设计"阻抗匹配"——例如电压跟随器的高输入阻抗和低输出阻抗。

### 4c. 形式化定义：四大好处的数学表达式

| 好处的维度 | 开环 | 闭环 | 改善倍数 |
|-----------|------|------|:------:|
| 增益灵敏度 | $\frac{dA}{A}$ | $\frac{dA_f}{A_f} = \frac{1}{1+A\beta} \cdot \frac{dA}{A}$ | $1+A\beta$ |
| 带宽（上限截止频率） | $f_H$ | $f_{Hf} = f_H(1+A\beta)$ | $1+A\beta$ |
| 谐波失真 | $D$ | $D_f = \frac{D}{1+A\beta}$ | $1+A\beta$ |
| 输入阻抗（串联反馈） | $R_i$ | $R_{if} = R_i(1+A\beta)$ | $1+A\beta$ |
| 输出阻抗（电压采样） | $R_o$ | $R_{of} = \frac{R_o}{1+A\beta}$ | $1+A\beta$ |

注意：这四个"好处"都是同一个 $1+A\beta$ 因子在发挥作用。$1+A\beta$ 越大，好处越明显——但稳定性裕度越紧张。这就是设计上的经典权衡。

### 4d. 手算示例：带宽展宽计算

题目：某运放的开环增益 $A_{OL} = 10^5$（100 dB），开环 -3dB 带宽 $f_H = 10\ \text{Hz}$。用该运放搭建增益为 $A_f = 100$（40 dB）的同相放大器。求闭环带宽。

解：

**步骤1：求反馈系数 $\beta$**

同相放大器 $A_f \approx 1/\beta$（深度负反馈），所以：
$$
\beta = \frac{1}{A_f} = \frac{1}{100} = 0.01
$$

**步骤2：计算反馈深度**

$$
1 + A\beta = 1 + 10^5 \times 0.01 = 1 + 1000 = 1001
$$

**步骤3：闭环带宽**

$$
f_{Hf} = f_H \times (1 + A\beta) = 10 \times 1001 = 10010\ \text{Hz} \approx 10.0\ \text{kHz}
$$

**步骤4：验证增益-带宽积守恒**

$$
\text{开环 GBW} = A_{OL} \times f_H = 10^5 \times 10 = 10^6\ \text{Hz} = 1\ \text{MHz}
$$

$$
\text{闭环 GBW} = A_f \times f_{Hf} = 100 \times 10010 \approx 10^6\ \text{Hz} = 1\ \text{MHz}
$$

增益-带宽积不变（都是 1 MHz），负反馈把低频端多余的增益"分配"到了高频端，换来了更大的带宽。

### 4e. Python 验证：增益-带宽权衡

```python
# 负反馈的增益-带宽权衡——GBW 守恒
import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

A0, f_p = 100000, 10          # 开环 DC 增益 100dB, 主极点 10Hz
GBW = A0 * f_p                 # GBW = 1 MHz
f = np.logspace(0, 7, 10000)
A_ol = A0 / np.sqrt(1 + (f/f_p)**2)

beta_vals = [1.0, 0.1, 0.01, 0.001]  # 对应 Af≈1,10,100,1000
colors = ['#e74c3c','#3498db','#2ecc71','#f39c12']

fig, ax = plt.subplots(figsize=(11, 7))
ax.loglog(f, A_ol, 'k-', linewidth=2.5, alpha=0.7, label='开环 (GBW=1MHz)')
for beta, c in zip(beta_vals, colors):
    Af = 1/beta; f_Hf = GBW/Af
    A_cl = Af / np.sqrt(1 + (f/f_Hf)**2)
    ax.loglog(f, A_cl, color=c, linewidth=2, label=f'Af={Af:.0f}, fH={f_Hf/1e3:.1f}kHz')
    if Af > 1: ax.plot(f_Hf, Af/np.sqrt(2), 'o', color=c, markersize=8)

ax.loglog(f, GBW/f, 'gray', linestyle=':', alpha=0.5, label='GBW 线')
ax.set_xlabel('频率 (Hz)'); ax.set_ylabel('增益 (V/V)')
ax.set_title('增益-带宽权衡：增益越小，带宽越大'); ax.legend(fontsize=9)
ax.grid(True, alpha=0.3, which='both'); ax.set_xlim(1, 1e7); ax.set_ylim(0.5, 2e5)
plt.tight_layout(); plt.show()

for beta in beta_vals:
    Af = 1/beta; f_Hf = GBW/Af
    print(f"Af={Af:6.0f} → fH={f_Hf/1e3:6.1f}kHz → GBW={Af*f_Hf/1e6:.2f}MHz")
```

### 4f. 常见误区

❌ **误区1："负反馈让带宽变大的同时，增益保持不变。"**
✓ 负反馈是用**降低增益**为代价来换带宽的。增益从 $A$ 降到 $A_f$，带宽从 $f_H$ 升到 $f_{Hf}$，但 GBW 不变。你要高增益就别指望宽带宽——这是根本的物理限制。

❌ **误区2："失真被完全消除了。"**
✓ 失真被**减小**了 $1+A\beta$ 倍，没有被消除。而且如果反馈深度不够（比如高频下 $A$ 已经下降），失真改善就不明显。此外，负反馈可能引入新的高频失真（交越失真经反馈后波形变尖锐）。

### 4g. 应用连接

增益-带宽积（GBW）是运放选型的第一参数。你需要多少增益、处理什么频率的信号，就可以算出需要多少 GBW 的运放。这也是为什么高速运放（GBW > 100 MHz）更贵——它们用更先进的工艺来把主极点推高。

## 5. 稳定性问题——负反馈的另一面

### 5a. 生活例子：淋浴水温振荡

还记得我们第1节用淋浴调水温做的类比吗？回到淋浴间：

你打开热水，水是凉的。你等了三秒，水还是凉的。你猛地把热水龙头拧大。五秒后——水变得滚烫。你被烫到了，猛地关小热水。又过了五秒——水变得冰冷。你又开大……如此循环。

发生了什么？**系统的延迟**让负反馈变成了正反馈。你做的每一步修正都是对前一个状态的"过度反应"，而且延迟让你永远在追上一个误差。这就是**不稳定**——系统在振荡，永远到不了平衡点。

在放大器中，同样的故事上演：高频下运放内部产生额外的**相移**，当相移达到 180° 时，负反馈变成正反馈，放大器开始振荡。

### 5b. 直观理解：相移怎么来的

运放内部是由很多晶体管级组成的，每一级都有一个"极点"（相当于一个低通 RC 滤波器）。每个极点在高频处产生 **-90°** 的相移（最大值）。

- 一个极点：最多 -90° 相移 → **绝对稳定**（负反馈 180° - 90° = 90°，还是负反馈）
- 两个极点：最多 -180° 相移 → 临界稳定（理论上可能振荡，但实际上需要第三个极点或增益不足）
- 三个极点：最多 -270° 相移 → **必然存在某个频率处相移刚好 -180°**，此时如果回路增益 $|A\beta| > 1$，就会振荡

所以判断稳定性的核心问题就变成：**在相移达到 180° 的那个频率处，回路增益 $|A\beta|$ 是否还大于 1？**

### 5c. 形式化定义：奈奎斯特稳定性判据（简化版）

**巴克豪森准则**（Barkhausen Criterion）——发生振荡的条件：

$$
\boxed{|A(j\omega)\beta(j\omega)| \geq 1 \quad \text{且} \quad \angle A(j\omega)\beta(j\omega) = -180^\circ \text{（或 } \pm 180^\circ\text{）}}
$$

两个条件必须同时满足：
1. **振幅条件**：回路增益的幅值 ≥ 1（0 dB）
2. **相位条件**：回路的总相移 = 180°（$360^\circ$ 的整数倍也可以，但 180° 是最典型的）

如果两个条件不同时满足——比如幅值 < 1 时相移才到 180°，或者相移还不到 180° 时幅值就降到 1 以下——系统就是稳定的。

**稳定性的两个关键指标：**

**相位裕度 PM**（Phase Margin）：在回路增益幅值 = 1（0 dB）的频率处，相移距离 -180° 还有多远。

$$
\boxed{PM = 180^\circ - |\angle A\beta|_{f=f_{0dB}}|}
$$

**增益裕度 GM**（Gain Margin）：在相移 = -180° 的频率处，回路增益幅值距离 0 dB 还有多远。

$$
\boxed{GM = -20\log_{10}|A\beta|_{f=f_{-180^\circ}} \quad \text{(单位: dB)}}
$$

经验法则：
- **PM > 45°**：稳定，有足够的阻尼，阶跃响应有轻微过冲
- **PM ≈ 60°**：最佳折衷——响应快且过冲小（过冲约 10%）
- **PM < 30°**：危险！大的过冲和振铃（响铃状阻尼振荡）
- **PM < 0°**：振荡！系统不稳定
- **GM > 10 dB**：一般认为有足够的增益裕度

用"飞行员"类比：**相位裕度**好比飞行高度——距离"撞山"（180° 相移）还有多高；**增益裕度**好比引擎推力余量——如果推力降多少就飞不动。

### 5d. 手算示例：三极点系统的稳定性判定

题目：一个放大器的开环传递函数为：
$$
A(s) = \frac{10^5}{\left(1 + \frac{s}{2\pi \times 10}\right)\left(1 + \frac{s}{2\pi \times 10^5}\right)\left(1 + \frac{s}{2\pi \times 10^6}\right)}
$$

反馈系数 $\beta = 0.1$（对应目标闭环增益 10）。判定系统是否稳定。

**步骤1：求各极点的转折频率**

- $f_{p1} = 10\ \text{Hz}$（主极点）
- $f_{p2} = 100\ \text{kHz}$（第二极点）
- $f_{p3} = 1\ \text{MHz}$（第三极点）

**步骤2：分析相移**

- 在 $f_{p1}$ 以下，相移 ≈ 0°
- 在 $f_{p1}$ 到 $f_{p2}$ 之间，主极点贡献约 -90°
- 在 $f_{p2}$ 附近，第二极点开始贡献额外相移（-135° 到 -180°）
- 在 $f_{p3}$ 附近及以上，三个极点总相移 → -270°

**步骤3：找 0dB 穿越频率（单位增益频率）**

DC 回路增益：$A_0 \beta = 10^5 \times 0.1 = 10^4$（80 dB）

增益从 80 dB 开始，以 -20dB/dec 下降（经过 $f_{p1}$ 后）。

穿越 0 dB 的频率 $f_{0dB}$ 满足：$10^4 \times (f_{p1} / f_{0dB}) \approx 1$（单极点区近似）

$$
f_{0dB} \approx 10^4 \times 10\ \text{Hz} = 100\ \text{kHz}
$$

但 $100\ \text{kHz}$ 在 $f_{p2}$ 附近，已进入双极点区。精确计算：

在 $f = f_{0dB}$ 处，前两个极点的幅值近似：
$$
|A\beta|_{f=f_{0dB}} \approx \frac{10^4}{\sqrt{\left(1 + \left(\frac{100\text{k}}{10}\right)^2\right) \times \left(1 + \left(\frac{100\text{k}}{100\text{k}}\right)^2\right)}}
$$

简化估算（数量级）：$f_{0dB} \approx 100\ \text{kHz}$ 左右。

在这个频率处：
- 第一极点贡献约 -90° 相移
- 第二极点贡献约 -45° 相移（正在转折点）
- 第三极点还很远，贡献接近 0°

总相移 ≈ -90° - 45° = -135°

相位裕度：$PM = 180° - 135° = 45°$

**步骤4：结论**

PM ≈ 45°：刚好稳定，但余量不大。如果元件参数漂移，$f_{p2}$ 稍微往低频移一点，PM 可能掉到 30° 以下。对于可靠的设计，通常希望 PM ≥ 60°。

**步骤5：如果 $\beta = 1$（跟随器，增益为 1）**

回路增益 $A_0\beta = 10^5$（100 dB）。$f_{0dB}$ 提高到了：

$$
f_{0dB} \approx 10^5 \times 10 = 1\ \text{MHz}
$$

在 1 MHz 处，三个极点都在贡献相移：
- 第一极点：约 -90°
- 第二极点：约 -84°（$f \gg f_{p2}$）
- 第三极点：约 -45°（正在转折点 $f_{p3}$）

总相移 ≈ -219°。相位裕度 = 180° - 219° = **-39°** → 不稳定！会振荡。

**这就是为什么很多运放 datasheet 上写"单位增益稳定"（Unity-Gain Stable）。** "单位增益稳定"意味着运放内部已经做过频率补偿，使得 $\beta=1$（跟随器）时 PM 仍然 > 45°。如果运放不是单位增益稳定的，就不能当跟随器用。

```python
import matplotlib.pyplot as plt
from scipy import signal

omega = 2 * np.pi * f

for p in poles_rad:
    A_s = A_s / (1 + s / p)

PM = 180 + phase_at_0dB  # 相位裕度

# 将相位对折到 [-180, 180) 范围
phase_wrapped = phase_deg.copy()

ax1.set_ylim(-80, 120)

ax2.axhline(y=-135, color='orange', linestyle=':', linewidth=1, alpha=0.5, label='-135° (PM=45°)')
ax2.axvline(x=f_0dB, color='blue', linestyle=':', alpha=0.6)
ax2.axvline(x=f_180, color='green', linestyle=':', alpha=0.6, 
            label=f'-180° 频率 = {f_180/1e3:.2f} kHz')
ax2.plot(f_0dB, phase_at_0dB, 'bo', markersize=8)
ax2.plot(f_180, -180, 'go', markersize=8)
# 标注相位裕度

ax2.set_ylabel('相位 (度)', fontsize=12)

print(f"低频回路增益: {20*np.log10(A0*beta):.1f} dB")
print(f"0dB 穿越频率: {f_0dB/1e3:.2f} kHz")
print(f"该频率处相位: {phase_at_0dB:.1f}°")
```

### 5f. 常见误区

❌ **误区1："只要加了负反馈，电路就自动稳定。"**
✓ 负反馈本身不保证稳定——恰恰相反，深度负反馈更容易振荡。反馈越深（$\beta$ 越大，$A_f$ 越小），0dB 穿越频率越高，越靠近第二、第三极点，相移越大，稳定性越差。跟随器（$\beta=1$）是稳定性最差的负反馈配置。

❌ **误区2："振荡只在高频发生，低频绝对安全。"**
✓ 多数运放的振荡确实在高频（几百 kHz 到几 MHz），但如果电路中有大电容负载、长走线电感、或者外部的 RC 网络，也可能在较低频率振荡。低速的精密运放也一样可能振荡——只要相位条件满足。

❌ **误区3："相位裕度越大越好，越大越稳定。"**
✓ 过大的相位裕度（PM > 75°）意味着系统过于"保守"，响应变慢，带宽利用率不足。工程上的"甜点"通常是 PM = 45°~60°。PM = 60° 给出最快的稳定时间（阶跃响应无过冲到 10% 过冲之间）。

### 5g. 应用连接

每次画运放电路原理图，你都在隐含地做了一个"稳定性判断"——你选的外部电阻值会影响反馈系数 $\beta$，而 $\beta$ 决定了稳定性。高速运放的数据手册通常会给出不同增益下的推荐反馈电阻值，就是为了保证足够的相位裕度。

## 6. 频率补偿——给"爱振荡"的放大器加阻尼

### 6a. 生活例子：给摇晃的桌子垫纸

餐厅里有一张四条腿不平的桌子，你一碰它就晃。怎么让它不晃？

- 在短的那条腿下面垫张纸（**主极点补偿**——把系统的主导不稳定因素推到低频）
- 或者在桌腿之间加一个横撑（**极点分离**——把两个靠得太近的极点分开）
- 再不济，你用手扶住桌子（**外部补偿**——直接加阻尼）

放大器的频率补偿用的就是这些思路。核心思想：**在运放内部（或外部）加一个电容，制造一个频率很低的"主极点"，让增益在相移积累到危险值之前就降到 0 dB 以下。**

### 6b. 直观理解：主极点补偿的原理

回想三极点系统：三个极点会产生 -270° 相移，在相移达到 -180° 时增益可能还大于 1，于是振荡。

**主极点补偿（Dominant-Pole Compensation）** 的策略：故意在非常低的频率处添加一个极点（称为主极点），使得整个系统的增益斜率在全频段内以 -20 dB/dec 下降，直到增益降到 0 dB。这意味着在 0dB 穿越频率之前，最多只有一个极点在全效工作（相移 ≤ -90°），相位裕度自动 ≥ 90°——绝对稳定。

实际运放中，这个主极点通常是用内部的一个**密勒电容**（Miller Capacitor）来实现的。密勒效应能够用一个小电容（几十 pF）产生等效的大电容效果（因为跨接在高增益级的输入和输出之间），从而把主极点推得很低（比如 10 Hz）。

### 6c. 形式化定义：补偿前后的极点位置变化

补偿前（未补偿运放），可能有两个极点靠得很近：
$$
f_{p1} = 1\ \text{MHz}, \quad f_{p2} = 2\ \text{MHz}
$$
0dB 穿越频率在 10 MHz 左右，两个极点都贡献 -90° 以上相移 → PM 接近 0°。

补偿后（添加主极点）：
$$
f_{p1} = 10\ \text{Hz（补偿产生的主极点）}, \quad f_{p2} = 1\ \text{MHz}, \quad f_{p3} = 5\ \text{MHz}
$$

增益从很低频率（10 Hz）开始以 -20 dB/dec 下降，在 1 MHz 之前就已经降到 0 dB（因为 DC 增益 100 dB，-20 dB/dec 下降，到 0 dB 需要 5 个 decade：$10\ \text{Hz} \times 10^5 = 1\ \text{MHz}$）。所以在 0dB 穿越频率处，第二极点刚开始抬头（贡献 < 45°），总相移约 -90° - 45° = -135°，PM ≈ 45°。这就稳定了。

### 6d. 手算示例：补偿电容的选择

题目：一个未补偿的跨导放大器（OTA，Operational Transconductance Amplifier）有一个高阻抗输出节点，输出极点 $f_{p1} = 100\ \text{kHz}$。需要在输出到地之间加一个电容 $C_c$ 作为主极点，使得单位增益频率 $f_u = 1\ \text{MHz}$，且 PM ≥ 60°。OTA 的跨导 $g_m = 1\ \text{mS}$，输出电阻 $R_o = 100\ \text{kΩ}$。

**步骤1：未补偿时的 DC 增益和主极点**

DC 增益：$A_0 = g_m \times R_o = 0.001 \times 100000 = 100$（40 dB）
未补偿主极点：$f_{p1} = 1 / (2\pi R_o C_{parasitic}) = 100\ \text{kHz}$

**步骤2：添加 $C_c$ 后**

新的主极点：$f_{p1}' = \dfrac{1}{2\pi R_o (C_{parasitic} + C_c)}$

单位增益频率近似（单极点模型）：
$$
f_u \approx A_0 \times f_{p1}' = g_m R_o \times \frac{1}{2\pi R_o C_c} = \frac{g_m}{2\pi C_c}
$$

（此近似假设 $C_c \gg C_{parasitic}$）

**步骤3：为 $f_u = 1\ \text{MHz}$ 选取 $C_c$**

$$
C_c = \frac{g_m}{2\pi f_u} = \frac{0.001}{2\pi \times 10^6} = \frac{10^{-3}}{6.283 \times 10^6} \approx 1.59 \times 10^{-10}\ \text{F} = 159\ \text{pF}
$$

**步骤4：验证相位裕度**

加了 $C_c$ 后的主极点：$f_{p1}' = \dfrac{1}{2\pi \times 100000 \times 159 \times 10^{-12}} \approx 10.0\ \text{kHz}$

在 $f_u = 1\ \text{MHz}$ 时，主极点已经贡献接近 -90° 相移。如果系统中还有一个寄生极点 $f_{p2} = 5\ \text{MHz}$（在更高的频率），则：

总相移 = -90°（主极点）- arctan(1 MHz / 5 MHz) = -90° - 11.3° = -101.3°

PM = 180° - 101.3° = 78.7°（> 60°，满足要求）。

### 6f. 常见误区

❌ **误区1："频率补偿就是加一个电容到地。"**
✓ 主极点补偿通常在芯片内部通过密勒电容实现（电容跨接在高增益级的输入输出之间），而不是简单并电容到地。密勒补偿不仅把主极点推向低频，还通过"极点分离"效应把第二极点推向高频，一举两得。

❌ **误区2："所有运放都需要外部频率补偿。"**
✓ 绝大多数现代运放是**内部补偿**的——补偿电容已经在芯片里面做好了。只有少数高速运放或特殊用途运放才需要在外部加补偿元件。看 datasheet：如果写着"Unit-Gain Stable"，就是内部补偿好的，不需要外部补偿。

### 6g. 应用连接

在高速运放（> 100 MHz GBW）和功率放大器（音频功放）中，频率补偿仍然是一个需要仔细设计的问题。音频功放中的"密勒电容"和"相位超前补偿"网络，就是为了在深度负反馈下保证稳定性。

## 7. 完整实战——从电路搭建到稳定性验证

### 7a. 场景描述

你要设计一个音频前置放大器：
- 输入：麦克风信号，最大 10 mV（峰值）
- 输出：最大 1 V（峰值）
- 所以电压增益 $|A_f| = 100$（40 dB）
- 带宽：20 Hz ~ 20 kHz（音频全频段）
- 运放选型：TL071（GBW = 3 MHz, $A_{OL} = 200\ \text{V/mV} = 2 \times 10^5$，开环 $f_H \approx 15\ \text{Hz}$）

要求：验证该电路在闭环增益 100 下是否稳定，如果不稳定提出补偿方案。

### 7b. 电路设计

选择同相放大器：

$$
A_f = 1 + \frac{R_2}{R_1} = 100
$$

取 $R_1 = 1\ \text{kΩ}$，$R_2 = 99\ \text{kΩ}$（用 100 kΩ 加 1 kΩ 串联近似）。

反馈系数：
$$
\beta = \frac{R_1}{R_1 + R_2} = \frac{1}{1 + 99} = 0.01
$$

### 7c. 稳定性分析

**步骤1：计算低频回路增益**

$$
A_0\beta = 2 \times 10^5 \times 0.01 = 2000 \approx 66\ \text{dB}
$$

**步骤2：GBW 和闭环带宽**

TL071 的 GBW = 3 MHz，所以闭环带宽：
$$
f_{Hf} = \frac{\text{GBW}}{A_f} = \frac{3 \times 10^6}{100} = 30\ \text{kHz}
$$

这个带宽 > 20 kHz，满足音频需求。

**步骤3：稳定性评估**

TL071 是内部补偿的单位增益稳定运放（datasheet 明确标注）。在增益 $A_f = 100$ 下，0dB 穿越频率比跟随器低 100 倍，相位裕度更加充足。经验上，对于单位增益稳定运放，闭环增益 > 1 时的 PM 通常 > 60°，非常安全。

**步骤4：相位裕度验算**

TL071 由内部主极点补偿，开环响应近似为单极点（在 $-20\ \text{dB/dec}$ 频率段内）。对于闭环增益 $A_f = 100$（40 dB），0dB 穿越频率为：
$$
f_{0dB} = \frac{\text{GBW}}{A_f} = 30\ \text{kHz}
$$

而 TL071 的第二极点通常 > 1 MHz，远大于 30 kHz。所以在 $f_{0dB}$ 处，相位裕度接近 90°（因为只有主极点在贡献相移，第二极点还远未起作用）。结论：该系统非常稳定，不需要额外补偿。

当你用运放设计一个负反馈放大器时，按以下清单检查：

| 步骤 | 检查项 | 问题 |
|:---:|--------|------|
| 1 | 增益 | $A_f$ 是否由外部电阻精确决定？$A_{OL}$ 是否足够大（满足精度要求）？ |
| 2 | 带宽 | $f_{Hf} = \text{GBW}/A_f$ 是否覆盖信号频段？ |
| 3 | 稳定性 | 运放是否单位增益稳定？如果不是，你的 $A_f$ 是否大于最小稳定增益？ |
| 4 | 阻抗 | 输入阻抗是否匹配信号源？输出阻抗是否足以驱动负载？ |
| 5 | 噪声 | 反馈电阻是否过大（热噪声）？是否需要在反相输入端加平衡电阻？ |
| 6 | 压摆率 | $SR > 2\pi f_{max} V_{peak}$？如果不够，大信号会失真 |

#### 题1 解答

"负反馈"的"负"指的是**极性相反**——反馈信号与输入信号相位相反，两者**相减**。称之为"负"反馈而非"减"反馈，是因为"负"直接点出了信号的相位关系（反相），而"减"只描述了数学操作。在电路物理中，反馈信号和输入信号在求和节点处以**相反的极性**叠加——这就是"负"的本质。

类比：你骑自行车时，眼睛看到车偏左了（这是"正误差"），大脑发出"往右拐"的指令（这是"负修正"——与误差方向相反）。因此叫"负反馈"而非"减反馈"。

#### 题2 解答

(1) 精确闭环增益：

$$
A_f = \frac{A}{1 + A\beta} = \frac{10000}{1 + 10000 \times 0.05} = \frac{10000}{1 + 500} = \frac{10000}{501} = 19.960
$$

(2) 深度负反馈近似值：

$$
A_f \approx \frac{1}{\beta} = \frac{1}{0.05} = 20.000
$$

(3) 相对误差：

$$
\frac{|19.960 - 20.000|}{20.000} \times 100\% = \frac{0.040}{20} \times 100\% = 0.20\%
$$

反馈深度 $1+A\beta = 501$，精度提升约 500 倍。

#### 题3 解答

- (a) 同相放大器：**电压串联负反馈**。采样输出电压（$R_1$+$R_2$ 分压并联在输出端），以电压形式串联叠加回反相输入端。
- (b) 反相放大器：**电压并联负反馈**。采样输出电压，以电流形式并联叠加回反相输入端。
- (c) BJT 射极跟随器：**电压串联负反馈**。输出在发射极，发射极电压反馈回基极-发射极回路（串联叠加），采样的是输出电压。
- (d) 电压跟随器：**电压串联负反馈**（是(a)的特例，$R_2=0$, $R_1=\infty$，$\beta=1$）。

#### 题4 解答

(1) 低频回路增益：
$$
A_0\beta = 10^5 \times 0.01 = 1000
$$
$$
20\log_{10}(1000) = 60\ \text{dB}
$$

(2) 0dB 穿越频率估算：
从 60 dB 开始，过 $f_{p1}=10\ \text{kHz}$ 后以 -20 dB/dec 下降。
从 10 kHz 到 0 dB（60 dB 下降），需要 $60/20 = 3$ 个 decade。
$$
f_{0dB} \approx 10\ \text{kHz} \times 10^3 = 10\ \text{MHz}
$$

但 $f_{p2}=100\ \text{kHz}$ 远低于 10 MHz，实际 $f_{0dB}$ 会更低（双极点区下降更快，-40 dB/dec）。

更精确估算：在 $f_{p2}$ 处，增益从 $f_{p1}$ 到 $f_{p2}$ 下降了 $20\log_{10}(100/10) = 20\ \text{dB}$。
所以在 $f_{p2}=100\ \text{kHz}$ 处 $\approx 40\ \text{dB}$。
从 100 kHz 开始以 -40 dB/dec 下降，到 0 dB 需要下降 40 dB = 1 个 decade。
$$
f_{0dB} \approx 100\ \text{kHz} \times 10^{1} = 1\ \text{MHz}
$$

在 $f_{0dB} \approx 1\ \text{MHz}$ 处：
- 第一极点（10 kHz）：约 -90°
- 第二极点（100 kHz）：约 -84°（$f=10 \times f_{p2}$，arctan(10) ≈ 84.3°）
- 第三极点（1 MHz）：约 -45°（正在转折点）

总相移 ≈ -90° - 84° - 45° = -219°

(3) PM = 180° - 219° = **-39° → 不稳定！**

(4) $\beta=1$（跟随器）：$A_0\beta = 10^5$（100 dB）。$f_{0dB}$ 更高（接近 $f_{p3}=1\ \text{MHz}$），三段极点都充分贡献相移，更加不稳定。这就是为什么这种运放**不能做跟随器**。

这个例子说明：对于多极点系统，深度负反馈（高 $\beta$、低 $A_f$）反而更容易振荡！

#### 题5 解答

(1) 反馈系数 $\beta$：

反相放大器中，反馈网络分压比：
$$
\beta = \frac{R_1}{R_1 + R_2} = \frac{500}{500 + 50000} = \frac{500}{50500} = \frac{1}{101} \approx 0.00990
$$

(2) 精确闭环增益：

利用第3d节推导的精确公式：
$$
A_{vf} = -\frac{\dfrac{1}{R_1}}{\dfrac{1}{R_2} + \dfrac{1}{A_{OL}}\left(\dfrac{1}{R_1} + \dfrac{1}{R_2}\right)}
$$

$A_{OL} = 120\ \text{dB} = 10^6$

$$
\frac{1}{R_1} = \frac{1}{500} = 0.002
$$
$$
\frac{1}{R_2} = \frac{1}{50000} = 2 \times 10^{-5}
$$
$$
\frac{1}{R_1} + \frac{1}{R_2} = 0.002 + 0.00002 = 0.00202
$$
$$
\frac{1}{A_{OL}} \cdot \left(\frac{1}{R_1} + \frac{1}{R_2}\right) = 10^{-6} \times 0.00202 = 2.02 \times 10^{-9}
$$
$$
A_{vf} = -\frac{0.002}{2 \times 10^{-5} + 2.02 \times 10^{-9}} = -\frac{0.002}{2.000202 \times 10^{-5}} = -99.990
$$

(3) 近似值：$A_f \approx -R_2/R_1 = -50000/500 = -100.000$

相对误差：
$$
\frac{|-99.990 - (-100.000)|}{|-100.000|} \times 100\% = \frac{0.010}{100} \times 100\% = 0.010\%
$$

反馈深度极大（$A_{OL}\beta \approx 9900$），误差仅 0.01%，完全可以忽略。

#### 题6 解答

**选运放 A（GBW=1 MHz, 单位增益稳定）。**

理由：

**运放 A 分析**：
- 所需带宽：信号频率 1 kHz 经过增益 1000 倍后，GBW 需求 = 1000 × 1 kHz = 1 MHz。刚好满足。
- 单位增益稳定 → PM ≥ 45°，在 $A_f=1000$ 下稳定性余量更大（更安全）。

**运放 B 分析**：
- GBW = 10 MHz，带宽充裕。
- 但最小稳定增益 = 5，意味着当 $A_f < 5$ 时就可能振荡。虽然 $A_f=1000$ 远大于 5，理论上稳定。但：
  - 非单位增益稳定的运放通常内部极点更多，高频特性复杂，设计难度大。
  - 如果以后要改版、降增益（比如改成 $A_f=2$），电路就会振荡。
  - 对于音频应用（1 kHz），1 MHz GBW 完全足够，不需要 10 MHz 的高速器件。高速运放更贵、功耗更大、噪声可能更高。

**结论**：运放 A 是更合理的选择。节省成本，设计简单，稳定性有保障。

如果必须用运放 B 且需要低增益，可以考虑外部频率补偿（在反馈电阻上并联一个小电容，产生相位超前），但增加了设计复杂度。

#### 题7 解答

(1) **电流串联负反馈**。

- 采样输出电流：集电极电流 $I_c$ 流过 $R_E$ 产生反馈电压 $V_E = I_E \times R_E \approx I_c \times R_E$。
- 串联叠加：$V_{BE} = V_{in} - V_E$，反馈电压 $V_E$ 与输入 $V_{in}$ 串联相减。
- 输出端采样的是电流（$I_c$ 流过 $R_E$），输入端叠加方式是串联（电压相减）。

(2) 与反馈理论的对应：

无反馈时（$R_E = 0$）：输入是 $V_{BE}$，跨导 $g_m = I_c/V_{BE}$，电压增益 $A_v = -g_m R_C$。
反馈引入后：反馈网络 $\beta = R_E$（因为 $V_f = I_c \times R_E$）。

电流串联反馈的闭环跨导：
$$
G_{mf} = \frac{I_c}{V_{in}} \approx \frac{g_m}{1 + g_m R_E}
$$

闭环电压增益：
$$
A_{vf} = -G_{mf} \cdot R_C = -\frac{g_m R_C}{1 + g_m R_E} \approx -\frac{R_C}{R_E} \quad (\text{当 } g_m R_E \gg 1)
$$

这与我们在第30章学到的公式 $A_v \approx -R_C / R_E$ 完全一致。

(3) 对阻抗的影响：

- **输入阻抗增大**：串联反馈使基极看进去的阻抗增大 $(1 + g_m R_E)$ 倍。$R_{in} \approx r_\pi (1 + g_m R_E)$。原 $r_\pi = \beta/g_m$，反馈后显著增大。
- **输出阻抗增大**：电流采样使集电极看进去的输出阻抗增大（$R_{out} \approx r_o(1 + g_m R_E)$，但这在共射放大器中通常不太明显，因为 $R_C$ 并联在输出端）。

---

#### 题8 解答

PM 不是越大越好。**响应速度与稳定性是一对矛盾。**

- **PM = 90°**：系统是过阻尼的，阶跃响应没有过冲，但上升时间很慢。系统过于"保守"，带宽利用率低——同样的 GBW，你得到的可用带宽更少。
- **PM = 60°**：最佳折衷点。过冲约 10%，上升时间约为 $0.35/f_{0dB}$，稳定时间（Settling Time）最短。这是工程实践中普遍推荐的值。
- **PM = 45°**：过冲约 23%，有一点振铃，但系统仍然是稳定的。在低速、对过冲不敏感的应用中可以接受。
- **PM < 30°**：过冲 > 40%，振铃严重，接近不稳定边缘。一般不接受。

具体来说，运放数据手册通常保证 PM ≥ 45°（在最坏情况下——温度、电源电压、工艺偏差）。典型值 PM = 60°～70°。这不是因为做不到更高，而是因为设计者要平衡响应速度。

类比：汽车的悬挂系统。太软的悬挂（像老式美国轿车）过坑很舒适但转弯侧倾大；太硬的悬挂（像赛车）转弯精确但过坑颠簸。工程上永远是权衡——设计不是追求某个单维度的极致，而是找到一个满足所有要求的平衡点。

</details>

## 本章总结

恭喜！你学会了模拟电路中最核心的设计思想之一——负反馈。

| 你学到的 | 一句话总结 |
|---------|-----------|
| 负反馈的本质 | 把输出的一部分反相送回输入，持续纠偏，让系统稳定在期望值 |
| 闭环增益公式 | $A_f = A/(1 + A\beta)$，深度负反馈时 $A_f \approx 1/\beta$——增益由电阻决定 |
| 四种拓扑 | 电压串联、电压并联、电流串联、电流并联——按"采样什么+怎么叠加"分类 |
| 四大好处 | 稳定增益、展宽带宽、减小失真、改变阻抗——都乘以因子 $(1+A\beta)$ |
| 稳定性判据 | PM > 45° 稳定，PM < 0° 振荡；多极点和深度负反馈是振荡的根源 |
| 频率补偿 | 主极点补偿：在低频处加一个极点，让增益在相移变危险前降到 0 dB 以下 |
| 设计清单 | 增益→带宽→稳定性→阻抗→噪声→压摆率，按顺序检查你的放大器 |

负反馈思想不只属于电路——它是控制理论的基础，在机械、化工、经济、生物等各个领域都有应用。你学会的 PM（相位裕度）、GM（增益裕度）、反馈深度这些概念，在任何一个控制系统中都通用。**你已经掌握了一个跨越学科的核心思维工具。**

# 第14章：二阶RLC电路与振荡

> **核心问题**：电路也会像秋千一样"来回摇摆"吗？

---

## 0. 本章导览

前面两章你学了一阶电路：RC 和 RL。它们的响应是单调的指数曲线。电容电压从 0 慢慢爬到终值，不会超过终点再折回来。电感电流也类似。一阶电路的响应就像你把一壶水烧开——温度只会一直上升直到沸腾，不会在中间忽冷忽热。

但真实世界里还有另一种更令人着迷的行为：**振荡（Oscillation）**。秋千来回摆、钟摆左右晃、吉他弦上下振动。这些都不是单调的，而是"过了头又折回来"的。

把电容 C 和电感 L 放在同一个电路里，你就造了一个"电秋千"。能量在电容的电场和电感的磁场之间来回转换，就像秋千在势能和动能之间来回转换。如果电路里有电阻 R，这个振荡会慢慢衰减下来；如果电阻为零，它会永远振荡下去，永不停歇。

这一章比前两章难度上一个台阶——你不是在分析"一个储能元件"的行为，而是在分析"两个储能元件相互配合"的行为。这引入了一个全新的维度：**振荡**。但别怕，你手里已经有了一阶电路的全部工具（KVL、KCL、微分方程、指数函数），这一章只是把它们组合起来用。

学完本章，你将能够：

1. 写出 RLC 串联电路的二阶微分方程，理解每一项的物理意义
2. 解特征方程 $s^2 + (R/L)s + 1/(LC) = 0$，根据根的类型判断系统的阻尼模式
3. 区分过阻尼、临界阻尼和欠阻尼三种响应，画出各自的波形特征
4. 对欠阻尼系统计算衰减系数 $\alpha$、谐振频率 $\omega_0$ 和阻尼振荡频率 $\omega_d$
5. 理解品质因数 Q 的物理意义（储能 vs 耗能的比值）
6. 用 Python 仿真三种阻尼模式，直观对比 R 对振荡行为的影响
7. 为后续学习交流稳态分析、滤波器和控制系统打好基础

## 1. 秋千、钟摆和 RLC——能量的来回转换

### 1a. 生活例子：荡秋千

想象你坐在一个秋千上。有人推了你一把，你开始前后摆动。

整个过程里，能量在两种形式之间不停地转换：

- 当你摆到**最高点**时，速度为零，高度最高。这时候你的能量全是**势能（Potential Energy）**。它储存在你相对于地面的"位置"里。
- 当你摆过**最低点**时，速度最快，高度最低。这时候你的能量全是**动能（Kinetic Energy）**。它储存在你的"运动"里。
- 在最高点和最低点之间，势能和动能同时存在，互相转换，此消彼长。

如果没有任何摩擦（空气阻力、轴承摩擦），秋千会永远摆下去，能量在势能和动能之间"弹来弹去"，幅度永不减小。这种永不衰减的来回运动就是**无阻尼振荡（Undamped Oscillation）**。

真实世界里当然有摩擦。每次摆动，空气阻力会吃掉一点点能量。摆幅一次比一次小，最终停下。这就是**阻尼振荡（Damped Oscillation）**。

### 1b. RLC 电路里的能量"秋千"

RLC 串联电路里的情况和秋千几乎一模一样：

- **电容（Capacitor）**储存的是**电场能**：$W_C = \frac{1}{2} C v_C^2$。电场能依赖于"电压差"，就像势能依赖于"高度差"。电容电压 $v_C$ 高，就相当于秋千摆到了高处。
- **电感（Inductor）**储存的是**磁场能**：$W_L = \frac{1}{2} L i_L^2$。磁场能依赖于"电流在流动"，就像动能依赖于"秋千在运动"。电感电流 $i_L$ 大，就相当于秋千猛冲过最低点。

在一个完整的振荡周期中，能量流转的过程是这样的：

1. **电容充满电**（$v_C$ 最大，$i_L = 0$）。全部能量是电场能 $W_C = \frac{1}{2} C v_{C,\max}^2$。这个时刻，秋千在最高点（势能最大，动能为零）。
2. 电容通过电感**放电**。电容电压开始下降，电感电流开始上升。电场能逐渐转化为磁场能。秋千从高处往下冲，势能转化动能。
3. **电容电压降到零**（$v_C = 0$，$i_L$ 最大）。全部能量变成了磁场能 $W_L = \frac{1}{2} L i_{L,\max}^2$。秋千冲过最低点（动能最大，势能为零）。
4. 电感利用储存的磁场能，继续推着电流给电容**反向充电**。磁场能又逐渐转化回电场能。秋千靠惯性继续往上摆，动能转化回势能。
5. **电容反向充满**（$v_C$ 负最大，$i_L = 0$）。全部能量变回电场能。秋千到了另一侧的最高点。
6. 然后又**反向放电**……循环往复。

```
电场能 (电容)  ←→  磁场能 (电感)

v_C 最大，i_L = 0   ←→   v_C = 0，i_L 最大
（秋千最高点，全势能） ←→ （秋千最低点，全动能）
```

这就是**电振荡（Electrical Oscillation）**。电容和电感就像两个"能量蓄水池"，能量在这两个池子之间轮流存储和释放。

### 1c. 电阻 R 的角色：阻尼

如果秋千完全无摩擦（理想情况），它会永远以同一幅度摆下去。RLC 电路也类似——如果 $R = 0$（理想无电阻），能量在 C 和 L 之间来回转换，总能量守恒，振荡永不衰减。

但真实电路里总有电阻。电阻 $R$ 扮演的就是"摩擦"的角色。每一轮振荡中，电阻把一部分电磁能转化成热能散掉，所以振荡的幅度一次比一次小，最终趋于零。

根据 R 的大小，系统表现出三种截然不同的行为：

- **R = 0**（理想无阻尼）：永远等幅振荡，频率 $\omega_0 = 1/\sqrt{LC}$。只存在于教科书里。
- **R 较小**（欠阻尼）：衰减振荡。来回摆好几个周期后才明显衰减。就像有轻微摩擦的秋千。
- **R 中等**（临界阻尼）：刚好不振荡，以最快速度回到稳态。就像秋千的摩擦刚好大到让它回到中间就停住，不多摆。
- **R 较大**（过阻尼）：根本荡不起来。电容慢慢泄放电荷，单调衰减，没有一次来回。就像把秋千泡在蜂蜜里推。

下面我们就用数学把这些"感觉"精确地描述出来。

## 2. RLC 串联电路的微分方程

### 2a. 电路和基本方程

考虑一个 R、L、C 串联的回路，外接一个直流电压源 $V_s$（单位：伏特，V）。假设 $t=0$ 时闭合开关。

```
         R          L
    ───[~~~]────(~~~~)────┬────
    │                     │
   (+)                    │
    │  Vs                ( ) C
   (-)                    │
    │                     │
    └─────────────────────┘
```

图中 `[~~~]` 代表电阻 R，`(~~~~)` 代表电感 L，`( )` 代表电容 C。

根据 KVL（基尔霍夫电压定律），沿回路绕一圈，各元件电压之和等于电源电压：

$$
V_s = v_R(t) + v_L(t) + v_C(t)
$$

因为是串联回路，流过所有元件的电流相同，记作 $i(t)$（单位：安培，A）。代入三个元件的 V-I 关系：

- 电阻：$v_R = i \cdot R$
- 电感：$v_L = L \cdot \dfrac{di}{dt}$
- 电容：$i = C \cdot \dfrac{dv_C}{dt}$

代入 KVL：

$$
V_s = R\, i(t) + L\, \frac{di}{dt} + v_C(t) \tag{1}
$$

### 2b. 统一用 $v_C$ 表达——得到二阶方程

现在这个方程里同时有 $i$、$di/dt$ 和 $v_C$，混在一起不好解。我们的策略是：**用一个变量表达所有量**。选择电容电压 $v_C$ 作为这个"主变量"，因为 $i = C \cdot dv_C/dt$。

把电流换成电容电压的表达式：

- $i = C \cdot \dfrac{dv_C}{dt}$
- $\dfrac{di}{dt} = C \cdot \dfrac{d^2 v_C}{dt^2}$
- $v_R = R \cdot i = RC \cdot \dfrac{dv_C}{dt}$

全部代回方程 (1)：

$$
V_s = RC \frac{dv_C}{dt} + LC \frac{d^2 v_C}{dt^2} + v_C(t)
$$

整理，把最高阶导数项放在最前面：

$$
\boxed{LC \frac{d^2 v_C}{dt^2} + RC \frac{dv_C}{dt} + v_C(t) = V_s}
$$

这就是 RLC 串联电路的**二阶线性常系数微分方程（Second-Order Linear Constant-Coefficient Differential Equation）**。

- **二阶（Second-Order）**：最高导数是二阶 $d^2 v_C/dt^2$。电路里有两个独立的储能元件（C 和 L），所以需要二阶方程来描述。
- **线性（Linear）**：$v_C$ 以及它各阶导数的次数都是 1，没有 $v_C^2$ 或 $(dv_C/dt)^2$ 这类项。
- **常系数（Constant Coefficient）**：R、L、C 都是不随时间变化的常数。

为了方便后续分析，将方程两边除以 $LC$，化成标准形式：

$$
\boxed{\frac{d^2 v_C}{dt^2} + \frac{R}{L} \frac{dv_C}{dt} + \frac{1}{LC} v_C(t) = \frac{V_s}{LC}}
$$

### 2c. 每一项的物理意义

这个方程在"说"什么？让我们逐项拆解：

| 项 | 数学 | 物理意义 |
|----|------|---------|
| 二阶导数项 | $\dfrac{d^2 v_C}{dt^2}$ | 电感"惯性"的体现。电感抗拒电流变化，这种抗拒表现为对电压二阶导数的贡献 |
| 一阶导数项 | $\dfrac{R}{L}\dfrac{dv_C}{dt}$ | 电阻"阻尼"的体现。电阻消耗能量，让电压变化"慢下来" |
| 零阶项 | $\dfrac{1}{LC} v_C(t)$ | 电容"恢复力"的体现。电容电压偏离平衡时，电容有把它拉回来的趋势 |
| 右端项 | $\dfrac{V_s}{LC}$ | 外部电源的"驱动力"。电源一直在推这个系统 |

这个结构——二阶导数 + 一阶导数 + 零阶 = 外力——在自然界中反复出现。下文你会看到它在力学里的孪生兄弟。

### 2d. 力学类比：弹簧-质量-阻尼系统

RLC 电路不是孤独的。把下面的力学方程和上面的电路方程放在一起看：

**力学方程：**

$$
m \frac{d^2 x}{dt^2} + b \frac{dx}{dt} + kx = F_{\text{ext}}
$$

- $m$：质量（惯性），单位 kg
- $b$：阻尼系数（摩擦），单位 N·s/m
- $k$：弹簧劲度系数（恢复力），单位 N/m
- $x$：位移，单位 m
- $F_{\text{ext}}$：外力，单位 N

**电学方程：**

$$
L C \frac{d^2 v_C}{dt^2} + RC \frac{dv_C}{dt} + v_C = V_s
$$

或者用 $q = C v_C$（电荷）做变量更清晰：

$$
L \frac{d^2 q}{dt^2} + R \frac{dq}{dt} + \frac{1}{C} q = V_s
$$

一一对应关系：

| 电学量 | 符号 | 力学量 | 符号 | 为什么对应 |
|--------|------|--------|------|-----------|
| 电感 | L | 质量 | m | 都是"惯性"——抗拒状态变化 |
| 电阻 | R | 阻尼系数 | b | 都是"摩擦"——消耗能量 |
| 1/电容 | 1/C | 弹簧劲度 | k | 都是"恢复力"——把系统拉回平衡 |
| 电荷 | q | 位移 | x | 都是"累积量" |
| 电流 | i = dq/dt | 速度 | v = dx/dt | 都是"累积量的变化率" |
| 电源电压 | $V_s$ | 外力 | $F_{\text{ext}}$ | 都是"驱动力" |
| 电场能 $\frac{1}{2}C v_C^2$ | — | 弹性势能 $\frac{1}{2}kx^2$ | — | 都是"位置能量" |
| 磁场能 $\frac{1}{2}L i_L^2$ | — | 动能 $\frac{1}{2}mv^2$ | — | 都是"运动能量" |

这不是巧合。物理学里，很多不同领域的系统都服从同样的二阶微分方程。你学会了分析 RLC 电路，你就同时学会了分析弹簧振子、建筑物在地震中的摇摆、汽车悬挂系统的颠簸、甚至某些经济周期模型。这就是数学的力量——剥离具体对象，提取共同结构。

## 3. 特征方程——两根定乾坤

### 3a. 从微分方程到代数方程

像一阶电路一样，我们关心的是"零输入响应"（电容初始有电、电感初始有电流，但没有外接电源）。把 $V_s = 0$，得到齐次方程：

$$
\boxed{\frac{d^2 v_C}{dt^2} + \frac{R}{L} \frac{dv_C}{dt} + \frac{1}{LC} v_C = 0}
$$

对于常系数线性齐次微分方程，解的形式永远是 $v_C(t) = K e^{st}$（指数函数），其中 $s$ 是待定常数（可以是实数，也可以是复数），$K$ 也是待定常数。

为什么是指数函数？因为指数函数的导数是它自己乘一个常数，正好满足"导数之间相互抵消"的需求。代入试一试：

$$
\frac{d}{dt} e^{st} = s e^{st},\qquad \frac{d^2}{dt^2} e^{st} = s^2 e^{st}
$$

把 $v_C = K e^{st}$ 代入齐次方程：

$$
s^2 K e^{st} + \frac{R}{L} s K e^{st} + \frac{1}{LC} K e^{st} = 0
$$

提取公因子 $K e^{st}$（它永远不为零，所以可以安全地消掉）：

$$
\boxed{s^2 + \frac{R}{L}s + \frac{1}{LC} = 0}
$$

这就是 RLC 电路的**特征方程（Characteristic Equation）**。它把微分方程变成了一个简单的一元二次代数方程。$s$ 的两个根（叫**特征根**，Characteristic Roots）完全决定了系统的动态行为。

### 3b. 求解特征根

这是一个标准的一元二次方程 $a s^2 + b s + c = 0$，其中 $a=1$，$b=R/L$，$c=1/(LC)$。求根公式：

$$
s_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} = -\frac{R}{2L} \pm \sqrt{\left(\frac{R}{2L}\right)^2 - \frac{1}{LC}}
$$

为书写简洁和物理意义清晰，定义两个极其重要的参数：

- **衰减系数（Neper Frequency）**：$\alpha = \dfrac{R}{2L}$，单位是 $\text{s}^{-1}$（每秒）。它描述振荡包络衰减的快慢。
- **谐振频率（Resonant Frequency）**：$\omega_0 = \dfrac{1}{\sqrt{LC}}$，单位是 $\text{rad/s}$（弧度每秒）。它描述无阻尼（R=0）时的自然振荡频率。

用这两个参数，特征根可以简写成：

$$
\boxed{s_{1,2} = -\alpha \pm \sqrt{\alpha^2 - \omega_0^2}}
$$

这两个根就是 RLC 电路的"基因"。根号里面东西的符号（正、零、负）决定了系统的命运。

### 3c. 三种情况：根号里的符号决定一切

根号里 $\alpha^2 - \omega_0^2$ 的符号决定了 $s_1$ 和 $s_2$ 是什么样的数：

| 条件 | $\alpha^2 - \omega_0^2$ 的符号 | 特征根类型 | 响应类型 |
|------|-------------------------------|-----------|---------|
| $\alpha > \omega_0$ | **正** | 两个不相等的**负实数** | **过阻尼**（Overdamped） |
| $\alpha = \omega_0$ | **零** | 一个**重根**（负实数） | **临界阻尼**（Critically Damped） |
| $\alpha < \omega_0$ | **负** | 一对**共轭复数** | **欠阻尼**（Underdamped） |

用 R、L、C 的具体值表达判断条件，因为 $\alpha = R/(2L)$ 而 $\omega_0 = 1/\sqrt{LC}$：

$$
\alpha^2 - \omega_0^2 = \left(\frac{R}{2L}\right)^2 - \frac{1}{LC}
$$

- **过阻尼**：$\left(\dfrac{R}{2L}\right)^2 > \dfrac{1}{LC}$，即 $R > 2\sqrt{\dfrac{L}{C}}$
- **临界阻尼**：$\left(\dfrac{R}{2L}\right)^2 = \dfrac{1}{LC}$，即 $R = 2\sqrt{\dfrac{L}{C}}$
- **欠阻尼**：$\left(\dfrac{R}{2L}\right)^2 < \dfrac{1}{LC}$，即 $R < 2\sqrt{\dfrac{L}{C}}$

数值 $R_{\text{crit}} = 2\sqrt{L/C}$ 叫**临界电阻（Critical Resistance）**。它是过阻尼和欠阻尼的分界线。

一个有用的记忆法：$R_{\text{crit}}$ 只和 $L/C$ 有关——$L$ 越大（惯性越强），需要更大的 $R$ 才能抑制振荡；$C$ 越大（"弹性"越弱，因为 $1/C$ 对应弹簧劲度），用更小的 $R$ 就能抑制振荡。

## 4. 过阻尼响应（Overdamped，$\alpha > \omega_0$）

### 4a. 解的形式

两个不相等的负实根 $s_1$ 和 $s_2$（都小于 0）。齐次通解是两个独立指数函数的叠加：

$$
\boxed{v_{C,h}(t) = A_1 e^{s_1 t} + A_2 e^{s_2 t}}
$$

其中 $A_1$、$A_2$ 是待定系数，由两个初始条件确定：
- $v_C(0^+)$：换路瞬间电容电压的值（利用"电容电压不能突变"原则）
- $\dfrac{dv_C}{dt}\Big|_{t=0^+} = \dfrac{i_L(0^+)}{C}$：换路瞬间的导数（利用"电感电流不能突变"原则）

$s_1$ 和 $s_2$ 都是负数，所以两个指数项都随时间衰减。但 $|s_1|$ 和 $|s_2|$ 不一样大——一个大一个小的负根意味着两个衰减速度不同的指数函数在叠加。

### 4b. 物理表现

过阻尼的物理图像：电阻太大，能量刚想从电容往电感里转，就被电阻吃掉一大半。剩下的能量不够"推"电感产生足够的磁场能再转回来，所以电压单调地、平缓地"滑"向稳态，没有一次来回。

过阻尼的特点是**慢**。$R$ 越大，阻尼越强，响应越慢。极端情况：$R \to \infty$（断路），电路退化成电容通过无限大电阻放电——实际上根本放不了。

### 4c. "两段式"衰减

过阻尼和单纯一阶 RC/RL 衰减有一个重要区别：一阶电路只有一个时间常数，曲线从头到尾都是一个指数在衰减。过阻尼有两个特征根，所以是两个指数函数在叠加。由于 $|s_1| \neq |s_2|$，早期由"快根"（绝对值大的那个）主导，后期由"慢根"（绝对值小的那个）主导。曲线上能看到一个"拐弯"——早期快速下降一段，然后转入更平缓的慢速下降。这种"两段式"衰减是二阶系统独有的特征。

---

## 5. 临界阻尼响应（Critically Damped，$\alpha = \omega_0$）

### 5a. 解的形式

当 $\alpha = \omega_0$ 时，根号里为 0，特征方程有重根：$s_1 = s_2 = -\alpha$。这时候两个指数函数 $e^{s_1 t}$ 和 $e^{s_2 t}$ 完全一样，线性相关，不能构成通解的两个独立部分。数学上，重根的通解要多乘一个 $t$：

$$
\boxed{v_{C,h}(t) = (A_1 + A_2 t) \, e^{-\alpha t}}
$$

$A_1$ 和 $A_2$ 同样由两个初始条件确定。

### 5b. 物理表现

临界阻尼是工程上追求的"黄金状态"：**在不产生振荡的前提下，以最快的速度回到稳态。**比过阻尼快（因为阻尼小了一些），又没有欠阻尼那种来回摆的"过冲"。

很多仪表的指针阻尼就设计在临界阻尼附近。万用表的表针通上电后，你希望它快速到位，不要晃来晃去让你等半天才读数。这个"不晃、最快到位"的状态就是临界阻尼。

从控制工程的角度看，临界阻尼是"响应速度"和"稳定性"之间的完美平衡点。

---

## 6. 欠阻尼响应（Underdamped，$\alpha < \omega_0$）——真正的振荡！

### 6a. 解的形式

当 $\alpha < \omega_0$ 时，根号里是负数，特征根为一对共轭复数。先定义一个新参数：

$$
\boxed{\omega_d = \sqrt{\omega_0^2 - \alpha^2}}
$$

$\omega_d$ 叫**阻尼振荡频率（Damped Natural Frequency）**，单位是 rad/s。它是有阻尼时的实际振荡频率。

则特征根为：

$$
s_{1,2} = -\alpha \pm j\omega_d
$$

其中 $j = \sqrt{-1}$ 是虚数单位。（电工里通常用 $j$ 而不是数学中的 $i$，因为 $i$ 已经被电流占用了。读作"j"，和英文里字母 J 的发音一样。）

一对共轭复根产生的齐次通解是指数衰减包络乘以正弦/余弦振荡：

$$
\boxed{v_{C,h}(t) = e^{-\alpha t} \left[B_1 \cos(\omega_d t) + B_2 \sin(\omega_d t)\right]}
$$

$B_1$ 和 $B_2$ 由初始条件确定。这个形式也可以等价的写成幅度-相位形式：

$$
\boxed{v_{C,h}(t) = K e^{-\alpha t} \cos(\omega_d t + \phi)}
$$

其中 $K = \sqrt{B_1^2 + B_2^2}$ 是初始振幅，$\phi = -\arctan(B_2/B_1)$ 是初始相位。

### 6b. 解的拆解：包络 × 振荡

把解拆成两部分看，物理意义非常清晰：

- **$e^{-\alpha t}$**：这是一个指数衰减的**包络（Envelope）**。它决定了振荡的"外轮廓"如何随时间减小。和一阶 RC/RL 一样，时间常数 $\tau = 1/\alpha = 2L/R$。经过 $\tau$ 秒，包络衰减到约 37%；经过 $5\tau$，包络衰减到不到 1%。

- **$\cos(\omega_d t + \phi)$**：这是一个频率为 $\omega_d$ 的正弦振荡。它决定了"来回摆"的快慢。

两者相乘：正弦波的幅度被指数包络"压着"，越来越小，最终趋近于零。这就是我们在示波器上看到的衰减振荡波形。

### 6c. 三个频率参数对比

| 参数 | 符号 | 公式 | 物理意义 |
|------|------|------|---------|
| 谐振频率 | $\omega_0$ | $\dfrac{1}{\sqrt{LC}}$ | 如果没有电阻（R=0），系统会以这个频率永远振荡。是系统的"天性"频率 |
| 衰减系数 | $\alpha$ | $\dfrac{R}{2L}$ | 振荡包络按 $e^{-\alpha t}$ 衰减。$\alpha$ 越大，衰减越快 |
| 阻尼振荡频率 | $\omega_d$ | $\sqrt{\omega_0^2 - \alpha^2}$ | 有阻尼时的**实际**振荡频率。由于电阻的"拖累"，$\omega_d < \omega_0$ |

一个很关键的观察：电阻不仅让振荡**衰减**，还让振荡**变慢**了。$\alpha$ 越大，根号里减掉的越多，$\omega_d$ 越小。当 $\alpha$ 大到等于 $\omega_0$ 时，$\omega_d = 0$，振荡频率降为零——系统不再振荡，恰好进入临界阻尼状态。

### 6d. 数值感受：从欠阻尼走向临界阻尼

固定 $L = 0.1\text{ H}$，$C = 10\text{ }\mu\text{F}$。计算核心参数：

$$
\omega_0 = \frac{1}{\sqrt{0.1 \times 10 \times 10^{-6}}} = \frac{1}{\sqrt{10^{-6}}} = 1000\text{ rad/s}
$$

$$
R_{\text{crit}} = 2\sqrt{\frac{L}{C}} = 2\sqrt{\frac{0.1}{10 \times 10^{-6}}} = 2\sqrt{10^4} = 200\text{ }\Omega
$$

现在变化 R，看 $\alpha$ 和 $\omega_d$ 如何跟着变：

| R ($\Omega$) | $\alpha$ (s$^{-1}$) | $\omega_d$ (rad/s) | 行为描述 |
|-------------|---------------------|--------------------|---------|
| 10 | 50 | $\sqrt{10^6 - 2500} \approx 998$ | 振荡频率几乎等于 $\omega_0$，衰减非常慢 |
| 40 | 200 | $\sqrt{10^6 - 40000} \approx 980$ | 衰减略有加快，频率微微降低 |
| 100 | 500 | $\sqrt{10^6 - 250000} \approx 866$ | 衰减明显加快，频率明显降低 |
| 180 | 900 | $\sqrt{10^6 - 810000} \approx 436$ | 频率不到原来一半，衰减很快 |
| 200 | 1000 | 0 | 临界阻尼，不再振荡 |

感受一下：R 从 10 变到 200（20 倍），$\omega_d$ 从 998 降到 0。电阻的"拖累"效应在接近临界阻尼时变得极其剧烈。

## 7. 品质因数 Q

### 7a. Q 的定义

描述一个谐振电路"品质"好不好的参数，叫**品质因数（Quality Factor）**，记作 $Q$（注意：这和电容的电荷 $Q = Cv$ 不是同一个东西，只是不幸撞了字母）。

对于 RLC 串联电路：

$$
\boxed{Q = \frac{\omega_0 L}{R} = \frac{1}{R} \sqrt{\frac{L}{C}}}
$$

$Q$ 是一个无量纲的数（单位全部约掉了）。

### 7b. Q 的物理意义

$Q$ 的核心定义是**储存能量与消耗能量的比值**：

$$
Q = 2\pi \times \frac{\text{每周期储存的最大能量}}{\text{每周期电阻消耗的能量}}
$$

- **高 Q**（$Q \gg 0.5$）：电阻小，每周期消耗的能量远小于储存的能量。振荡幅度衰减慢，很多个周期后才明显减小。高 Q 还意味着**频率选择性好**（后面交流电路章节会展开）。
- **低 Q**（$Q \approx 0.5$）：接近临界阻尼，振荡一两个周期就没了。
- **$Q < 0.5$**：过阻尼，完全不振。

$Q$ 和 $\alpha$、$\omega_0$ 的关系：

$$
Q = \frac{\omega_0}{2\alpha}
$$

因为 $\alpha = R/(2L)$，所以 $Q = \omega_0 L / R = \omega_0/(2\alpha)$。高 Q 意味着 $\alpha \ll \omega_0$——衰减相对于振荡来说非常慢。

### 7c. Q 值与阻尼类型的对应

| Q 范围 | 阻尼类型 | 行为描述 |
|--------|---------|---------|
| $Q > 0.5$ | 欠阻尼 | 衰减振荡。Q 越大，振荡周期越多 |
| $Q = 0.5$ | 临界阻尼 | 刚好不振荡，最快回到稳态 |
| $Q < 0.5$ | 过阻尼 | 单调衰减，不振 |

当 $Q = 0.5$ 时，$\alpha = \omega_0$，等价于 $R = 2\sqrt{L/C}$，与临界阻尼条件完全一致。

### 7d. 能量视角下的 Q

如果把欠阻尼下的振幅（不是能量）写作 $A(t) = A_0 e^{-\alpha t}$，那么经过一个振荡周期 $T_d = 2\pi/\omega_d$，振幅变为：

$$
\frac{A(t + T_d)}{A(t)} = e^{-\alpha T_d} = e^{-2\pi\alpha/\omega_d}
$$

对于高 Q 情况（$\alpha \ll \omega_0$，$\omega_d \approx \omega_0$），近似有：

$$
\frac{A(t + T_d)}{A(t)} \approx e^{-2\pi/(2Q)} = e^{-\pi/Q}
$$

每周期能量（正比于振幅的平方）衰减到 $e^{-2\pi/Q}$。

拿几个具体数值感受：

- $Q = 5$：每周期振幅衰减到 $e^{-\pi/5} \approx 53\%$，能量到 28%。大约 7 个周期后能量降到 1% 以下。
- $Q = 10$：每周期振幅到 73%，能量到 53%。大约 15 个周期后能量到 1%。
- $Q = 100$：每周期振幅到 97%，能量到 94%。大约 150 个周期后能量才到 1%。

这就解释了为什么高 Q 谐振器（比如石英晶振，Q 可达数万到数十万）能振荡成千上万个周期才明显衰减。

## 8. RLC 解题标准流程

面对任何一个 RLC 电路问题，按以下六步走，不会出错：

**步骤 1：识别电路类型和所求变量**

- 串联还是并联？所求变量是 $v_C$、$i_L$ 还是别的？
- 串联时用回路电流列方程最自然；并联时用节点电压列方程最自然。

**步骤 2：计算核心参数**

- $\alpha = \dfrac{R}{2L}$（串联），或 $\alpha = \dfrac{1}{2RC}$（并联）
- $\omega_0 = \dfrac{1}{\sqrt{LC}}$（串联和并联相同！）
- 比较 $\alpha$ 和 $\omega_0$（或比较 $R$ 和 $2\sqrt{L/C}$），判断阻尼类型

**步骤 3：根据阻尼类型写出通解的形式**

- 过阻尼：$v(t) = A_1 e^{s_1 t} + A_2 e^{s_2 t}$，其中 $s_{1,2} = -\alpha \pm \sqrt{\alpha^2 - \omega_0^2}$
- 临界阻尼：$v(t) = (A_1 + A_2 t) e^{-\alpha t}$
- 欠阻尼：$v(t) = e^{-\alpha t}[B_1 \cos(\omega_d t) + B_2 \sin(\omega_d t)]$，其中 $\omega_d = \sqrt{\omega_0^2 - \alpha^2}$

**步骤 4：找特解（如果有外接激励）**

- 直流激励：特解即直流稳态值。稳态下电容视为断路、电感视为短路。
- 正弦激励（后续章节）：特解为同频率正弦波，用相量法求。

**步骤 5：用初始条件定系数**

- 条件 1：$v_C(0^+)$ 或 $i_L(0^+)$ 的值。利用换路前电路求出，并应用"电容电压不能突变、电感电流不能突变"原则。
- 条件 2：导数 $\dfrac{dv_C}{dt}\Big|_{0^+} = \dfrac{i_L(0^+)}{C}$ 或 $\dfrac{di_L}{dt}\Big|_{0^+} = \dfrac{v_L(0^+)}{L}$。

两个条件，刚好解出通解中的两个待定系数。

**步骤 6：验证**

- 代入 $t=0$：是否满足初始条件？
- 代入 $t \to \infty$：是否趋近直流稳态？
- 阻尼类型判断和实际波形是否一致？（欠阻尼应该振荡，过阻尼应该单调）

## 9. 手算实战

### 9a. 例题 1：判断阻尼类型并写出特征根

**题目**：$R = 400\text{ }\Omega$，$L = 0.2\text{ H}$，$C = 5\text{ }\mu\text{F}$。判断阻尼类型，写出特征根。

**解：**

第 1 步，计算 $\alpha$：

$$
\alpha = \frac{R}{2L} = \frac{400}{2 \times 0.2} = \frac{400}{0.4} = 1000\text{ s}^{-1}
$$

第 2 步，计算 $\omega_0$：

$$
\omega_0 = \frac{1}{\sqrt{LC}} = \frac{1}{\sqrt{0.2 \times 5 \times 10^{-6}}} = \frac{1}{\sqrt{10^{-6}}} = 1000\text{ rad/s}
$$

第 3 步，比较：$\alpha = \omega_0 = 1000$。刚好相等，**临界阻尼**。

第 4 步，特征根：$s_1 = s_2 = -\alpha = -1000\text{ s}^{-1}$（重根）。

**验证**：$R_{\text{crit}} = 2\sqrt{L/C} = 2\sqrt{0.2 / (5 \times 10^{-6})} = 2\sqrt{40000} = 400\text{ }\Omega$。$R = 400 = R_{\text{crit}}$，确认临界阻尼。

---

### 9b. 例题 2：过阻尼的完整求解（零输入响应）

**题目**：RLC 串联电路，$R = 400\text{ }\Omega$，$L = 0.05\text{ H}$，$C = 2\text{ }\mu\text{F}$。初始条件：$v_C(0) = 10\text{ V}$，$i_L(0) = 0$。$t=0$ 时开关闭合，没有外接电源。求 $v_C(t)$ 的完整表达式。

**解：**

**第 1 步：计算 $\alpha$、$\omega_0$，判断阻尼类型。**

$$
\alpha = \frac{R}{2L} = \frac{400}{2 \times 0.05} = \frac{400}{0.1} = 4000\text{ s}^{-1}
$$

$$
\omega_0 = \frac{1}{\sqrt{LC}} = \frac{1}{\sqrt{0.05 \times 2 \times 10^{-6}}} = \frac{1}{\sqrt{10^{-7}}} = \sqrt{10^7} \approx 3162.3\text{ rad/s}
$$

$\alpha = 4000 > 3162.3 = \omega_0$，**过阻尼**。✓

**第 2 步：计算特征根。**

$$
\sqrt{\alpha^2 - \omega_0^2} = \sqrt{4000^2 - 3162.3^2} = \sqrt{16 \times 10^6 - 10 \times 10^6} = \sqrt{6 \times 10^6} \approx 2449.5
$$

$$
s_1 = -\alpha + \sqrt{\alpha^2 - \omega_0^2} = -4000 + 2449.5 = -1550.5\text{ s}^{-1}
$$

$$
s_2 = -\alpha - \sqrt{\alpha^2 - \omega_0^2} = -4000 - 2449.5 = -6449.5\text{ s}^{-1}
$$

$s_1$ 和 $s_2$ 都是负数（必须的，因为是稳定系统）。注意 $|s_2| > |s_1|$，$s_2$ 对应的指数项衰减更快。

**第 3 步：写出通解。**

无外接电源（$V_s = 0$），特解为 0，全解即齐次解：

$$
v_C(t) = A_1 e^{-1550.5 t} + A_2 e^{-6449.5 t}
$$

**第 4 步：用初始条件定系数。**

条件 1：$v_C(0) = 10$ V。

代入 $t=0$：$e^0 = 1$，所以 $A_1 + A_2 = 10$。……①

条件 2：$i_L(0) = 0$，即 $C \cdot \dfrac{dv_C}{dt}\Big|_{t=0} = 0$，所以 $\dfrac{dv_C}{dt}\Big|_{t=0} = 0$。

先求导：

$$
\frac{dv_C}{dt} = -1550.5 A_1 e^{-1550.5 t} - 6449.5 A_2 e^{-6449.5 t}
$$

$t=0$ 时：$-1550.5 A_1 - 6449.5 A_2 = 0$。……②

解这个二元一次方程组：

由 ②：$A_1 = -\dfrac{6449.5}{1550.5} A_2 \approx -4.16 A_2$。

代入 ①：$-4.16 A_2 + A_2 = -3.16 A_2 = 10$，所以 $A_2 \approx -3.16$。

$A_1 = 10 - A_2 = 10 - (-3.16) = 13.16$。

**第 5 步：写出最终解。**

$$
\boxed{v_C(t) = 13.16\, e^{-1550.5 t} - 3.16\, e^{-6449.5 t}}
$$

验证：
- $t=0$：$13.16 - 3.16 = 10$ V ✓
- $t \to \infty$：两项都趋向于 0，$v_C \to 0$ ✓（零输入，电容最终放完电）
- 导数在 $t=0$ 处：$-1550.5 \times 13.16 + 6449.5 \times 3.16 = -20408 + 20392 \approx -16$（接近零，舍入误差）✓

**物理图像**：$v_C(t)$ 从 10 V 开始单调下降，趋向于 0。早期由 $s_2$（衰减快）主导，后期由 $s_1$（衰减慢）主导。曲线在 10 V 到 0 V 之间平滑过渡，没有振荡。

---

### 9c. 例题 3：欠阻尼的完整求解（零输入响应）

**题目**：RLC 串联电路，$R = 20\text{ }\Omega$，$L = 0.1\text{ H}$，$C = 100\text{ }\mu\text{F}$。初始条件：$v_C(0) = 5\text{ V}$，$i_L(0) = 0$。零输入响应。求 $v_C(t)$ 的完整表达式。

**解：**

**第 1 步：计算 $\alpha$、$\omega_0$，判断阻尼类型。**

$$
\alpha = \frac{R}{2L} = \frac{20}{2 \times 0.1} = \frac{20}{0.2} = 100\text{ s}^{-1}
$$

$$
\omega_0 = \frac{1}{\sqrt{LC}} = \frac{1}{\sqrt{0.1 \times 100 \times 10^{-6}}} = \frac{1}{\sqrt{10^{-5}}} = \sqrt{10^5} \approx 316.23\text{ rad/s}
$$

$\alpha = 100 < 316.23 = \omega_0$，**欠阻尼**。✓

$$
\omega_d = \sqrt{\omega_0^2 - \alpha^2} = \sqrt{100000 - 10000} = \sqrt{90000} = 300\text{ rad/s}
$$

**第 2 步：写出通解形式。**

欠阻尼下的齐次通解：

$$
v_C(t) = e^{-100t} \left[B_1 \cos(300 t) + B_2 \sin(300 t)\right]
$$

**第 3 步：用初始条件定系数。**

条件 1：$v_C(0) = 5$ V。$t=0$ 时 $e^{-0} = 1$，$\cos 0 = 1$，$\sin 0 = 0$：

$$
B_1 \cdot 1 + B_2 \cdot 0 = 5 \quad \Rightarrow \quad B_1 = 5
$$

条件 2：$i_L(0) = 0$，即 $\dfrac{dv_C}{dt}\Big|_{t=0} = 0$。

求导（注意乘积法则：$(uv)' = u'v + uv'$）：

$$
\begin{aligned}
\frac{dv_C}{dt} &= \frac{d}{dt}\left[ e^{-100t} (5\cos 300t + B_2 \sin 300t) \right] \\
&= -100 e^{-100t} (5\cos 300t + B_2 \sin 300t) \\
&\quad + e^{-100t} (-5 \cdot 300 \sin 300t + B_2 \cdot 300 \cos 300t)
\end{aligned}
$$

代入 $t=0$：$e^0 = 1$，$\cos 0 = 1$，$\sin 0 = 0$：

$$
\frac{dv_C}{dt}\Big|_{t=0} = -100(5 \cdot 1 + B_2 \cdot 0) + 1(-1500 \cdot 0 + 300 B_2 \cdot 1) = -500 + 300 B_2
$$

令它等于 0：

$$
-500 + 300 B_2 = 0 \quad \Rightarrow \quad B_2 = \frac{500}{300} \approx 1.667
$$

**第 4 步：写出最终解。**

$$
\boxed{v_C(t) = e^{-100t} \left[5 \cos(300 t) + 1.667 \sin(300 t)\right]}
$$

转化为幅度-相位形式。幅值 $K = \sqrt{5^2 + 1.667^2} = \sqrt{25 + 2.78} \approx 5.27$ V。

相位 $\phi = \arctan\left(\dfrac{1.667}{5}\right) \approx 18.4^\circ$（$18.4^\circ = 0.321$ 弧度）。

$$
v_C(t) \approx 5.27\, e^{-100t} \cos(300t - 18.4^\circ)
$$

**物理图像**：这是一个频率为 300 rad/s（$f \approx 47.7$ Hz）的余弦振荡，包络按 $e^{-100t}$ 衰减。时间常数 $\tau = 1/\alpha = 0.01\text{ s} = 10\text{ ms}$。$5\tau = 50\text{ ms}$ 后包络衰减到初始的约 $0.7\%$，振荡基本消失。

**验证**：
- $t=0$：$e^0(5 \cdot 1 + 1.667 \cdot 0) = 5$ V ✓
- 导数在 $t=0$ 处为零 ✓（由初始条件保证）
- $\omega_d = 300$，周期 $T_d = 2\pi/300 \approx 0.0209\text{ s} \approx 21\text{ ms}$。在 $50\text{ ms}$ 窗口内大约有 2.4 个周期——Q 不高的欠阻尼，振荡几个来回就差不多了。

### 9d. 例题 4：计算品质因数 Q 并用 Q 验证阻尼类型

**题目**：对于例题 3 中的电路（$R=20\text{ }\Omega$，$L=0.1\text{ H}$，$C=100\text{ }\mu\text{F}$），求品质因数 Q。

**解：**

直接用公式 $Q = \dfrac{\omega_0 L}{R}$：

$$
Q = \frac{316.23 \times 0.1}{20} = \frac{31.623}{20} = 1.581
$$

也可以用 $Q = \dfrac{\omega_0}{2\alpha} = \dfrac{316.23}{2 \times 100} = \dfrac{316.23}{200} = 1.581$。两种算法完全一致。

$Q = 1.581 > 0.5$，确认是欠阻尼 ✓。Q 不大（只有约 1.6），所以大约振荡 2-3 个周期就基本消失了，和我们的分析一致。

### 9e. 例题 5：设计一个特定 Q 值的 RLC 电路

**题目**：设计一个 RLC 串联电路，要求谐振频率 $f_0 = 1\text{ kHz}$（$\omega_0 = 2\pi \times 1000 \approx 6283\text{ rad/s}$），品质因数 $Q = 10$。电感选用 $L = 10\text{ mH}$。求电容 C 和电阻 R。

**解：**

**第 1 步：从 $\omega_0$ 求 $C$。**

由 $\omega_0 = 1/\sqrt{LC}$ 解出 $C$：

$$
C = \frac{1}{\omega_0^2 L} = \frac{1}{6283^2 \times 0.01} = \frac{1}{39.48 \times 10^6 \times 0.01} = \frac{1}{3.948 \times 10^5}
$$

$$
C \approx 2.53 \times 10^{-6}\text{ F} = 2.53\text{ }\mu\text{F}
$$

**第 2 步：从 $Q$ 求 $R$。**

由 $Q = \omega_0 L / R$ 解出 $R$：

$$
R = \frac{\omega_0 L}{Q} = \frac{6283 \times 0.01}{10} = \frac{62.83}{10} = 6.283\text{ }\Omega
$$

**第 3 步：验证设计。**

临界电阻 $R_{\text{crit}} = 2\sqrt{L/C} = 2\sqrt{0.01 / (2.53 \times 10^{-6})} = 2\sqrt{3953} \approx 125.7\text{ }\Omega$。

$R = 6.283 \ll 125.7 = R_{\text{crit}}$，确认是**强欠阻尼**。

$\alpha = R/(2L) = 6.283/0.02 = 314.15\text{ s}^{-1}$。

$\omega_d = \sqrt{6283^2 - 314^2} = \sqrt{39.48 \times 10^6 - 98600} \approx 6275\text{ rad/s}$，几乎等于 $\omega_0$。

$Q = \omega_0/(2\alpha) = 6283/628.3 = 10$，设计目标达成。$Q=10$ 意味着大概振荡 $10/\pi \approx 3.2$ 个周期后振幅降到初始的 37%。

**答**：$C = 2.53\text{ }\mu\text{F}$，$R = 6.28\text{ }\Omega$。

## 10. Python 仿真

### 10a. 三种阻尼模式对比（零输入响应）

```python
# ============================================================
# RLC 串联电路仿真：过阻尼、临界阻尼、欠阻尼对比
# 零输入响应：v_C(0) = 10 V，i_L(0) = 0
# ============================================================
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# 设置中文字体（matplotlib 默认不支持中文，必须加这两行）
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# --- 电路参数 ---
L = 0.1          # 电感，单位：亨 (H)
C = 100e-6       # 电容，单位：法 (F)，即 100 uF

# 计算关键参数
omega0 = 1 / np.sqrt(L * C)
R_crit = 2 * np.sqrt(L / C)  # 临界电阻

print(f"谐振频率 ω₀ = {omega0:.1f} rad/s")
print(f"临界电阻 R_crit = {R_crit:.1f} Ω")

# 选择三种电阻值，分别对应过阻尼、临界阻尼、欠阻尼
R_over = 3 * R_crit        # 过阻尼：R 远大于临界值
R_crit_val = R_crit         # 临界阻尼
R_under = 0.2 * R_crit      # 欠阻尼：R 远小于临界值

# --- 定义 RLC 电路的微分方程组 ---
# 状态变量：y = [v_C, i_L]
# dv_C/dt = i_L / C
# di_L/dt = (-v_C - R*i_L) / L
def rlc_ode(t, y, R, L, C):
    vC, iL = y
    dvC_dt = iL / C
    diL_dt = (-vC - R * iL) / L
    return [dvC_dt, diL_dt]

# --- 仿真设置 ---
t_span = [0, 0.08]           # 仿真时间 0 到 80 ms
t_eval = np.linspace(0, 0.08, 2000)  # 2000 个采样点，保证曲线平滑
y0 = [10.0, 0.0]             # 初始条件：v_C=10 V, i_L=0 A

# --- 求解三种情况的 ODE ---
sol_over = solve_ivp(rlc_ode, t_span, y0, t_eval=t_eval,
                     args=(R_over, L, C))
sol_crit = solve_ivp(rlc_ode, t_span, y0, t_eval=t_eval,
                     args=(R_crit_val, L, C))
sol_under = solve_ivp(rlc_ode, t_span, y0, t_eval=t_eval,
                      args=(R_under, L, C))

# --- 绘图 ---
fig, axes = plt.subplots(2, 1, figsize=(13, 9))

# 上图：电容电压 v_C(t)
ax1 = axes[0]
ax1.plot(sol_over.t * 1000, sol_over.y[0], 'b-', linewidth=2,
         label=f'过阻尼 (R = {R_over:.0f} Ω)')
ax1.plot(sol_crit.t * 1000, sol_crit.y[0], 'g-', linewidth=2.5,
         label=f'临界阻尼 (R = {R_crit_val:.1f} Ω)')
ax1.plot(sol_under.t * 1000, sol_under.y[0], 'r-', linewidth=1.8,
         label=f'欠阻尼 (R = {R_under:.0f} Ω)')
ax1.axhline(y=0, color='gray', linestyle='--', alpha=0.4)
ax1.set_xlabel('时间 t (ms)', fontsize=13)
ax1.set_ylabel('电容电压 v_C (V)', fontsize=13)
ax1.set_title('RLC 串联电路零输入响应：v_C(t) — 三种阻尼模式对比',
              fontsize=15, fontweight='bold')
ax1.legend(fontsize=10, loc='upper right')
ax1.grid(True, alpha=0.3)

# 下图：电感电流 i_L(t)
ax2 = axes[1]
ax2.plot(sol_over.t * 1000, sol_over.y[1], 'b-', linewidth=2,
         label=f'过阻尼 (R = {R_over:.0f} Ω)')
ax2.plot(sol_crit.t * 1000, sol_crit.y[1], 'g-', linewidth=2.5,
         label=f'临界阻尼 (R = {R_crit_val:.1f} Ω)')
ax2.plot(sol_under.t * 1000, sol_under.y[1], 'r-', linewidth=1.8,
         label=f'欠阻尼 (R = {R_under:.0f} Ω)')
ax2.axhline(y=0, color='gray', linestyle='--', alpha=0.4)
ax2.set_xlabel('时间 t (ms)', fontsize=13)
ax2.set_ylabel('电感电流 i_L (A)', fontsize=13)
ax2.set_title('RLC 串联电路零输入响应：i_L(t) — 三种阻尼模式对比',
              fontsize=15, fontweight='bold')
ax2.legend(fontsize=10, loc='lower right')
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('/tmp/rlc_three_modes.png', dpi=150, bbox_inches='tight')
plt.show()

# --- 文字总结 ---
print("\n===== 三种阻尼模式总结 =====")
print(f"过阻尼 (R={R_over:.0f} Ω > R_crit): v_C 单调衰减，无振荡。i_L 先负后归零。")
print(f"临界阻尼 (R={R_crit_val:.1f} Ω = R_crit): v_C 最快归零，无振荡、无过冲。")
print(f"欠阻尼 (R={R_under:.0f} Ω < R_crit): v_C 衰减振荡，来回穿越零轴。")
```

### 10b. 不同 Q 值对欠阻尼振荡的影响

```python
# ============================================================
# RLC 电路：不同品质因数 Q 对欠阻尼振荡的影响
# 固定 ω₀，通过改变 R 来改变 Q
# ============================================================
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# --- 固定 L 和 C，只变 R ---
L = 0.1           # H
C = 100e-6        # 100 uF
omega0 = 1 / np.sqrt(L * C)

# 选择四个 Q 值（从低到高），对应的 R 由 R = ω₀L/Q 算出
Q_values = [2, 5, 10, 20]
R_values = [omega0 * L / Q for Q in Q_values]

# 时间轴（0 到 100 ms，高 Q 需要更长时间才能看到多周期振荡）
t = np.linspace(0, 0.1, 3000)

# --- 计算和绘图 ---
plt.figure(figsize=(13, 7))
colors = ['#e74c3c', '#f39c12', '#2ecc71', '#3498db']  # 从暖到冷的配色

for Q, R, color in zip(Q_values, R_values, colors):
    alpha = R / (2 * L)
    omega_d = np.sqrt(omega0**2 - alpha**2)

    # 欠阻尼零输入响应：v_C(0)=10V, i_L(0)=0
    vc = np.exp(-alpha * t) * (
        10 * np.cos(omega_d * t) + (alpha * 10 / omega_d) * np.sin(omega_d * t)
    )

    # 包络线（上下各一条）
    envelope = 10 * np.exp(-alpha * t)

    line, = plt.plot(t * 1000, vc, linewidth=1.8, color=color,
                     label=f'Q = {Q:2d}  (R = {R:.2f} Ω)')
    plt.plot(t * 1000, envelope, '--', linewidth=0.8, alpha=0.4, color=color)
    plt.plot(t * 1000, -envelope, '--', linewidth=0.8, alpha=0.4, color=color)

plt.axhline(y=0, color='gray', linestyle='-', alpha=0.3)
plt.xlabel('时间 t (ms)', fontsize=13)
plt.ylabel('电容电压 v_C (V)', fontsize=13)
plt.title('RLC 欠阻尼响应：不同 Q 值对振荡的影响  (ω₀ ≈ {:.0f} rad/s)'.format(omega0),
          fontsize=15, fontweight='bold')
plt.legend(fontsize=10, loc='upper right')
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('/tmp/rlc_q_factor.png', dpi=150, bbox_inches='tight')
plt.show()

# --- 数值报告 ---
print("\n===== Q 值对比报告 =====")
print(f"{'Q':>4s}  {'R (Ω)':>8s}  {'α (s⁻¹)':>10s}  {'τ (ms)':>8s}  {'ω_d (rad/s)':>12s}  {'约衰减周期数':>12s}")
print("-" * 65)
for Q, R in zip(Q_values, R_values):
    alpha = R / (2 * L)
    tau = 1 / alpha * 1000  # 转为 ms
    omega_d = np.sqrt(omega0**2 - alpha**2)
    cycles_to_37pct = Q / np.pi  # 振幅衰减到 37% 大约需要的周期数
    print(f"{Q:4d}  {R:8.2f}  {alpha:10.1f}  {tau:8.2f}  {omega_d:12.1f}  {cycles_to_37pct:12.1f}")
```

### 10c. RLC 零状态阶跃响应（接通直流电源，观察过冲）

```python
# ============================================================
# RLC 零状态阶跃响应：t=0 接通直流电源 V_s
# 观察不同 Q 值下的过冲（Overshoot）和振荡建立过程
# ============================================================
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# --- 电路参数 ---
Vs = 10.0          # 电源电压 (V)
L = 0.1            # H
C = 100e-6         # 100 uF
omega0 = 1 / np.sqrt(L * C)

# 不同 Q -> 不同 R（Q < 0.5 过阻尼，Q = 0.5 临界阻尼，Q > 0.5 欠阻尼）
Q_values = [0.3, 0.5, 1.0, 3.0, 10.0]
R_values = [omega0 * L / Q for Q in Q_values]

# 时间轴
t = np.linspace(0, 0.06, 2000)  # 0 到 60 ms

# --- 解析计算 v_C(t) ---
plt.figure(figsize=(13, 7))
colors = ['#95a5a6', '#2c3e50', '#e67e22', '#2980b9', '#c0392b']

for Q, R, color in zip(Q_values, R_values, colors):
    alpha = R / (2 * L)
    disc = alpha**2 - omega0**2

    if disc > 0:  # 过阻尼
        # 两个不等实根，零状态响应的系数推导见正文公式
        s1 = -alpha + np.sqrt(disc)
        s2 = -alpha - np.sqrt(disc)
        A1 = Vs * s2 / (s2 - s1)
        A2 = -Vs * s1 / (s2 - s1)
        vc = Vs + A1 * np.exp(s1 * t) + A2 * np.exp(s2 * t)
    elif abs(disc) < 1e-6:  # 临界阻尼
        vc = Vs * (1 - (1 + alpha * t) * np.exp(-alpha * t))
    else:  # 欠阻尼
        omega_d = np.sqrt(-disc)
        vc = Vs * (1 - np.exp(-alpha * t) * (
            np.cos(omega_d * t) + alpha / omega_d * np.sin(omega_d * t)
        ))

    label = f'Q = {Q}  (R = {R:.1f} Ω)'
    if Q < 0.5:
        label += ' [过阻尼]'
    elif abs(Q - 0.5) < 0.01:
        label += ' [临界]'
    else:
        label += ' [欠阻尼]'
    plt.plot(t * 1000, vc, linewidth=2, color=color, label=label)

plt.axhline(y=Vs, color='gray', linestyle='--', alpha=0.6, linewidth=1.2,
            label=f'终值 V_s = {Vs} V')
plt.xlabel('时间 t (ms)', fontsize=13)
plt.ylabel('电容电压 v_C (V)', fontsize=13)
plt.title('RLC 零状态阶跃响应：不同 Q 值下的过冲行为',
          fontsize=15, fontweight='bold')
plt.legend(fontsize=9, loc='lower right')
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('/tmp/rlc_step_response.png', dpi=150, bbox_inches='tight')
plt.show()

print("\n===== 阶跃响应总结 =====")
print("Q < 0.5 (过阻尼)：无过冲，v_C 单调上升到 V_s。R 越大上升越慢。")
print("Q = 0.5 (临界阻尼)：无过冲，v_C 以最快速度到达 V_s。")
print("Q > 0.5 (欠阻尼)：v_C 超过 V_s 后再振荡回落。Q 越大过冲越多、振荡越久。")
```

预期输出：五条曲线从 0 V 出发。灰色（Q=0.3）最慢，深灰（Q=0.5）最快无过冲，橙色（Q=1）有轻微过冲，蓝色（Q=3）和红色（Q=10）有明显过冲和振荡。这正是控制系统中"响应速度 vs 稳定性"权衡的直观体现。

## 11. 应用连接

### 11a. 无线电调谐与选频

你小时候用过那种带旋钮的老式收音机吗？旋转旋钮换台，背后就是一个可调电容器（或者可调电感）。转动旋钮改变 $C$，从而改变 RLC 回路的谐振频率 $\omega_0 = 1/\sqrt{LC}$。

每个电台在不同的频率上发射信号。当你的 RLC 回路的 $\omega_0$ 恰好等于某个电台的发射频率时，那个电台的信号被"选中"放大，其他频率的信号被抑制。这就是**谐振（Resonance）**原理。

Q 值在这里起关键作用：Q 越高，选频越精准——你只听到你想听的台，不会被隔壁频率的台串扰；Q 太低，选频太宽，你可能同时听到两个台混在一起。

下一章交流电路分析会详细展开谐振的数学原理。

### 11b. 开关电源的输出滤波

你手机充电器、笔记本电脑电源适配器内部都有 RLC 滤波器。开关电源的工作原理是快速通断直流（几万到几十万次每秒），产生一串高频脉冲。这些脉冲不能直接给手机用，必须"平滑"成稳定直流。

让脉冲通过一个由 L 和 C 组成的低通滤波器：电感阻挡高频电流变化，电容吸收电压波动。两者合力把高频"毛刺"滤掉，输出光滑的直流。

R 来自电感绕线的电阻和电容的 ESR（等效串联电阻）。欠阻尼设计可能产生输出电压的"振铃"（ringing）——多余的振荡，过阻尼设计响应又太慢。工程师在响应速度和过冲之间权衡，通常选择轻微欠阻尼（比如 $Q \approx 1$ 到 $3$）。

### 11c. 晶振与精密时钟

单片机、手表、手机里都有**石英晶振（Crystal Oscillator）**。晶振的电学模型是一个极高 Q 值（几万到几十万）的 RLC 谐振回路。

晶振的等效电路中，等效电感 $L$ 非常大（几亨到几千亨），等效电容 $C$ 非常小（飞法级，$10^{-15}$ F），等效电阻 $R$ 非常小（几十欧姆）。所以 $\omega_0 = 1/\sqrt{LC}$ 非常精确且稳定，$Q = \omega_0 L / R$ 极高。

高 Q 意味着什么？意味着晶振只在一个极窄的频率范围内振荡，对外界温度变化、电压波动、机械振动都非常不敏感。这就是为什么几块钱的石英手表一个月误差不超过几秒。

### 11d. 音响分频器

你的音箱里面通常不止一个喇叭单元。低音需要大振膜、大功率，高音需要小振膜、快速响应。把一个喇叭用来同时播低音和高音，效果很差。

把全频音频信号分配给不同单元的是**分频器（Crossover）**。分频器的核心就是 RLC 滤波器：低音通路是一个低通滤波器（串 L、并 C——低频通过，高频被挡），高音通路是一个高通滤波器（串 C、并 L——高频通过，低频被挡）。

扬声器本身的音圈也有电感和电阻，所以整套系统就是一个二阶 RLC 网络。分频频率（比如 2.5 kHz）和 Q 值的选取直接影响音箱的频率响应平坦度和音质。

### 11e. 串联与并联 RLC 的对偶

虽然本章以串联 RLC 为重点，但并联 RLC 在射频电路和无线电接收机中极其常见。天线接收到的微弱信号相当于一个电流源，调谐回路通常是并联 RLC。

串联 vs 并联的参数对照：

| 特性 | 串联 RLC | 并联 RLC |
|------|---------|---------|
| 输入源 | 电压源 $V_s$ | 电流源 $I_s$ |
| 微分方程变量 | 回路电流 $i$ | 节点电压 $v$ |
| $\alpha$ | $\dfrac{R}{2L}$ | $\dfrac{1}{2RC}$ |
| $\omega_0$ | $\dfrac{1}{\sqrt{LC}}$ | $\dfrac{1}{\sqrt{LC}}$（相同！） |
| Q | $\dfrac{\omega_0 L}{R}$ | $\omega_0 RC$ |
| 临界阻尼条件 | $R = 2\sqrt{L/C}$ | $R = \dfrac{1}{2}\sqrt{L/C}$ |

有趣的对偶：串联 RLC 中 R 越大越接近过阻尼；并联 RLC 中 R 越大越接近欠阻尼（并联时 R 大意味着能量消耗少，正好反过来了）。但 $\omega_0$ 完全一样——谐振频率只取决于 L 和 C 的乘积，和 R 无关。

## 12. 本章小结

### 核心三角

每个 RLC 电路由三个参数完全决定：

| 参数 | 符号 | 公式 | 一句话描述 |
|------|------|------|-----------|
| 衰减系数 | $\alpha$ | $R/(2L)$ | 包络衰减有多快 |
| 谐振频率 | $\omega_0$ | $1/\sqrt{LC}$ | 没有 R 时的自然振荡频率 |
| 阻尼振荡频率 | $\omega_d$ | $\sqrt{\omega_0^2 - \alpha^2}$ | 有 R 时的实际振荡频率 |

三者关系：

- $\alpha > \omega_0$ → 过阻尼：两个负实根，单调衰减，无振荡
- $\alpha = \omega_0$ → 临界阻尼：重根，最快无振荡回到稳态
- $\alpha < \omega_0$ → 欠阻尼：共轭复根，衰减振荡，$\omega_d$ 是实际振荡频率

### 品质因数 Q

$$
Q = \frac{\omega_0}{2\alpha} = \frac{\omega_0 L}{R} = \frac{1}{R}\sqrt{\frac{L}{C}}
$$

$Q > 0.5$ 才振荡，$Q$ 越大振荡越持久、频率选择性越好。

### 从一阶到二阶：什么变了，什么没变

**没变的**：
- 指数函数 $e^{st}$ 仍然是解的基础
- 稳态求法不变：电容断路、电感短路
- 叠加原理仍然成立
- 初始条件（$v_C(0^+)$ 和 $i_L(0^+)$）仍然由换路前的电路和"不能突变"原则确定

**变了的**：
- 多了一种全新的行为：**振荡**。一阶电路只会"爬"，二阶电路可以"荡"
- 两个参数（$\alpha$ 和 $\omega_0$）代替一个 $\tau$
- 解的形式丰富了三倍：两个实指数、$t e^{-\alpha t}$、$e^{-\alpha t}\cos(\omega_d t + \phi)$
- 三要素法不再适用。二阶电路没有简单的"三个数"公式，必须老老实实解特征方程

### 更高的视角

二阶线性微分方程 $a\ddot{x} + b\dot{x} + cx = F$ 在物理学中无处不在。你今天学会了分析 RLC 电路，你就拥有了分析弹簧振子、建筑物抗震模型、汽车悬挂、音箱分频器、甚至某些经济周期模型的数学语言。变量名字不同，但方程的结构一模一样。这就是为什么电路分析是工科的基础课程——它教会你用数学的眼光看世界。

## 13. 常见误区

### ❌ 误区 1："二阶电路一定会振荡"

不一定。只有**欠阻尼**（$\alpha < \omega_0$，即 $R < 2\sqrt{L/C}$）才振荡。过阻尼和临界阻尼都是单调衰减或单调趋近稳态，没有任何来回。二阶电路**有能力**振荡，但不是**一定**振荡。

### ❌ 误区 2："$\alpha$ 越大，振荡得越快"

正好相反。欠阻尼下振荡频率 $\omega_d = \sqrt{\omega_0^2 - \alpha^2}$。$\alpha$ 越大，$\omega_d$ 越小。当 $\alpha$ 大到等于 $\omega_0$ 时，$\omega_d = 0$，完全不振了。

$\alpha$ 是**衰减**系数，不是**振荡**系数。它越大，振荡越慢（频率越低）、衰减越快（包络越陡）。这两者同时发生，因为同一个 R 同时影响 $\alpha$ 和 $\omega_d$。

### ❌ 误区 3："Q 值和电荷 Q 是同一个东西"

只是符号撞车。电荷 $Q = C v$（单位：库仑 C）和品质因数 $Q = \omega_0 L / R$（无量纲）没有任何关系。从上下文可以区分：如果 $Q$ 和 $R$、$L$、$\omega_0$ 一起出现，就是品质因数；如果 $Q$ 和 $C$、$v$ 一起出现，就是电荷。

### ❌ 误区 4："直接套 $R/(2L)$ 就行，不用管电路怎么接的"

公式里的 $R$ 必须是**从储能元件（L 和 C）看进去的等效电阻**。如果电路里还有别的电阻、或者开关切换后等效电路变了，你得先用戴维南或诺顿等效化简电路，然后再算 $\alpha$。

永远记住第一步：**化简电路**——把 L 和 C 两端的等效电路抓出来，再看 R 到底是多少。

### ❌ 误区 5："临界阻尼在任何情况下都是最优的"

临界阻尼确实最快且无过冲，但很多工程场景下轻微欠阻尼反而更好。欠阻尼虽然有过冲，但初始响应更快（上升时间更短）。在允许少量超调的应用中（比如某些电机驱动、开关电源），设计成 $Q \approx 1$ 到 $2$ 能在"速度"和"稳定"之间达到更好的平衡。

**题 7**：RLC 串联电路，$R = 4\text{ }\Omega$，$L = 0.2\text{ H}$，$C = 500\text{ }\mu\text{F}$，$V_s = 12\text{ V}$（直流）。$t=0$ 接通电源，初始状态全零（零状态响应）。(a) 判断阻尼类型。(b) 计算 $\omega_d$。(c) 稳态下 $v_C$ 的值是多少？

$$

$\alpha = 250 < 316.2 = \omega_0$，**欠阻尼**。

验证：$R_{\text{crit}} = 2\sqrt{L/C} = 2\sqrt{0.2/(50 \times 10^{-6})} = 2\sqrt{4000} \approx 126.5\text{ }\Omega$。$R = 100 < 126.5$，确认欠阻尼。

#### 题 2

$$
\alpha = \frac{R}{2L} = \frac{10}{2 \times 0.05} = 100

$$
\omega_0 = \frac{1}{\sqrt{0.05 \times 2 \times 10^{-6}}}
         = \frac{1}{\sqrt{10^{-7}}} \approx 3162.3

$\alpha = 100 < 3162.3 = \omega_0$，欠阻尼。

\omega_d = \sqrt{\omega_0^2 - \alpha^2} = \sqrt{3162.3^2 - 100^2}

#### 题 6

**临界阻尼最优的原因**：在不产生过冲的前提下，响应速度最快。对万用表指针、机器人关节定位等"必须精确到位、不能晃"的场景，临界阻尼是理想选择。

**欠阻尼也有优势**：虽然有过冲，但初始响应更快（上升时间更短）。在允许少量超调的应用中（比如某些开关电源的瞬态响应、电机启动），故意设计成轻微欠阻尼可以缩短总到达时间。此外，需要振荡的场合——振荡器、无线电调谐——当然必须欠阻尼。

#### 题 7

**(a)** 计算 $\alpha$ 和 $\omega_0$：

$$
\alpha = \frac{R}{2L} = \frac{4}{2 \times 0.2} = \frac{4}{0.4} = 10
$$

$$
\omega_0 = \frac{1}{\sqrt{LC}} = \frac{1}{\sqrt{0.2 \times 500 \times 10^{-6}}}
         = \frac{1}{\sqrt{10^{-4}}} = 100
$$

$\alpha = 10 < 100 = \omega_0$，**欠阻尼**。

**(b)** 阻尼振荡频率：

$$
\omega_d = \sqrt{\omega_0^2 - \alpha^2} = \sqrt{10\,000 - 100} = \sqrt{9900} \approx 99.5\text{ rad/s}
$$

**(c)** 因为是直流电源，稳态下电容相当于断路，其电压等于电源电压：$v_C(\infty) = V_s = 12\text{ V}$。

零状态全解形式（通解 + 特解 $V_s$）：

$$
v_C(t) = 12 + e^{-10t} \left[B_1 \cos(99.5 t) + B_2 \sin(99.5 t)\right]
$$

由 $v_C(0)=0$ 和 $i_L(0)=0$ 可定出 $B_1 = -12$，$B_2 = -\alpha \cdot 12 / \omega_d \approx -1.21$。这条曲线从 0 V 出发，振荡上升，超过 12 V 后回落，再超再回，最终稳定在 12 V。

#### 题 8

RLC 电路和弹簧-质量-阻尼系统的数学方程都是：

$$
\text{（惯性项）}\frac{d^2 x}{dt^2} + \text{（阻尼项）}\frac{dx}{dt} + \text{（恢复项）}x = \text{（外力项）}
$$

前者用 $v_C$ 或 $q$ 做变量，后者用位移 $x$ 做变量。方程的**结构完全相同**，只是每个符号代表的物理实体不同。

对应关系表（至少 5 对）：

| 电学量 | 力学量 |
|--------|--------|
| 电感 $L$ | 质量 $m$（惯性） |
| 电阻 $R$ | 阻尼系数 $b$（摩擦） |
| $1/C$ | 弹簧劲度 $k$（恢复力） |
| 电荷 $q = C v_C$ | 位移 $x$ |
| 电流 $i = dq/dt$ | 速度 $v = dx/dt$ |
| 电源电压 $V_s$ | 外力 $F_{\text{ext}}$ |
| 电场能 $\frac{1}{2}C v_C^2$ | 弹性势能 $\frac{1}{2} k x^2$ |
| 磁场能 $\frac{1}{2}L i_L^2$ | 动能 $\frac{1}{2} m v^2$ |

"同一个数学结构"意味着：你在 RLC 电路中发现的过阻尼、临界阻尼、欠阻尼、Q 值、谐振……所有这些概念在力学系统里完全适用，反之亦然。数学剥离了具体物理对象，只保留最核心的结构关系。

#### 题 9

**(a)** 谐振频率：

$$
\omega_0 = \sqrt{\omega_d^2 + \alpha^2} = \sqrt{800^2 + 200^2}
         = \sqrt{640\,000 + 40\,000} = \sqrt{680\,000} \approx 824.6\text{ rad/s}
$$

**(b)** 品质因数：

$$
Q = \frac{\omega_0}{2\alpha} = \frac{824.6}{2 \times 200} = \frac{824.6}{400} \approx 2.06
$$

**(c)** 振幅按 $e^{-\alpha t}$ 衰减。要衰减到初始的 10%，即 $e^{-\alpha t} = 0.1$：

$$
t = -\frac{\ln(0.1)}{\alpha} = -\frac{-2.3026}{200} = 0.0115\text{ s} = 11.5\text{ ms}
$$

振荡周期 $T_d = 2\pi / \omega_d = 2\pi / 800 \approx 0.00785\text{ s} = 7.85\text{ ms}$。

需要的周期数：$N = t / T_d = 11.5 / 7.85 \approx 1.47$。

大约 1.5 个周期后振幅就只剩 10% 了。$Q \approx 2$ 属于低 Q 欠阻尼，振荡很快就消失了。这和一个 $Q=100$ 的晶振需要约 150 个周期才衰减到 10% 形成鲜明对比。
